## 一、基础储备

### 1.1 计算机网络
- OSI 七层模型 / TCP-IP 四层模型
- 常见协议：HTTP/HTTPS、DNS、ARP、DHCP、SMTP、FTP
- 抓包分析：Wireshark、Fiddler

### 1.2 操作系统
- **Linux**：用户权限、进程管理、网络配置、Shell 脚本
- **Windows**：AD 域、注册表、PowerShell、事件日志

### 1.3 编程基础
- **Python**：自动化脚本、网络编程、正则表达式
- **Bash**：Linux 批处理
- **可选**：JavaScript（Web 漏洞）、C/C++（逆向基础）

### 1.4 Web 基础
- HTML / CSS / JavaScript 基础
- HTTP 请求方法、状态码、Header、Cookie、Session
- 常见 Web 服务器：Nginx、Apache、Tomcat

### 2.2 工具链
- **信息收集**：Nmap、Masscan、Shodan、FOFA、OneForAll
- **漏洞扫描**：Burp Suite、Nuclei、AWVS
- **渗透利用**：Burp Suite Intruder、SQLMap、XSStrike

- DVWA（Web 安全基础）

## 三、内网渗透

### 3.1 主机存活发现
- `arp-scan`、`nbtstat`、`nping`

### 3.2 端口扫描与服务识别
- Nmap 高级用法：端口状态识别、操作系统指纹
-常见端口及对应服务

### 3.3 横向移动
- Pass The Hash（哈希传递）
- 票据传递（Kerberos）
- 端口转发 / 代理工具：Cobalt Strike、frp、EarthWorm

### 3.4 权限维持
- 计划任务、注册表 Run Key、服务创建
- WebShell 免杀、内存马

### 3.5 域渗透
- Active Directory 基础
- 域信息收集：BloodHound、AdFind
- 域内横向：MS14-068、NTLM Relay、Kerberoasting

## 六、学习路线建议

```
第一阶段（1-2月）：网络基础 + Web 安全入门
  → 熟悉 HTTP 协议、抓包工具、SQL 注入、XSS

第二阶段（2-3月）：工具熟练 + 靶场实战
  → 掌握 Burp Suite、Nmap、SQLMap，攻克 DVWA/Pikachu

第三阶段（3-6月）：内网渗透 + CTF
  → 学习内网信息收集、横向移动，参加 CTF 比赛

第四阶段（6月+）：红队实战 / CVE 挖掘
  → 真实目标渗透、漏洞挖掘、工具开发
```

## 七、注意事项

- ⚠️ **授权优先**：所有渗透测试必须在授权范围内进行
- 📝 **记录笔记**：每挖一个漏洞都要写详细的复现报告
- 🔄 **持续学习**：安全领域变化快，关注漏洞预警平台
- 🤝 **加入社区**：多交流，闭门造车效率低
