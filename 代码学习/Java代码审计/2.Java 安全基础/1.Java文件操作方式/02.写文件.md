## 二、核心写入方式详解
### （一）基础 IO 流方式
#### 1. FileOutputStream（字节输出流）
+ **特点**：直接操作字节，适用于所有文件类型（文本、二进制文件等）
+ **核心代码**：

```java
package FileWriteTest;
import java.io.FileOutputStream;

public class FileOutputStreamTestWriteDemo {
    public static void main(String[] args) {
        try {
            FileOutputStream fos = new FileOutputStream("fos.txt");
            byte[] bytes = "hello world".getBytes();
            fos.write(bytes); // 写入字节数组
            fos.close(); // 关闭流
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

+ **执行结果**：根目录生成 `fos.txt`，内容为 `hello world`

#### 2. BufferedOutputStream（缓冲字节输出流）
+ **特点**：基于 FileOutputStream，增加缓冲区，减少 IO 次数，提高写入效率
+ **核心代码**：

```java
package FileWriteTest;
import java.io.BufferedOutputStream;
import java.io.FileOutputStream;

public class BufferedOutputStreamTestWriteDemo {
    public static void main(String[] args) {
        try {
            // 包装 FileOutputStream
            BufferedOutputStream bos = new BufferedOutputStream(new FileOutputStream("bos.txt"));
            bos.write("hello world".getBytes());
            bos.flush(); // 强制刷新缓冲区（必须调用，否则内容可能未写入）
            bos.close();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

+ **执行结果**：根目录生成 `bos.txt`，内容为 `hello world`

#### 3. FileWriter（字符输出流）
+ **特点**：直接操作字符，适用于文本文件，自动处理字符编码（默认系统编码）
+ **核心代码**：

```java
package FileWriteTest;
import java.io.FileWriter;

public class FileWriterTestWriteDemo {
    public static void main(String[] args) {
        try {
            FileWriter f = new FileWriter("fw.txt");
            f.write("abcd"); // 写入完整字符串
            // 截取字符串写入：从索引3开始，截取9个字符（注意：不是结束位置）
            f.write("qwertyuiopasdfghjklzxcvbnm", 3, 9);
            f.close(); // 关闭时自动刷新缓冲区
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

+ **执行结果**：根目录生成 `fw.txt`，内容为 `abcdrtyuiopas`

#### 4. BufferedWriter（缓冲字符输出流）
+ **特点**：基于 FileWriter，增加缓冲区，提高文本写入效率
+ **核心代码**：

```java
package FileWriteTest;
import java.io.BufferedWriter;
import java.io.FileWriter;

public class BufferedWriterTestWriteDemo {
    public static void main(String[] args) {
        try {
            // 包装 FileWriter
            BufferedWriter bw = new BufferedWriter(new FileWriter("bw.txt"));
            bw.write("hello ");
            bw.write("world");
            bw.flush(); // 强制刷新缓冲区
            bw.close();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

+ **执行结果**：根目录生成 `bw.txt`，内容为 `hello world`

#### 5. PrintWriter（打印输出流）
+ **特点**：基于 FileWriter，支持格式化输出，API 更友好
+ **核心代码**：

```java
package FileWriteTest;
import java.io.FileWriter;
import java.io.PrintWriter;

public class PrintWriterTestWriteDemo {
    public static void main(String[] args) {
        try {
            // 包装 FileWriter
            PrintWriter prw = new PrintWriter(new FileWriter("prw.txt"));
            prw.print("hello world"); // 支持 print/println/printf 等方法
            prw.flush();
            prw.close();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

+ **执行结果**：根目录生成 `prw.txt`，内容为 `hello world`

#### 6. RandomAccessFile（随机访问文件）
+ **特点**：支持读写操作，可随机定位文件指针，适用于大文件部分写入
+ **核心代码**：

```java
package FileWriteTest;
import java.io.File;
import java.io.RandomAccessFile;

public class RandomAccessFileTestWriteDemo {
    public static void main(String[] args) {
        try {
            File f = new File("raf.txt");
            String content = "hello world";
            // 打开模式：rw（读写）、r（只读）、rws（读写+内容同步）、rwd（读写+内容/元数据同步）
            RandomAccessFile raf = new RandomAccessFile(f, "rw");
            raf.write(content.getBytes()); // 写入字节数组
            raf.close();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

+ **执行结果**：根目录生成 `raf.txt`，内容为 `hello world`

### （二）NIO Files 工具类（JDK 7+）
+ **特点**：JDK 7 引入的 NIO 工具类，API 简洁，支持多种写入模式
+ **核心代码**：

```java
package FileWriteTest;
import java.nio.charset.StandardCharsets;
import java.nio.file.Files;
import java.nio.file.Paths;
import java.nio.file.StandardOpenOption;

public class FilesTestWriteDemo {
    public static void main(String[] args) {
        try {
            // 普通写模式（覆盖原有内容）
            Files.write(Paths.get("f.txt"), "hello ".getBytes(StandardCharsets.UTF_8));
            // 追加写模式（在原有内容后添加）
            Files.write(
                Paths.get("f.txt"),
                "world".getBytes(StandardCharsets.UTF_8),
                StandardOpenOption.APPEND
            );
            System.out.println("文件创建成功");
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

+ **执行结果**：根目录生成 `f.txt`，内容为 `hello world`

### （三）commons-io 第三方包
需先在 `pom.xml` 中添加依赖：

```xml
<!-- commons-io 依赖 -->
<dependency>
  <groupId>commons-io</groupId>
  <artifactId>commons-io</artifactId>
  <version>2.11.0</version>
</dependency>
```

#### 1. FileUtils 工具类
提供三种常用写入方式，无需手动关闭流，自动处理异常。

##### （1）写入集合内容
```java
package FileWriteTest;
import org.apache.commons.io.FileUtils;
import java.io.File;
import java.util.ArrayList;

public class FileUtilsFileTestWriteDemo1 {
    public static void main(String[] args) throws Exception {
        ArrayList<String> list = new ArrayList<>();
        list.add("test11");
        list.add("test22");

        File f = new File("fu1.txt");
        // 参数：文件、编码、内容集合、分隔符、是否追加
        FileUtils.writeLines(f, "UTF-8", list, ",", false);
    }
}
```

+ 结果：`fu1.txt` 内容为 `test11,test22,`

##### （2）写入字符串
```java
package FileWriteTest;
import org.apache.commons.io.FileUtils;
import java.io.File;

public class FileUtilsFileTestWriteDemo2 {
    public static void main(String[] args) throws Exception {
        File f = new File("fu2.txt");
        String data = "datatest";
        // 参数：文件、内容、编码、是否追加
        FileUtils.writeStringToFile(f, data, "UTF-8", false);
    }
}
```

+ 结果：`fu2.txt` 内容为 `datatest`

##### （3）写入字节数组
```java
package FileWriteTest;
import org.apache.commons.io.FileUtils;
import java.io.File;

public class FileUtilsFileTestWriteDemo3 {
    public static void main(String[] args) throws Exception {
        File f = new File("fu3.txt");
        String str = "hello";
        byte[] data = str.getBytes();
        // 参数：文件、字节数组、起始索引、长度、是否追加
        FileUtils.writeByteArrayToFile(f, data, 0, data.length, false);
    }
}
```

+ 结果：`fu3.txt` 内容为 `hello`

#### 2. IOUtils 工具类
配合输出流使用，简化流操作。

```java
package FileWriteTest;
import org.apache.commons.io.IOUtils;
import java.io.File;
import java.io.FileOutputStream;

public class IOUtilsFileTestWriteDemo3 {
    public static void main(String[] args) throws Exception {
        File file = new File("iu3.txt");
        FileOutputStream out = new FileOutputStream(file);
        String data = "hello";
        IOUtils.write(data, out); // 直接写入字符串，自动处理编码
        out.close();
    }
}
```

+ 结果：`iu3.txt` 内容为 `hello`

## 三、关键注意事项
1. **流关闭**：所有基于流的操作（如 `FileOutputStream`、`BufferedWriter`）必须关闭流（`close()`），否则可能导致资源泄漏。
2. **缓冲区刷新**：缓冲流（`BufferedOutputStream`、`BufferedWriter`）需调用 `flush()` 强制刷新缓冲区，否则内容可能未写入文件。
3. **编码统一**：写入时指定编码（如 `StandardCharsets.UTF_8`），避免中文乱码。
4. **追加模式**：需要追加内容时，使用 `StandardOpenOption.APPEND`（Files）或设置 `append=true`（commons-io）。
5. **commons-io 优势**：第三方包简化代码，无需手动处理流关闭和异常，推荐实际开发中使用。
