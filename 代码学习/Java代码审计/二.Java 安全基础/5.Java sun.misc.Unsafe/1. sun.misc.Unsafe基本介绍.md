##  sun.misc.Unsafe 基本介绍（面向安全人员）
### **一、概述**
+ **sun.misc.Unsafe** 是 Java 提供的一个底层、非公开的 API。
+ 主要用于 JVM 内部操作，**正常情况下开发者无法直接实例化**该类，获取实例通常需通过 **反射** 方式。
+ 所有方法均为 **“不安全”操作**（绕过 Java 安全机制），但性能极高。

### 三、安全人员重点关注
+ **核心目标**：使用 `Unsafe.allocateInstance(Class<?> clazz)`  
→ **绕过构造函数直接创建对象实例**。
+ 此能力常用于：
    - 反序列化漏洞利用
    - 绕过构造器中的安全校验
    - 动态对象构造（无参/私有构造器场景）

+ 初学阶段 **无需深入全部功能**，用时再深入，按需学习。
+ 掌握 `allocateInstance` 的使用即可满足多数安全研究需求。
