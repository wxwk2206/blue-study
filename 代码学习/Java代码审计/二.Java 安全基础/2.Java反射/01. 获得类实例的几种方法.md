## 一、核心前言
获取`java.lang.Class`对象是 Java 反射的基础，所有反射操作均围绕`Class`对象展开，掌握其获取方法是入门反射的关键。

### 重点：
Class类的对象 != 其他类（例如Person），也 != 其他类（例如Person）的对象，**Class类的对象（实例）是JVM 用来描述 Person 类的运行时对象**

## 二、4 种核心获取方法（附示例与说明）
### 准备示例类
```java
package 反射;
class Foo {
    void print() {
        System.out.println("foo的print方法冲冲冲");
    }
}
```

注意：Foo 类无显式构造方法，Java 会默认提供空参构造方法，可通过反射实例化（后续验证）。

### 方法 1：类的`.class`语法⭐
+ 语法：`类名.class`
+ 特点：直接通过类名获取，编译期确定，无需实例化对象
+ 示例代码：

```java
package 反射;
public class Test {
    public static void main(String[] args) {
        Class fooClass1 = Foo.class;
        try {
            Foo foo = (Foo) fooClass1.newInstance();
            foo.print(); // 调用方法
        } catch (InstantiationException | IllegalAccessException e) {
            e.printStackTrace();
        }
    }
}
```

### 方法 2：对象的`getClass()`方法⭐
+ 语法：`对象实例.getClass()`
+ 特点：基于已实例化的对象获取，运行期确定
+ 示例代码：

```java
package 反射;
public class Test {
    public static void main(String[] args) {
        Foo f = new Foo(); // 先实例化对象
        Class fooClass2 = f.getClass();
        try {
            Foo foo = (Foo) fooClass2.newInstance();
            foo.print();
        } catch (InstantiationException | IllegalAccessException e) {
            e.printStackTrace();
        }
    }
}
```

### 方法 3：通过类加载器（ClassLoader）⭐
**ClassLoader不是反射，反射是用Class，ClassLoader是造Class，更加底层。**

+ 语法：`类加载器.loadClass("全类名")`
+ 特点：需指定类的全路径（包名 + 类名），支持动态加载
+ 示例代码：

```java
package 反射;
public class Test {
    public static void main(String[] args) {
        Class fooClass3 = null;
        try {
            // 加载"反射.Foo"类（全类名）
            fooClass3 = ClassLoader.getSystemClassLoader().loadClass("反射.Foo");
            Foo foo = (Foo) fooClass3.newInstance();
            foo.print();
        } catch (ClassNotFoundException | InstantiationException | IllegalAccessException e) {
            e.printStackTrace();
        }
    }
}
```

### 方法 4：Class 的静态方法`forName(String classPath)`⭐
+ 语法：`Class.forName("全类名")`
+ 特点：最常用，支持动态加载，需指定全类名，抛出`ClassNotFoundException`
+ 示例代码：

```java
package 反射;
public class Test {
    public static void main(String[] args) {
        Class fooClass4 = null;
        try {
            fooClass4 = Class.forName("反射.Foo"); // 全类名
            // 无显式构造方法仍可实例化（默认空参构造）
            Foo foo = (Foo) fooClass4.newInstance();
            foo.print();
        } catch (ClassNotFoundException | InstantiationException | IllegalAccessException e) {
            e.printStackTrace();
        }
    }
}
```

![[Pasted image 20260701211024.png]]

+ 关键说明：代码审计时，若`Class.forName`参数可控但`newInstance()`不可控，可寻找无显式构造方法的工具类进行实例化。

## 三、额外知识拓展
### （一）数组类型的 Class 对象（JNI 字段描述符）
当获取数组类型的`Class`对象时，需使用 Java JNI 字段描述符，格式规则：

+ 一维数组：`[` + 元素类型描述符
+ 多维数组：`[[`（二维）、`[[[`（三维）+ 元素类型描述符

| **数组类型** | **JNI 字段描述符** | **验证代码示例** |
| --- | --- | --- |
| String[] | [Ljava.lang.String; | `Class<?> clazz = Class.forName("[Ljava.lang.String;");`<br/> 与 `String[].class`相等 |
| Object[] | [Ljava.lang.Object; | `Class<?> clazz = Class.forName("[Ljava.lang.Object;");`<br/> 与 `Object[].class`相等 |
| Integer[] | [Ljava.lang.Integer; | `Class<?> clazz = Class.forName("[Ljava.lang.Integer;");`<br/> 与 `Integer[].class`相等 |
| Boolean[] | [Ljava.lang.Boolean; | `Class<?> clazz = Class.forName("[Ljava.lang.Boolean;");`<br/> 与 `Boolean[].class`相等 |
| byte[] | [B | `Class<?> clazz = Class.forName("[B");`<br/> 与 `byte[].class`相等 |
| char[] | [C | `Class<?> clazz = Class.forName("[C");`<br/> 与 `char[].class`相等 |
| short[] | [S | `Class<?> clazz = Class.forName("[S");`<br/> 与 `short[].class`相等 |
| int[] | [I | `Class<?> clazz = Class.forName("[I");`<br/> 与 `int[].class`相等 |
| long[] | [J | `Class<?> clazz = Class.forName("[J");`<br/> 与 `long[].class`相等 |
| float[] | [F | `Class<?> clazz = Class.forName("[F");`<br/> 与 `float[].class`相等 |
| double[] | [D | `Class<?> clazz = Class.forName("[D");`<br/> 与 `double[].class`相等 |


### （二）内部类的 Class 对象获取
+ 规则：使用`Class.forName()`获取内部类时，需用`$`代替外部类与内部类之间的`.`（点号）
+ 示例：
    - 外部类：`com.alibaba.Test`
    - 内部类：`Hello`（嵌套在 Test 中）
    - 全类名：`com.alibaba.Test$Hello`
+ 代码示例：

```java
public class Test {
    public static void main(String[] args) {
        Class tClass = null;
        try {
            // 内部类全类名：外部类$内部类
            tClass = Class.forName("com.alibaba.Test$Hello");
        } catch (ClassNotFoundException e) {
            e.printStackTrace();
        }
    }
}
```

## 四、核心总结
| **获取方法** | **语法格式** | **关键特点** |
| --- | --- | --- |
| 类的`.class`<br/>语法 | 类名.class | 编译期确定，无需实例化 |
| 对象的`getClass()`<br/>方法 | 对象实例.getClass () | 运行期确定，需先实例化对象 |
| 类加载器 | 类加载器.loadClass (全类名) | 动态加载，需处理 ClassNotFoundException |
| Class.forName() | Class.forName (全类名) | 最常用，动态加载，支持数组 / 内部类，需处理异常 |
