## 二、获取当前类名称的方法
### 1. getName () 方法
+ 功能：获取类的完整名称（包含包路径）
+ 适用场景：基本数据类型、引用数据类型、void 类型、普通自定义类

#### 代码示例
```java
package 反射.java获取类信息3;

public class Test {
    public static void main(String[] args) {
        // 基本类型、引用类型、void 类型
        Class c1 = int.class;
        Class c2 = String.class; 
        Class c3 = double.class;          //类名.class
        Class c4 = Double.class;
        Class c5 = void.class;

        System.out.println("============int.class===============");
        System.out.println(c1.getName()); // 输出：int
        System.out.println("============String.class===============");
        System.out.println(c2.getName()); // 输出：java.lang.String
        System.out.println("============double.class===============");
        System.out.println(c3.getName()); // 输出：double
        System.out.println("============Double.class===============");
        System.out.println(c4.getName()); // 输出：java.lang.Double
        System.out.println("============void.class===============");
        System.out.println(c5.getName()); // 输出：void

        // 自定义类（Word类）
        Class w = Word.class;
        System.out.println("============Word.class===============");
        System.out.println(w.getName()); // 输出：反射.java获取类信息3.Word
    }
}
```

## 三、获取当前类所有的方法
### 1. getMethods () 方法
+ 功能：获取类中所有 `public` 修饰的方法，包括从父类继承的 public 方法
+ 返回值：`Method[]` 数组（Method 类表示方法对象）
+ 核心操作：
    1. 获取类的 Class 对象
    2. 调用 `getMethods()` 得到方法数组
    3. 遍历数组，获取方法名称、参数类型、返回值类型

#### 代码示例
```java
package 反射.java获取类信息3;

import java.lang.reflect.Method;

public class Test {
    public static void main(String[] args) {
        try {
            // 1. 获取 Word 类的 Class 对象（通过全类名）
            Class w = Class.forName("反射.java获取类信息3.Word");//Class.forName("全类名")
            // 2. 创建类实例（默认调用无参构造）
            Object wObject = w.newInstance();
            // 3. 获取运行时类的 Class 对象（与 w 本质一致）
            Class wc = wObject.getClass();            //实例.getclass（）
            // 4. 获取所有 public 方法（含父类继承）
            Method[] ms = wc.getMethods();

            // 5. 遍历方法数组，打印方法信息
            for (Method method : ms) {
                System.out.println("\n方法名称是: " + method.getName());
                // 获取参数类型
                Class[] paramTypes = method.getParameterTypes();
                for (int j = 0; j < paramTypes.length; j++) {
                    System.out.println("参数" + (j + 1) + "类型: " + paramTypes[j].getName());
                }
                // 获取返回值类型
                System.out.println("方法的返回值的类类型: " + method.getReturnType());
            }
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

#### 运行结果说明
+ 输出的方法包括 Word 类自身的 public 方法（add、start、save、end）
+ 同时包含父类（Object 类）的 public 方法（wait、equals、toString、hashCode 等）

### 2. getDeclaredMethods () 方法
+ 功能：获取类自身**声明的**所有方法（无论访问权限，public/private/protected/default 均可）
+ 特点：不包含从父类继承的方法，仅获取当前类定义的方法
+ 用法：与 getMethods () 类似，遍历逻辑一致

#### 代码示例
```java
package 反射.java获取类信息3;

import java.lang.reflect.Method;

public class Test {
    public static void main(String[] args) {
        try {
            Class w = Class.forName("反射.java获取类信息3.Word");
            Object wObject = w.newInstance();
            Class wc = wObject.getClass();
            // 获取当前类声明的所有方法（不问访问权限）
            Method[] mds = wc.getDeclaredMethods();

            for (Method method : mds) {
                System.out.println("\n方法名称是: " + method.getName());
                Class[] paramTypes = method.getParameterTypes();
                for (int j = 0; j < paramTypes.length; j++) {
                    System.out.println("参数" + (j + 1) + "类型: " + paramTypes[j].getName());
                }
                System.out.println("方法的返回值的类类型: " + method.getReturnType());
            }
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

#### 运行结果说明
+ 输出 Word 类自身声明的所有方法：add、start、save、upload（private）、delUpload（private）、end
+ 不包含 Object 类的继承方法

### 3. 两种方法对比
| **方法** | **访问权限范围** | **是否包含父类方法** |
| --- | --- | --- |
| getMethods() | 仅 public | 是 |
| getDeclaredMethods() | 所有权限 | 否 |


## 四、调用某个类指定的成员方法
通过反射获取到 Method 对象后，可通过 `invoke()` 方法调用目标方法，核心是先获取指定方法（getMethod () /getDeclaredMethod ()），再执行调用。

### 1. getMethod () 方法
+ 功能：获取类中指定的 `public` 方法（含父类继承的 public 方法）
+ 语法格式：

```java
// 无参数方法
Method method = clazz.getMethod("方法名");
// 有参数方法（指定参数类型，多个参数用逗号分隔）
Method method = clazz.getMethod("方法名", 参数类型1.class, 参数类型2.class);
```

#### 示例 1：调用无参数的 public 方法（start ()）
```java
package 反射.java获取类信息3;

import java.lang.reflect.Method;

public class Test {
    public static void main(String[] args) {
        try {
            // 1. 获取 Class 对象
            Class w = Class.forName("反射.java获取类信息3.Word");
            // 2. 创建实例
            Object wObject = w.newInstance();
            // 3. 获取指定 public 方法（无参数）
            Method startMethod = w.getMethod("start");
            // 4. 调用方法（无参数，第二个参数不传）
            Object result = startMethod.invoke(wObject);
            System.out.println(result); // 输出：null（start() 返回值为 void）
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

### 2. getDeclaredMethod () 方法
+ 功能：获取类自身声明的指定方法（无论访问权限）
+ 语法格式：与 getMethod () 一致，仅方法名不同
+ 关键注意：调用私有方法（private）时，必须先设置 `method.setAccessible(true)`，否则会抛出访问权限异常

#### 示例 3：调用有数组参数的 private 方法（upload ()）
```java
package 反射.java获取类信息3;

import java.lang.reflect.Method;

public class Test {
    public static void main(String[] args) {
        try {
            Class w = Class.forName("反射.java获取类信息3.Word");
            Object wObject = w.newInstance();
            // 获取指定私有方法（参数类型：String[]）
            Method uploadMethod = w.getDeclaredMethod("upload", String[].class);
            // 关键：忽略访问权限检查（允许调用私有方法）
            uploadMethod.setAccessible(true);

            // 数组参数传递注意事项：直接传数组会被识别为多个参数，需包装为 Object 数组
            String[] fileArr = new String[]{"上传1", "上传2"};
            // 调用方法（第二个参数：new Object[]{数组}）
            Object result = uploadMethod.invoke(wObject, new Object[]{fileArr});
            System.out.println(result);
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

### 3. invoke () 方法核心说明
+ 功能：执行 Method 对象对应的方法
+ 语法格式：

```plain
Object result = method.invoke(实例对象, 参数值1, 参数值2, ...);
```

+ 参数说明：
    1. 实例对象：非静态方法需传入类实例（静态方法可传 null）
    2. 参数值：与方法定义的参数顺序、类型一致，数组参数需特殊处理
+ 返回值：方法的执行结果（无返回值则返回 null）

### 4. 两种获取方法的调用对比
| **方法** | **适用场景** | **调用私有方法？** | **特殊注意事项** |
| --- | --- | --- | --- |
| getMethod() + invoke() | 调用 public 方法（含父类） | 否 | 无需设置访问权限 |
| getDeclaredMethod() + invoke() | 调用当前类任意权限方法 | 是 | 私有方法需设置 setAccessible (true)；数组参数需包装为 Object 数组 |
