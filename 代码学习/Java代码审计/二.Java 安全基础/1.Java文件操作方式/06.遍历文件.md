## 一、核心功能
遍历指定目录下的所有文件和子目录，递归输出文件 / 文件夹的绝对路径，若指定路径不存在则提示 "文件不存在"。

## 二、关键技术点
### 1. 核心类：java.io.File
+ 作用：表示文件或目录的路径抽象，提供文件 / 目录的创建、查询、遍历等操作
+ 核心构造方法：`new File(String path)`：通过路径字符串创建 File 对象

### 2. 核心方法
| **方法名** | **作用** | **返回值** |
| --- | --- | --- |
| `exists()` | 判断文件 / 目录是否存在 | boolean（存在返回 true，否则 false） |
| `listFiles()` | 获取当前目录下的所有文件和子目录 | File []（目录为空或非目录时返回 null） |
| `isDirectory()` | 判断当前 File 对象是否为目录 | boolean（是目录返回 true，否则 false） |
| `getAbsolutePath()` | 获取文件 / 目录的绝对路径 | String（绝对路径字符串） |


## 三、完整代码
```java
import java.io.File;

public class ListFiles {
    public static void main(String[] args) {
        // 遍历当前目录（"./" 表示当前工作目录）
        printFiles("./");
    }

    // 递归遍历文件/目录的方法
    public static void printFiles(String path) {
        // 1. 根据路径创建 File 对象
        File file = new File(path);

        // 2. 判断文件/目录是否存在
        if (file.exists()) {
            // 3. 获取当前目录下的所有文件和子目录
            File[] files = file.listFiles();

            // 4. 避免空指针异常（非目录或空目录时 listFiles() 返回null）
            if (null != files) {
                // 5. 遍历所有文件/目录
                for (File file2 : files) {
                    // 6. 判断是否为目录：若是，递归遍历；若否，输出文件路径
                    if (file2.isDirectory()) {
                        System.out.println("文件夹:" + file2.getAbsolutePath());
                        printFiles(file2.getAbsolutePath()); // 递归遍历子目录
                    } else {
                        System.out.println("文件:" + file2.getAbsolutePath());
                    }
                }
            }
        } else {
            // 路径不存在时的提示
            System.out.println("文件不存在");
        }
    }
}
```

## 四、代码说明
### 1. 递归逻辑
+ 当遍历到目录时，先输出目录路径，再调用自身遍历该目录下的内容
+ 当遍历到文件时，直接输出文件路径
+ 递归终止条件：目录下无更多子目录或文件

### 2. 异常处理
+ 通过 `if (null != files)` 避免空指针异常（如传入的路径是文件而非目录时，`listFiles()` 返回 null）
+ 通过 `file.exists()` 判断路径有效性，避免无效路径导致的错误

## 五、执行结果
+ 输出当前目录及所有子目录下的内容
+ 格式：文件夹 + 绝对路径 / 文件 + 绝对路径

## 六、注意事项
1. 路径格式：Windows 系统使用 `\` 或 `\\` 作为路径分隔符，Linux/Mac 系统使用 `/`，`./` 表示当前目录，`../` 表示上级目录
2. 权限问题：若访问无权限的目录，`listFiles()` 可能返回 null 或抛出异常
3. 递归深度：遍历层级过深可能导致栈溢出（可通过非递归方式优化，如使用栈或队列实现）
