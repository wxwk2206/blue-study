# <font style="color:rgb(0, 0, 0);">0x01 漏洞成因</font>
```plain
在业务中为了扩展性,单一的语言并不能很好的解决业务问题,所以开发们,会尝试引入其它的一些脚本解析引擎
作为一个嵌入式规则引擎在业务系统中使用,让业务规则定义简便而不失灵活,业务人员就可以定义业务规则
但是如果未过滤,并且外部可控的话,那么就会很容易产生安全风险
```

# 0x02 审计策略
```plain
对于这种漏洞,主要先看是否有使用这些库,然后在看看是否外部可控,最后尝试进行利用
```

```plain
注意:
ScriptEngineManager从Java 6开始自带:
Java 6/7采用的JS解析引擎是Rhino
Java 8开始换成了Nashorn

JavaScriptEngine: 
jar包class: javax.script.ScriptEngine
搜索,eval(...),查看参数一是否外部可控

推荐搜索: 
javax.script.ScriptEngine然后查看引用该类的功能是否外部可控
```

```plain
Groovy:
jar包class: javax.script.ScriptEngine
搜索,eval(...),查看参数一是否外部可控

jar包class: groovy.lang.GroovyClassLoader
搜索,parseClass(...),查看参数一是否外部可控

jar包class: groovy.lang.GroovyShell
搜索,parse(...),查看参数一是否外部可控

推荐搜索: 
groovy.lang.GroovyClassLoader
groovy.lang.GroovyShell
然后查看引用该类的功能是否外部可控
```

```plain
BeanShell:
jar包class: bsh.Interpreter
搜索,eval(...),查看参数一是否外部可控

jar包class: bsh.servlet.BshServlet
搜索,doGet(...),查看参数一是否外部可控
搜索,doPost(...),查看参数一是否外部可控

推荐搜索: 
bsh.Interpreter
bsh.servlet.BshServlet
然后查看引用该类的功能是否外部可控
```

## 0x04.4 JavaScriptEngine
### 0x04.4.1 测试环境搭建
```java
// 路径: ./SpringMVCTest2/src/main/com/test/controller/scriptEngine/JavaScriptEngineTest.java
package test.controller.scriptEngine;

import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.ResponseBody;

import javax.script.ScriptEngine;
import javax.script.ScriptEngineManager;
import javax.script.ScriptException;
import java.io.IOException;
import java.io.InputStream;

@Controller
@RequestMapping("/JavaScriptEngineTest")
public class JavaScriptEngineTest {
    @RequestMapping("/test")
    @ResponseBody
    public String test(String scriptContent) throws IOException, ScriptException {
        // 查找并创建指定脚本引擎
        ScriptEngineManager manager = new ScriptEngineManager();
        ScriptEngine engine = manager.getEngineByName("js");
        // ScriptEngine engine = manager.getEngineByName("javascript");

        // 代码执行的地方
        InputStream result = (InputStream) engine.eval(scriptContent);

        // 创建一个数组用来存储读取到的数据
        byte[] bytes = new byte[1024];
        result.read(bytes);
        result.close();

        // 输出数据
        return new String(bytes);
    }
}
```

## 0x04.5 Groovy
### 0x04.5.1 Groovy常用POC
```plain
Groovy中调用Java执行命令的方法:
POC1: java.lang.Runtime.getRuntime().exec("whoami").getText()
POC2: java.lang.Math.class.forName("java.lang.Runtime").getRuntime().exec("id").getText()

Goovy直接执行命令的方法:
POC1: 'whoami'.execute().text
POC2: def command='whoami';def res=command.execute().text;res
```

### 0x04.5.2 测试环境搭建
```java
// 路径: ./SpringMVCtest/src/main/com/test/controller/scriptEngine/GroovyTest.java
package test.controller.scriptEngine;

import groovy.lang.GroovyClassLoader;
import groovy.lang.GroovyObject;
import groovy.lang.GroovyShell;
import groovy.lang.Script;
import org.codehaus.groovy.jsr223.GroovyScriptEngineFactory;
import org.codehaus.groovy.runtime.InvokerHelper;
import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.ResponseBody;

import javax.script.*;
import java.util.ArrayList;
import java.util.Date;
import java.util.List;

@Controller
@RequestMapping("/GroovyTest")
public class GroovyTest {
    /**
     * 方法一,使用GroovyScriptEngineFactory类的ScriptEngine调用,简单版
     *
     * @param scriptContent
     * @return
     * @throws ScriptException
     */
    @RequestMapping("/test")
    @ResponseBody
    public String test(String scriptContent) throws ScriptException {
        // 查找并创建指定脚本引擎
        ScriptEngine scriptEngine = new GroovyScriptEngineFactory().getScriptEngine();

        // 执行脚本
        Object result = scriptEngine.eval(scriptContent);

        // 输出
        return result.toString();
    }

    /**
     * 方法二,使用GroovyScriptEngineFactory类的ScriptEngine调用,混合版
     *
     * @param s
     * @return
     * @throws ScriptException
     * @throws NoSuchMethodException
     */
    @RequestMapping("/test2")
    @ResponseBody
    public String test2(String s) throws ScriptException, NoSuchMethodException {
        // 查找并创建指定脚本引擎
        ScriptEngine scriptEngine = new GroovyScriptEngineFactory().getScriptEngine();

        // 定义groovy脚本中执行方法的名称
        String scriptName = "execute2";

        // 定义groovy脚本中执行方法的参数
        String name1 = "张三";
        String name2 = "李四";

        // 定义groovy脚本内容
        String scriptContent = "def " + scriptName + "(name1, name2){" +
                "$a = " + s + ";" +
                "$b = \"现在的时间戳是:${date.getTime()}, \" ;" +
                "$c = \"我的大名是:$name1, \";" +
                "$d = \"我的小名是:$name2\";" +
                "return $a + $b + $c + $d;" +
                "}";

        // 初始化Bindings
        Bindings bindings = scriptEngine.createBindings();

        // 绑定参数
        bindings.put("date", new Date());

        // 执行脚本
        scriptEngine.eval(scriptContent, bindings);

        // 调用脚本中定义的顶级过程和函数
        // 参数一要调用的过程或函数
        // 参数一后面所有的参数为传递给过程或函数的参数
        Object result = ((Invocable) scriptEngine).invokeFunction(scriptName, name1, name2);

        // 输出结果
        return result.toString();
    }

    /**
     * 方法三,使用ScriptEngineManager类的ScriptEngine调用
     *
     * @param s
     * @return
     * @throws ScriptException
     * @throws NoSuchMethodException
     */
    @RequestMapping("/test3")
    @ResponseBody
    public String test3(String s) throws ScriptException, NoSuchMethodException {
        // 查找并创建指定脚本引擎
        ScriptEngine scriptEngine = new ScriptEngineManager().getEngineByName("groovy");

        // 定义groovy脚本中执行方法的名称
        String scriptName = "execute3";

        // 定义groovy脚本中执行方法的参数
        String name = "王五";

        // 定义groovy脚本内容
        final String scriptContent = "def " + scriptName + "(name){" +
                "$a = " + s + ";" +
                "$b = \"现在的时间戳是:${date.getTime()}, \";" +
                "$c = \"我的名字是:$name\";" +
                "return $a + $b + $c;" +
                "}";

        // 初始化Bindings
        Bindings bindings = scriptEngine.createBindings();

        // 绑定参数
        bindings.put("date", new Date());

        // 执行脚本
        scriptEngine.eval(scriptContent, bindings);

        // 调用脚本中定义的顶级过程和函数
        // 参数一要调用的过程或函数
        // 参数一后面所有的参数为传递给过程或函数的参数
        Object result = ((Invocable) scriptEngine).invokeFunction(scriptName, name);

        // 输出结果
        return result.toString();
    }

    /**
     * 方法四,使用GroovyClassLoader调用
     *
     * @param s
     * @return
     * @throws IllegalAccessException
     * @throws InstantiationException
     */
    @RequestMapping("/test4")
    @ResponseBody
    public String test4(String s) throws IllegalAccessException, InstantiationException {
        // 查找并创建指定脚本引擎
        GroovyClassLoader groovyClassLoader = new GroovyClassLoader();

        // 定义groovy脚本中执行方法的名称
        final String scriptName = "execute4";

        // 定义groovy脚本中执行方法的参数
        List<String> args = new ArrayList<>();
        args.add("马六");
        args.add("超七");

        // 定义groovy脚本内容
        final String scriptContent = "def " + scriptName + "(name1, name2){" +
                "$a = " + s + ";" +
                "$b = \"我有个朋友名字叫:$name1, \";" +
                "$c = \"我还有个朋友名字叫:$name2\";" +
                "return $a + $b + $c;" +
                "}";

        // 执行脚本
        Class groovyClazz = groovyClassLoader.parseClass(scriptContent);
        GroovyObject groovyObject = (GroovyObject) groovyClazz.newInstance();
        Object result = groovyObject.invokeMethod(scriptName, args);

        // 输出结果
        return result.toString();
    }

    /**
     * 方法五,使用GroovyShell调用(推荐)
     *
     * @param s
     * @return
     */
    @RequestMapping("/test5")
    @ResponseBody
    public String test5(String s) {
        // 查找并创建指定脚本引擎
        GroovyShell groovyShell = new GroovyShell();

        // 定义groovy脚本中执行方法的名称
        String scriptName = "execute5";

        // 定义groovy脚本中执行方法的参数
        List<String> args = new ArrayList<>();
        args.add("郑八");
        args.add("陈九");

        // 定义groovy脚本内容
        String scriptContent = "def " + scriptName + "(name1, name2){" +
                "$a = " + s + ";" +
                "$b = \"我这个朋友叫:$name1, \";" +
                "$c = \"我那个朋友叫:$name2\";" +
                "return $a + $b + $c;" +
                "}";

        // 执行脚本
        Script script = groovyShell.parse(scriptContent);
        Object result = InvokerHelper.invokeMethod(script, scriptName, args);

        // 输出结果
        return result.toString();
    }
}
```

## 0x04.6 BeanShell
### 0x04.6.1 测试环境搭建
```java
// 路径: ./SpringMVCTest2/src/main/com/test/controller/scriptEngine/BeanShellTest.java
package test.controller.scriptEngine;

import bsh.EvalError;
import bsh.Interpreter;
import bsh.servlet.BshServlet;
import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.ResponseBody;

import javax.servlet.ServletException;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;
import java.io.IOException;
import java.io.InputStream;

@Controller
@RequestMapping("/BeanShellTest")
public class BeanShellTest {
    @RequestMapping("/test")
    @ResponseBody
    public String test(String s) throws EvalError, IOException {
        Interpreter interpreter = new Interpreter();
        InputStream result = (InputStream) interpreter.eval(s);

        // 创建一个数组用来存储读取到的数据
        byte[] bytes = new byte[1024];
        result.read(bytes);
        result.close();

        return new String(bytes);
    }

    @RequestMapping("/test2")
    @ResponseBody
    public String test2(HttpServletRequest request, HttpServletResponse response) throws ServletException, IOException {
        BshServlet bshServlet = new BshServlet();
        bshServlet.doPost(request, response);
        return "";
    }
}
```
