# <font style="color:rgb(0, 0, 0);">0x01 漏洞成因</font>
```plain
在业务中可能会存在上传Zip文件并解压的功能(例如:管理员上传主题包,升级补丁包,等等...)

如果Zip解压时,校验不当,攻击者可以通过此功能造成如下安全问题:
1. Zip解压时,Zip文件未过滤,导致文件后缀用户可控制,可以利用此功能造成,任意文件写入
2. Zip解压时,Zip文件未过滤,导致文件路径用户可控制,可以利用“../”配合此功能将文件解压到任意位置
3. Zip解压时,文件总个数未限制/单个文件大小未限制,导致消耗过多的系统资源造成,Dos攻击/Zip炸弹
```

# 0x02 审计策略
## 0x02.1 常规情况下审计策略
```plain
1. 关注应用是否存在Zip解压缩功能(注:如果没有就不审)
2. 该Zip解压功能,是否不限制解压的后缀(例如:1.zip,里面有个文件为“1.jsp”)
3. 该Zip解压功能,是否文件路径用户可控制(例如:1.zip,里面有个文件为“../../www/1.jsp”)
4. 该Zip解压功能,是否文件总个数未限制/单个文件大小未限制
(例如:1.zip,里面9万个文件,每个文件1MB,在解压的时候可形成Dos攻击/Zip炸弹)
```

## 0x02.2 方便快速审计的污点跟踪类
```plain

```

# 0x03 修复方法
```plain
修复方法一:
在解压文件之前对每个文件后缀进行校验,如果校验不通过,就终止整个解压过程,或对其进行忽略
(例如:不允许出现后缀为“jsp,java,class,jar”的文件进行解压)

修复方法二:
在解压文件之前对每个文件名进行校验,确保文件名不包含任何可能被解释为目录的子字符串
如果验证不通过,就终止整个解压过程,或对其进行忽略
(例如:不允许出现“../../data/www/js/jQuery.js”这种名字进行解压)

修复方法三:
在解压文件之前对每个文件大小进行校验,过大的单个文件应该不予处理,或抛出异常
(例如:1.zip,里面有个1.txt,它有2GB)

修复方案四:
在解压文件之前对文件总数进行限制,文件总数超过100的(根据业务修改),抛出异常
(例如:1.zip,里面有1万个文件,每个文件10KB-1MB)
```

## 0x04.3 Zip解压测试代码
```java
// 路径: ./SpringMVCTest2/src/main/com/test/controller/zip/ZipTest1.java
package test.controller.zip;

import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.ResponseBody;
import org.springframework.web.multipart.MultipartFile;

import javax.servlet.http.HttpServletRequest;
import java.io.*;
import java.util.UUID;
import java.util.zip.ZipEntry;
import java.util.zip.ZipInputStream;

@Controller
@RequestMapping(value = "/ZipTest1")
public class ZipTest1 {
    @RequestMapping(value = "/uploadZip", produces = "text/html;charset=UTF-8")
    @ResponseBody
    public String uploadZip(MultipartFile uploadFile, HttpServletRequest request) throws IOException {
        // 上传的⽂件原名,如xxx.zip
        String originalFilename = uploadFile.getOriginalFilename();

        // 获取⽂件的扩展名,如zip
        String extendName = originalFilename.substring(originalFilename.lastIndexOf(".") + 1).trim().toLowerCase();

        // 设置只允许zip包上传
        if (!"zip".equals(extendName)) {
            return "请上传zip压缩包";
        }

        // 修改zip文件名称
        String newName = UUID.randomUUID().toString() + "." + extendName;

        // 文件保存地址
        String realPath = request.getSession().getServletContext().getRealPath("/zip");

        // 创建zip目录
        File floder = new File(realPath);
        if (!floder.exists()) {
            floder.mkdirs();
        }

        // 把MultipartFile转换换成File
        // 保存文件在本地
        uploadFile.transferTo(new File(floder, newName));

        // 输出上传路径
        return "上传路径: " + realPath + "/" + newName;
    }

    @RequestMapping(value = "/unZip", produces = "text/html;charset=UTF-8")
    @ResponseBody
    public String unZip(HttpServletRequest request, String zipName) throws IOException {
        String unZipFilePath = request.getSession().getServletContext().getRealPath("/unzip");

        // 创建解压目录
        File destDir = new File(unZipFilePath);
        if (!destDir.exists()) {
            destDir.mkdir();
        }

        // 获取Zip数据
        String zipFilePath = request.getSession().getServletContext().getRealPath("/zip");
        ZipInputStream zipIn = new ZipInputStream(new FileInputStream(zipFilePath + "/" + zipName));
        ZipEntry entry = zipIn.getNextEntry();

        // 遍历Zip文件中的条目
        while (entry != null) {
            String filePath = unZipFilePath + File.separator + entry.getName();
            if (!entry.isDirectory()) {
                // 如果条目是文件直接解压
                BufferedOutputStream bos = new BufferedOutputStream(new FileOutputStream(filePath));
                byte[] bytesIn = new byte[4096];
                int read = 0;
                while ((read = zipIn.read(bytesIn)) != -1) {
                    bos.write(bytesIn, 0, read);
                }
                bos.close();
            } else {
                // 如果条目是目录,创建对应的目录
                File dir = new File(filePath);
                dir.mkdir();
            }
            zipIn.closeEntry();
            entry = zipIn.getNextEntry();
        }
        zipIn.close();
        return "解压地址:" + unZipFilePath;
    }
}
```


```python
# 版本: Python 3.8.2
# python文件名: 1.py
# 执行命令: python3 1.py
# 执行完毕以后该文件的根目录就会生成一个1.zip
import zipfile
zf = zipfile.ZipFile('1.zip', 'w')

fname1 = '1.txt'
zf.write(fname1, '../1.txt')

fname2 = "2.txt"
zf.write(fname2, '2.txt')

print("ok")
```
