# <font style="color:rgb(0, 0, 0);">0x01 漏洞成因</font>
```plain
在WEB应用,由于业务的需求,可能会在后台功能上,提供文件操作的一系列功能
例如:读文件,写文件,删除文件,复制文件,遍历文件,重命名文件等等功能...
如果校验不当的话,攻击者就可以对服务器的任意文件进行操作
```

# 0x02 审计策略
```plain
1. 查看是否限制了可操作文件的路径(例如:程序设置路径为upload目录,现在跳到了www目录,进行增删改查)
2. 查看是否限制了可操作文件的类型(例如:程序原本只能读logs目录下的log后缀文件,现在可读jsp文件)
3. 查看是否将敏感文件进行排除(例如:web.xml这种文件一般就不该可被外部控制)
4. 查看是否使用java.io.FilePermission限制了文件夹/文件权限
```

# 0x04 修复方法
```plain
1. 对要操作的文件夹/文件名进行黑白名单限制
2. 对用户输入的数据进行过滤,禁止用户输入“./”、“../”、“/”、“\”、“%”
3. 使用java.io.FilePermission限制文件夹/文件权限
```

## 0x05.4 读文件-FileUtils
```java
// 路径: ./SpringMVCtest/src/main/com/test/controller/fileOperation/FileTest1.java
package test.controller.fileOperation;

import org.apache.commons.io.FileUtils;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.ResponseBody;
import org.springframework.web.bind.annotation.RestController;

import javax.servlet.http.HttpServletRequest;
import java.io.File;
import java.io.IOException;

@RestController
@RequestMapping(value = "/FileTest1")
public class FileTest1 {
    @RequestMapping("/read")
    @ResponseBody
    public String read(HttpServletRequest request, String name) throws IOException {
        String realPath = request.getSession().getServletContext().getRealPath("/");
        String readPath = realPath + name;

        File f = new File(readPath);
        String data = FileUtils.readFileToString(f, "UTF-8");
        return data;
    }
}
```

## 0x05.5 写文件-FileWriter
```java
// 路径: ./SpringMVCTest2/src/main/com/test/controller/fileOperation/FileTest2.java
package test.controller.fileOperation;

import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.ResponseBody;
import org.springframework.web.bind.annotation.RestController;

import javax.servlet.http.HttpServletRequest;
import java.io.FileWriter;
import java.io.IOException;

@RestController
@RequestMapping(value = "/FileTest2")
public class FileTest2 {
    @RequestMapping("/write")
    @ResponseBody
    public String write(HttpServletRequest request, String name, String data) throws IOException {
        String realPath = request.getSession().getServletContext().getRealPath("/");
        String writePath = realPath + name;
        FileWriter f = new FileWriter(writePath);

        // 普通写模式
        f.write(data);

        // 关闭写入流,同时会把缓冲区内容写入文件
        f.close();

        return "写入的地址为:" + writePath;
    }
}
```

```java
import java.io.File;
import java.io.IOException;
import java.nio.file.Files;

@RestController
@RequestMapping(value = "/FileTest4")
public class FileTest4 {
    @RequestMapping("/copy")
    @ResponseBody
    public String copy(HttpServletRequest request, String name, String name2) throws IOException {
        String realPath = request.getSession().getServletContext().getRealPath("/");

        // 要复制的文件
        String sourcePath = realPath + name;
        File sourceFile = new File(sourcePath);

        // 复制的路径,并且为文件强制加上"copy-"的前缀
        String targetPath = realPath + "copy-" + name2;
        File targetFile = new File(targetPath);

        // 复制文件
        Files.copy(sourceFile.toPath(), targetFile.toPath());

        return "复制成功:" + targetPath;
    }
}
```

## 0x05.9 遍历文件-Files
```java
// 路径: ./SpringMVCTest2/src/main/com/test/controller/fileOperation/FileTest6.java
package test.controller.fileOperation;

import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.ResponseBody;
import org.springframework.web.bind.annotation.RestController;

import javax.servlet.http.HttpServletRequest;
import java.io.File;
import java.util.ArrayList;
import java.util.List;

@RestController
@RequestMapping(value = "/FileTest6")
public class FileTest6 {
    @RequestMapping("/listFiles")
    @ResponseBody
    public String listFiles(HttpServletRequest request, String path) {
        String realPath = request.getSession().getServletContext().getRealPath("/");
        return printFiles(realPath + path);
    }

    public String printFiles(String path) {
        List<String> data = new ArrayList<String>();

        File file = new File(path);
        if (file.exists()) {
            File[] files = file.listFiles();
            if (null != files) {
                for (File file2 : files) {
                    if (file2.isDirectory()) {
                        data.add("文件夹:" + file2.getAbsolutePath() + "<br/>\r\n");
                        printFiles(file2.getAbsolutePath());
                    } else {
                        data.add("文件:" + file2.getAbsolutePath() + "<br/>\r\n");
                    }
                }
            }
        } else {
            return "文件不存在";
        }
        return data.toString();
    }
}
```
