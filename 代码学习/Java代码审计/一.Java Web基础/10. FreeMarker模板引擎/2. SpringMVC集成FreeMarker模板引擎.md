## 一、测试环境搭建
### （一）基础环境准备
若未搭建 Spring MVC 基础环境，需先完成该环境搭建，为后续集成 FreeMarker 做好铺垫。

### （二）修改 pom.xml 导入依赖
1. 文件路径：`./项目/pom.xml`
2. 注意事项：首次添加依赖可能出现报红，需点击 IDEA 右侧 Maven 按钮刷新，等待工具自动导入依赖文件。
3. 依赖内容：在`<<dependencies></</dependencies>`标签内添加以下依赖（无该标签则自行创建）
    - FreeMarker 核心依赖

```xml
<!-- https://mvnrepository.com/artifact/org.freemarker/freemarker -->
<dependency>
  <groupId>org.freemarker</groupId>
  <artifactId>freemarker</artifactId>
  <version>2.3.30</version>
</dependency>
```

    - Spring 对 FreeMarker 的支持依赖

```xml
<!-- https://mvnrepository.com/artifact/org.springframework/spring-context-support -->
<dependency>
  <groupId>org.springframework</groupId>
  <artifactId>spring-context-support</artifactId>
  <version>4.3.6.RELEASE</version>
</dependency>
```

### （三）修改 springmvc.xml 配置
1. 文件路径：`./项目/src/main/resources/springmvc.xml`
2. 注意事项：若存在重复 id 的 bean 标签，需先注释原有标签，再添加新配置。
3. 配置内容：在`<beans></beans>`标签内添加以下配置
    - 配置 FreeMarker 视图解析器

```xml
<!-- 配置FreeMarker视图解析器 -->
<bean id="viewResolver" class="org.springframework.web.servlet.view.freemarker.FreeMarkerViewResolver">
  <!-- 设置响应字符集 -->
  <property name="contentType" value="text/html;charset=utf-8"/>
  <!-- 设置视图文件后缀名称，一般采用【.ftl】后缀 -->
  <property name="suffix" value=".ftl"/>
</bean>
```

    - 配置 FreeMarker 模板信息

```xml
<bean id="freeMarkerConfigurer" class="org.springframework.web.servlet.view.freemarker.FreeMarkerConfigurer">
  <!-- 设置视图模板文件的存放位置 -->
  <property name="templateLoaderPath" value="/WEB-INF/view/"/>
  <!-- 其他配置信息 -->
  <property name="freemarkerSettings">
    <props>
      <!-- 设置freemarker渲染时采用的字符集 -->
      <prop key="defaultEncoding">UTF-8</prop>
    </props>
  </property>
</bean>
```

## 二、案例实现
### （一）创建 Controller 控制器
1. 文件路径：`./项目/src/main/com/test/controller/FreeMarkerController.java`
2. 代码内容

```java
package test.controller;

import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;
import javax.servlet.http.HttpServletRequest;

@Controller
@RequestMapping("/FreeMarkerTest")
public class FreeMarkerController {

    @RequestMapping("/test")
    public String test(HttpServletRequest request) {
        // 向请求域中添加数据
        request.setAttribute("freeMarker", "FreeMarker");
        // 返回视图名称（对应ftl文件名称）
        return "freeMarkerViewTest";
    }
}
```

### （二）创建 FreeMarker 视图
1. 文件路径：`./项目/src/main/webapp/WEB-INF/view/freeMarkerViewTest.ftl`
2. 代码内容

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>SpringMVC集成Freemarker</title>
  </head>
  <body>
    <!-- 读取请求域中的数据并渲染 -->
    <h3>恭喜你,完成SpringMVC集成${freeMarker}模板引擎</h3>
  </body>
</html>
```

### （三）访问测试
1. 访问地址：`http://localhost:8080/freeMarker_war/FreeMarkerTest/test`
2. 结果：页面显示 “恭喜你，完成 SpringMVC 集成 Freemarker 模板引擎”，查看页面源码可看到渲染后的完整 HTML 结构。
