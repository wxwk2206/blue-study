## 一、测试环境搭建
### （一）基础环境准备
若未搭建 Spring MVC 基础环境，需先完成该环境搭建（可参考相关教程）。

### （二）修改 pom.xml 导入依赖
1. 文件路径：`./项目/pom.xml`
2. 操作说明：在`<<dependencies></</dependencies>`标签内添加依赖（无该标签则手动创建）；首次添加依赖报红时，点击 IDEA 的 Maven 按钮刷新，等待自动导入。
3. 依赖配置：

```xml
<!-- Spring WebMVC依赖 -->
<dependency>
  <groupId>org.springframework</groupId>
  <artifactId>spring-webmvc</artifactId>
  <version>4.3.30.RELEASE</version>
</dependency>

<!-- Spring上下文支持依赖 -->
<dependency>
  <groupId>org.springframework</groupId>
  <artifactId>spring-context-support</artifactId>
  <version>4.3.6.RELEASE</version>
</dependency>

<!-- Velocity核心依赖 -->
<dependency>
  <groupId>org.apache.velocity</groupId>
  <artifactId>velocity</artifactId>
  <version>1.7</version>
</dependency>

<!-- Velocity工具类依赖 -->
<dependency>
  <groupId>org.apache.velocity</groupId>
  <artifactId>velocity-tools</artifactId>
  <version>2.0</version>
</dependency>
```

### （三）添加 Velocity 配置文件
1. 文件路径：`./项目/src/main/resources/velocity.properties`
2. 配置内容（设置字符集）：

```properties
output.encoding=UTF-8
input.encoding=UTF-8
contentType=text/html;charset=UTF-8
```

### （四）修改 springmvc.xml
1. 文件路径：`./项目/src/main/resources/springmvc.xml`
2. 操作说明：若存在重复 id 的 bean 标签，先注释原有标签，再添加以下配置。
3. 配置内容：

```xml
<!-- Velocity模板配置 -->
<bean id="velocityConfig" class="org.springframework.web.servlet.view.velocity.VelocityConfigurer">
  <!-- 模板文件存放路径 -->
  <property name="resourceLoaderPath" value="/WEB-INF/view/"/>
  <!-- 配置文件路径 -->
  <property name="configLocation" value="classpath:velocity.properties"/>
  <!-- 字符集配置 -->
  <property name="velocityProperties">
    <props>
      <prop key="input.encoding">UTF-8</prop>
      <prop key="output.encoding">UTF-8</prop>
    </props>
  </property>
</bean>

<!-- Velocity视图解析器配置 -->
<bean id="viewResolver" class="org.springframework.web.servlet.view.velocity.VelocityViewResolver">
  <!-- 模板文件后缀 -->
  <property name="suffix" value=".vm"/>
  <!-- 模板文件前缀（此处为空，直接使用文件名） -->
  <property name="prefix" value=""/>
  <!-- 响应内容类型 -->
  <property name="contentType" value="text/html;charset=UTF-8"/>
</bean>
```

## 二、案例实现
### （一）创建 Controller 控制器
1. 文件路径：`./项目/src/main/com/test/controller/VelocityController.java`
2. 代码内容：

```java
package test.controller;

import org.springframework.ui.Model;
import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;

@Controller
public class VelocityController {
    // 映射路径：/velocity 和 /test
    @RequestMapping({"/velocity", "/test"})
    public String test(Model model) {
        // 向视图传递数据
        model.addAttribute("velocity", "Velocity");
        // 返回视图名称（对应vm文件，无需后缀）
        return "velocityViewTest";
    }
}
```

### （二）创建 Velocity 视图
1. 文件路径：`./项目/src/main/webapp/WEB-INF/view/velocityViewTest.vm`
2. 视图内容：

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>SpringMVC集成Velocity</title>
  </head>
  <body>
    <!-- 读取Controller传递的数据 -->
    <h3>恭喜你，完成SpringMVC集成$velocity模板引擎</h3>
  </body>
</html>
```

### （三）访问测试
1. 访问地址：`http://127.0.0.1:8081/SpringMVCTest_war/velocity` 或 `http://127.0.0.1:8081/SpringMVCTest_war/test`
2. 结果：页面显示 “恭喜你，完成 SpringMVC 集成 Velocity 模板引擎”，查看页面源码可看到正确的 HTML 结构。
