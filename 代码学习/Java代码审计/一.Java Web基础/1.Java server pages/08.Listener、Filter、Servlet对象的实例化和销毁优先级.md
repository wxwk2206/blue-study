##  实例化优先级  
 Listener( 监听器 )->Filter( 过滤器 )->Servlet  

+  Listener: 实例化通常发⽣在 Web 应⽤程序启动时 , 即 ServletContext 被初始化时 
+ Filter: 实例化通常发⽣在第⼀ 次请求到达时 , 之后该实例通常会被缓存
+ Servlet: 实例化通常发⽣在第⼀ 次请求到达时  

##  销毁优先级  
 Servlet->Filter( 过滤器 )->Listener( 监听器 )  

+  Servlet: 容器⾸先调⽤ Servlet 的 destroy() ⽅法 
+ Filter: 容器接着调⽤ Filter 的 destroy() ⽅法 
+ Listener: 最后 , 容器通知 Listener 销毁事件 , 如调⽤ ServletContextListener 的 contextDestroyed() ⽅法 

##   详细说明  
 Listener: 

+ 在 Web 应⽤程序启动时 , 容器会实例化所有的 Listener 对象 
+ 在 Web 应⽤程序卸载时 , 容器会通知所有的 Listener 对象 

Filter: 

+ 在第⼀ 次请求到达时实例化 
+ 在 Web 应⽤程序卸载时销毁 

Servlet: 

+ 在第⼀ 次请求到达时实例化 
+ 在 Web 应⽤程序卸载时销毁
