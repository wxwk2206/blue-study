## 一、依赖导入（修改 pom.xml）
需在项目的 `pom.xml` 文件中添加两个核心依赖，用于文件上传相关功能实现。

### 1. 依赖添加位置
在 `<<dependencies></</dependencies>` 标签内添加（若标签不存在则手动创建）。

### 2. 具体依赖配置
+ commons-fileupload 依赖（文件上传核心依赖）：

```xml
<!-- https://mvnrepository.com/artifact/commons-fileupload/commons-fileupload -->
<dependency>
  <groupId>commons-fileupload</groupId>
  <artifactId>commons-fileupload</artifactId>
  <version>1.4</version>
</dependency>
```

+ commons-io 依赖（IO 操作辅助依赖）：

```xml
<!-- https://mvnrepository.com/artifact/commons-io/commons-io -->
<dependency>
  <groupId>commons-io</groupId>
  <artifactId>commons-io</artifactId>
  <version>2.11.0</version>
</dependency>
```

### 3. 注意事项
首次添加依赖后可能出现报红，需点击 IDEA 中的 Maven 刷新按钮，等待工具自动导入依赖文件。

## 二、核心类与方法解析
### 1. ServletFileUpload 类
核心作用：解析上传的文件数据

| **方法名** | **作用** | **返回值** |
| --- | --- | --- |
| isMultipartContent | 判断上传的数据格式是否为多段格式（文件上传需满足此格式） | boolean |
| parseRequest | 解析上传的完整数据 | List<FileItem> |


### 2. FileItem 类
核心作用：封装上传数据中的每个表单项（普通表单项或文件表单项）

| **方法名** | **作用** | **返回值** |
| --- | --- | --- |
| isFormField | 判断当前表单项是否为普通表单项（文件表单项返回 false） | boolean |
| getFieldName | 获取表单项的 name 属性值 | String |
| getString | 获取普通表单项的值 | String |
| getName | 获取上传文件的原始文件名 | String |
| write(file) | 将上传的文件写入到参数 file 指向的磁盘位置 | void |

### 2. 核心代码（Upload.java）
```java
package com.Servlet;

import org.apache.commons.fileupload.servlet.ServletFileUpload;
import org.apache.commons.fileupload.FileItemFactory;
import org.apache.commons.fileupload.FileItem;
import org.apache.commons.fileupload.disk.DiskFileItemFactory;
import javax.servlet.annotation.WebServlet;
import javax.servlet.http.HttpServlet;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;
import java.io.File;
import java.io.IOException;
import java.text.SimpleDateFormat;
import java.util.Date;
import java.util.List;
import java.util.UUID;

@WebServlet("/upload")
public class Upload extends HttpServlet {
    @Override
    protected void doPost(HttpServletRequest request, HttpServletResponse response) throws IOException {
        // 1. 判断上传数据是否为多段格式（文件上传必须满足）
        if (!ServletFileUpload.isMultipartContent(request)) {
            response.getWriter().println("no");
            return;
        }

        // 2. 定义文件保存路径：项目下的 uploads 目录，按日期分文件夹存储（解决文件数量限制）
        String realPath = request.getSession().getServletContext().getRealPath("/uploads");
        String datePath = new SimpleDateFormat("yyyy-MM-dd").format(new Date());
        String uploadPath = realPath + "/" + datePath;

        // 3. 创建保存目录（若不存在则自动创建多级目录）
        File folder = new File(uploadPath);
        if (!folder.exists()) {
            folder.mkdirs();
        }

        // 4. 创建文件上传工厂和解析工具
        FileItemFactory factory = new DiskFileItemFactory();
        ServletFileUpload servletFileUpload = new ServletFileUpload(factory);

        try {
            // 5. 解析上传数据，获取所有表单项
            List<FileItem> list = servletFileUpload.parseRequest(request);

            // 6. 遍历表单项，处理文件上传
            for (FileItem item : list) {
                // 判断是否为文件表单项（非普通表单项）
                if (!item.isFormField()) {
                    // 获取文件扩展名（如 txt、jpg）
                    String extendName = item.getName().substring(item.getName().lastIndexOf(".") + 1).trim().toLowerCase();
                    // 生成唯一文件名（UUID 避免文件名重复）
                    String newName = UUID.randomUUID().toString() + "." + extendName;
                    // 保存文件到指定路径
                    item.write(new File(uploadPath + "/" + newName));

                    // 响应上传结果（返回表单项 name 和文件保存路径）
                    response.getWriter().println("uploadName:" + item.getFieldName());
                    response.getWriter().println("uploadPath:" + uploadPath + "/" + newName);
                    return;
                }
            }
        } catch (Exception e) {
            e.printStackTrace();
            response.getWriter().println("nonono");
            return;
        }
    }
}
```
