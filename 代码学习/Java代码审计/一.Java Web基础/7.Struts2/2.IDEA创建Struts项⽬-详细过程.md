## <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">一、创建项目</font>
### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（一）打开 IDEA 并创建新项目</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">打开 IntelliJ IDEA，点击 “创建新项目”。</font>

### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（二）选择创建方式与组件</font>
1. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">选择 “Maven” 作为项目构建工具，使用 JDK1.8。</font>
2. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">组件选择</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">org.apache.maven.archetypes</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">下的</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">maven-archetype-webapp</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（版本 1.4）。</font>

### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（三）完善项目信息</font>
1. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">填写项目核心信息：</font>
    - <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Name（项目根文件名）：Struts2Test</font>
    - <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Group（组名，一般为公司名）：org.example</font>
    - <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Artifact（项目名）：Struts2Test</font>
    - <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Version：1.0-SNAPSHOT</font>
2. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">配置 Maven 相关：</font>
3. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">点击 “完成” 创建项目。</font>

### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（四）IDEA 关键设置(</font><font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">⭐</font><font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">)</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">对准</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">webapp</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">目录右键，选择 “标记目录为”→“源码根”，此步骤为前置必需操作。</font>

### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（五）修改 pom.xml 导入依赖</font>
1. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">在</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"><project></project></font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">标签中添加 JDK 编译版本配置：</font>

```xml
<properties>
  <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
  <maven.compiler.source>1.8</maven.compiler.source>
  <maven.compiler.target>1.8</maven.compiler.target>
</properties>
```

2. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">在</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"><<dependencies></</dependencies></font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">标签中添加 Struts2 核心依赖（无该标签则手动创建）：</font>

```xml
<dependency>
  <groupId>org.apache.struts</groupId>
  <artifactId>struts2-core</artifactId>
  <version>2.5.30</version>
</dependency>
```

3. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">点击 Maven 刷新按钮，等待 IDEA 自动导入依赖（首次添加可能报红，刷新后即可解决）。</font>

### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（六）修改 web.xml 配置</font>
1. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">打开路径</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">./项目/src/main/webapp/WEB-INF/web.xml</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">。</font>
2. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">在</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"><web-app></web-app></font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">标签中添加 Struts2 核心过滤器（Struts2.5 及以后版本）：</font>

```xml
<display-name>Struts2Test</display-name>
<filter>
  <filter-name>struts2</filter-name>
  <filter-class>org.apache.struts2.dispatcher.filter.StrutsPrepareAndExecuteFilter</filter-class>
</filter>
<filter-mapping>
  <filter-name>struts2</filter-name>
  <url-pattern>/*</url-pattern>
</filter-mapping>
```

<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">注意：不同 Struts2 版本过滤器类不同</font>

+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">2.1 版本之前：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">org.apache.struts2.dispatcher.FilterDispatcher</font>`
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">2.2.1-2.3.37：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">org.apache.struts2.dispatcher.ng.filter.StrutsPrepareAndExecuteFilter</font>`
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">2.5 及以后：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">org.apache.struts2.dispatcher.filter.StrutsPrepareAndExecuteFilter</font>`

### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（七）创建 Action 类</font>
1. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">在</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">src/main/webapp</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">下创建</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">action</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">目录。</font>
2. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">在该目录下创建</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">HelloAction.java</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">文件，代码如下：</font>

```java
package action;
import com.opensymphony.xwork2.ActionSupport;
public class HelloAction extends ActionSupport {
    @Override
    public String execute() throws Exception {
        return "test";
    }
}
```

### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（八）创建 testView.jsp 页面</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">在</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">src/main/webapp</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">目录下创建</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">testView.jsp</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">文件，内容如下：</font>

```plain
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<html>
<head>
    <title>Hello</title>
</head>
<body>
    <div>HelloActionTest</div>
</body>
</html>
```

### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（九）创建 struts.xml 配置文件</font>
1. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">在</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">WEB-INF</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">目录下创建</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">classes</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">目录。</font>
2. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">在</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">classes</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">目录下创建</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">struts.xml</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">文件，内容如下：</font>

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE struts PUBLIC "-//Apache Software Foundation//DTD Struts Configuration 2.3//EN" "http://struts.apache.org/dtds/struts-2.3.dtd">
<struts>
  <package name="helloDemo" namespace="/" extends="struts-default">
    <action name="hello" class="action.HelloAction" method="execute">
      <result name="test">/testView.jsp</result>
    </action>
  </package>
</struts>
```
