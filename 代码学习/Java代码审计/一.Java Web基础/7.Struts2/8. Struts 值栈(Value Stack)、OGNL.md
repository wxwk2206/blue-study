## 一、值栈（Value Stack）
### （一）核心概念
值栈是 Struts2 提供的一种数据存储机制，替代了 Servlet 中域对象的部分功能。在 Action 中存入数据后，可在 JSP 页面通过 OGNL 表达式读取，是 Struts2 中数据传递的核心载体。

### （二）Action 与 Servlet 的核心区别

| **特性** | **Action** | **Servlet** |
| --- | --- | --- |
| 创建时机 | 每次访问时创建一个实例 | 第一次访问时创建，生命周期内仅创建一次 |
| 实例个数 | 多实例 | 单实例 |


### （三）生命周期
值栈与 Action 生命周期保持一致，随 request 的创建而创建，随 request 的销毁而销毁，即一个请求对应一个值栈。

### （四）获取相关对象
#### 1. ActionContext 对象获取
ActionContext 是 Struts2 的上下文对象，包含值栈、请求、会话等核心数据，获取方式分场景：

+ **Action 中获取**（推荐）：

```java
// 方式1：通过 ActionContext 直接获取
ActionContext actionContext1 = ActionContext.getContext();
// 方式2：通过 ServletActionContext 获取
ActionContext actionContext2 = ServletActionContext.getContext();
```

+ **拦截器（Interceptor）中获取**：

```java
public class MyInterceptor extends AbstractInterceptor {
    @Override
    public String intercept(ActionInvocation invocation) throws Exception {
        // 通过 ActionInvocation 获取
        ActionContext actionContext = invocation.getInvocationContext();
        return invocation.invoke();
    }
}
```

#### 2. ActionContext 常用方法

| **方法** | **功能描述** |
| --- | --- |
| `get(String key)` | 根据 key 从上下文获取值 |
| `put(String key, Object value)` | 向上下文存储键值对 |
| `getApplication()` | 返回 ServletContext 中的数据（Map 类型） |
| `getSession()` | 返回 HttpSession 中的数据（Map 类型） |
| `getParameters()` | 返回客户端提交的请求参数（Map 类型） |
| `getValueStack()` | 获取值栈对象（核心方法） |
| `getContextMap()` | 返回上下文存储的所有数据（Map 类型） |


#### 3. 值栈对象获取
值栈被包含在 ActionContext 中，获取方式与 ActionContext 场景对应：

+ **Action 中获取**：

```java
// 方式1：通过 ActionContext 获取（推荐）
ValueStack stack1 = ActionContext.getContext().getValueStack();
// 方式2：通过 ServletActionContext 获取
ValueStack stack2 = ServletActionContext.getValueStack();
```

+ **拦截器中获取**：

```java
ValueStack stack = invocation.getInvocationContext().getValueStack();
```

#### 4. 值栈唯一性
每个 Action 实例对应唯一值栈对象，可通过以下代码验证：

```java
public class TestAction extends ActionSupport {
    @Override
    public String execute() throws Exception {
        ValueStack stack1 = ActionContext.getContext().getValueStack();
        ValueStack stack2 = ActionContext.getContext().getValueStack();
        System.out.println(stack1 == stack2); // 输出 true
        return SUCCESS;
    }
}
```

### （五）值栈的基本使用
#### 1. 核心特性
+ 数据存储：支持存储任意对象，通过 OGNL 表达式或 Struts2 标签访问；
+ 查找规则：当多个对象存在同名属性时，按 “栈顶到栈底” 顺序查找第一个匹配对象。

#### 2. 值栈常用方法

| **方法** | **功能描述** |
| --- | --- |
| `findValue(String expr)` | 按表达式从值栈中获取数据（遵循栈顶优先） |
| `push(Object o)` | 将对象压入栈顶 |
| `pop()` | 取出栈顶对象并从栈中移除 |
| `peek()` | 获取栈顶对象，不移除 |
| `set(String key, Object o)` | 以键值对形式存储数据（可通过 key 检索） |
| `setValue(String expr, Object value)` | 按表达式向值栈设置数据 |
| `getRoot()` | 获取值栈的根容器（CompoundRoot） |
| `size()` | 获取值栈中对象的个数 |
| `setDefaultType(Class defaultType)` | 设置数据获取时的默认转换类型 |


#### 3. 向值栈存入数据（3 种方式）
+ **方式 1：使用 set 方法（键值对形式）**：

```java
public class ValueStack1Action extends ActionSupport {
    @Override
    public String execute() throws Exception {
        ValueStack stack = ActionContext.getContext().getValueStack();
        stack.set("test1", "testValue1"); // key: test1, value: testValue1
        return SUCCESS;
    }
}
```

+ **方式 2：使用 push 方法（压入对象）**：

```java
public class ValueStack2Action extends ActionSupport {
    @Override
    public String execute() throws Exception {
        Map<String, Object> data = new HashMap<>();
        data.put("key1", "This is key1");
        data.put("key2", "This is key2");
        ValueStack stack = ActionContext.getContext().getValueStack();
        stack.push(data); // 将 Map 对象压入栈顶
        return SUCCESS;
    }
}
```

+ **方式 3：通过 Action 类变量（推荐）**：

```java
public class ValueStack3Action extends ActionSupport {
    // 定义变量（必须提供 get 方法）
    private String url;
    // get 方法（不可或缺，用于 OGNL 访问）
    public String getUrl() {
        return url;
    }
    @Override
    public String execute() throws Exception {
        url = "www.baidu.com"; // 变量赋值
        return SUCCESS;
    }
}
```

#### 4. 从值栈读取数据（JSP 页面）
通过 Struts2 标签 + OGNL 表达式读取，核心规则：

+ 值栈中的对象：直接引用属性名；
+ ActionContext 中的对象（如 session、application）：用 `#` 前缀引用。

示例代码：

```html
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
  <%@ taglib prefix="s" uri="/struts-tags" %> <!-- 引入 Struts2 标签库 -->
    <html>
      <body>
        <!-- 读取值栈中通过 set 方法存入的数据 -->
        <div>test1: <s:property value="test1"/></div>
        <!-- 读取值栈中通过 push 方法存入的 Map 数据 -->
        <div>key1: <s:property value="key1"/></div>
        <div>key2: <s:property value="key2"/></div>
        <!-- 读取 Action 变量的数据 -->
        <div>url: <s:property value="url"/></div>
        <!-- 读取 ActionContext 中 session 的数据（需加 #） -->
        <div>session.username: <s:property value="#session.username"/></div>
      </body>
    </html>
```

## 二、OGNL（Object-Graph Navigation Language）
### （一）核心概念
对象图导航语言，是一种强大的表达式语言，用于操作值栈数据，支持数据引用、传输和类型转换，类似于 JSP 表达式语言，但功能更强大。

### （二）核心特性
+ 基于上下文：Struts2 中 OGNL 依赖 ActionContext 上下文，上下文包含根对象（值栈）和非根对象（session、request 等）；
+ 引用规则：
    - 根对象（值栈中的数据）：直接引用属性名（如 `value="name"`）；
    - 非根对象（ActionContext 中的数据）：用 `#` 前缀引用（如 `#session.username`）。

## 三、值栈 + OGNL 完整示例
### （一）步骤 1：编写 Action 类
```java
package action;
import com.opensymphony.xwork2.ActionContext;
import com.opensymphony.xwork2.ActionSupport;
import com.opensymphony.xwork2.util.ValueStack;
import java.util.HashMap;
import java.util.Map;

public class ValueStackAction extends ActionSupport {
    // 方式3：Action 变量（需 get 方法）
    private String url;
    public String getUrl() {
        return url;
    }

    @Override
    public String execute() throws Exception {
        ValueStack stack = ActionContext.getContext().getValueStack();

        // 方式1：set 方法存数据
        stack.set("test1", "testValue1");

        // 方式2：push 方法存 Map 数据
        Map<String, Object> data = new HashMap<>();
        data.put("key1", "This is key1");
        data.put("key2", "This is key2");
        stack.push(data);

        // 方式3：Action 变量赋值
        this.url = "www.baidu.com";

        // 向 session 存入数据（ActionContext 非根对象）
        Map<String, Object> session = ActionContext.getContext().getSession();
        session.put("username", "usernameTestValue");

        return "valueStackView";
    }
}
```

### （二）步骤 2：编写 JSP 页面（valueStackView.jsp）
```html
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
  <%@ taglib prefix="s" uri="/struts-tags" %>
    <html>
      <head>
        <title>值栈测试</title>
      </head>
      <body>
        <div>test1（set方法）: <s:property value="test1"/></div>
        <div>key1（push方法）: <s:property value="key1"/></div>
        <div>key2（push方法）: <s:property value="key2"/></div>
        <div>url（Action变量）: <s:property value="url"/></div>
        <div>session.username（非根对象）: <s:property value="#session.username"/></div>
      </body>
    </html>
```

### （三）步骤 3：配置 struts.xml
在 `WEB-INF/classes/struts.xml` 中添加 Action 配置：

```xml
<struts>
  <package name="valueStackDemo" namespace="/" extends="struts-default">
    <action name="valueStack" class="action.ValueStackAction">
      <!-- 结果映射：返回值对应 JSP 页面 -->
      <result name="valueStackView">/valueStackView.jsp</result>
    </action>
  </package>
</struts>
```

## 四、核心总结
1. **值栈**：Struts2 数据存储核心，与 request 生命周期绑定，每个 Action 对应唯一值栈；
2. **ActionContext**：上下文容器，封装值栈、session、request 等，是获取核心对象的入口；
3. **OGNL**：值栈数据访问工具，根对象（值栈）直接引用，非根对象（如 session）加 `#` 引用；
4. **数据存取**：存数据支持 set/push/Action 变量 3 种方式，读数据通过 `s:property` 标签 + OGNL 表达式
