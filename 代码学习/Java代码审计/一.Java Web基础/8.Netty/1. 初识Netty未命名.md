## 一、Netty 简介
1. 来源：由 JBOSS 提供的 Java 开源框架，现为 Github 上的独立项目。
2. 核心特性：提供异步的、事件驱动的网络应用程序框架和工具。
3. 核心用途：用于快速开发高性能、高可靠性的网络服务器和客户端程序，是基于 NIO 的客户端与服务器端编程框架。
4. 优势：简化和流线化网络应用的编程开发过程，例如基于 TCP 和 UDP 的 Socket 服务开发，能帮助开发者快速、简单地实现各类网络应用（如特定协议的客户端 / 服务端应用）。

## 二、Netty 的应用场景
1. 网络游戏服务器
    - 背景：Java 语言在手游服务端、大型网络游戏中应用广泛。
    - 优势：Netty 提供 TCP/UDP 和 HTTP 协议栈，便于定制和开发私有协议栈，是高性能的基础通信组件。
2. 通信行业
    - 核心优势：凭借异步高性能、高可靠性和高成熟度的特点，在通信行业得到大量应用。
3. 互联网应用（分布式系统）
    - 应用场景：分布式系统中各节点需远程服务调用，依赖高性能 RPC 框架。
    - 作用：Netty 作为异步高性能的通讯框架，常作为基础通讯组件被 RPC 框架采用。

## 三、安全人员学习 Netty 的原因
1. 应用普及性：Netty 框架在 Java 项目中应用较为广泛，安全人员大概率会在工作中接触到。
2. 实际工作需求：无需通读 Netty 源码，但需掌握基础使用方法，以便在项目中快速识别基于 Netty 开发的业务模块，进而通读业务代码开展代码审计和漏洞挖掘工作
