## <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">一、Bean 核心概念</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Bean 是构成 Spring 应用程序的核心组件，是被 Spring IOC 容器实例化、组装并管理的对象，其创建依据容器提供的配置元数据。</font>
