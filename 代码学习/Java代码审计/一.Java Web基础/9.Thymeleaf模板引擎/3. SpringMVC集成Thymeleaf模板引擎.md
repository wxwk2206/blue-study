## 一、测试环境搭建
### （一）基础环境准备
若未搭建 Spring MVC 环境，可参考教程：

[1.IDEA创建Spring MVC项⽬-详细过程](https://www.yuque.com/laziwuhui-k9vrd/qgg1ge/ccyfe7z7vlfr91yi)

### （二）修改 pom.xml 导入依赖
1. 路径：`./项目/pom.xml`（首次添加依赖报红需点击 Maven 按钮刷新，等待 IDEA 自动导入）
2. 在`<<dependencies></</dependencies>`标签中添加以下依赖（无该标签则手动创建）：

```xml
<!-- thymeleaf核心依赖 -->
<dependency>
  <groupId>org.thymeleaf</groupId>
  <artifactId>thymeleaf</artifactId>
  <version>3.0.11.RELEASE</version>
</dependency>

<!-- thymeleaf与Spring5集成依赖 -->
<dependency>
  <groupId>org.thymeleaf</groupId>
  <artifactId>thymeleaf-spring5</artifactId>
  <version>3.0.11.RELEASE</version>
</dependency>
```

### （三）修改 springmvc.xml 配置
1. 路径：`./项目/src/main/resources/springmvc.xml`
2. 若存在重复 id 的 bean 标签，需先注释，再添加以下配置：

```xml
<!-- 配置Thymeleaf视图解析器 -->
<bean id="viewResolver" class="org.thymeleaf.spring5.view.ThymeleafViewResolver">
  <!-- 关联模板引擎 -->
  <property name="templateEngine" ref="templateEngine"/>
  <!-- 设置字符集 -->
  <property name="characterEncoding" value="utf-8"/>
</bean>

<!-- 配置模板引擎 -->
<bean id="templateEngine" class="org.thymeleaf.spring5.SpringTemplateEngine">
  <!-- 关联模板解析器 -->
  <property name="templateResolver" ref="templateResolver"/>
</bean>

<!-- 配置模板解析器 -->
<bean id="templateResolver" class="org.thymeleaf.spring5.templateresolver.SpringResourceTemplateResolver">
  <!-- 配置模板前缀（视图文件存放路径） -->
  <property name="prefix" value="/WEB-INF/view/"/>
  <!-- 配置模板后缀（视图文件格式） -->
  <property name="suffix" value=".html"/>
  <!-- 设置模板字符集 -->
  <property name="characterEncoding" value="utf-8"/>
</bean>
```

## 二、案例实现
### （一）创建 Controller 控制器
1. 路径：`./项目/src/main/com/test/controller/ThymeleafController.java`
2. 代码实现：

```java
package test.controller;

import org.springframework.ui.Model;
import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;

@Controller
public class ThymeleafController {
    // 配置访问路径：/thymeleaf 和 /index
    @RequestMapping({"/thymeleaf", "/index"})
    public String index(Model model, String value) {
        // 向视图传递数据，key为message，value为请求参数中的value值
        model.addAttribute("message", value);
        // 返回视图名称（SpringMVC会自动拼接前缀和后缀：/WEB-INF/view/index.html）
        return "index";
    }
}
```

### （二）创建 Thymeleaf 视图
1. 路径：`./项目/src/main/webapp/WEB-INF/view/index.html`
2. 代码实现：

```html
<!DOCTYPE html>
<html lang="en" xmlns:th="http://www.thymeleaf.org">
  <head>
    <meta charset="UTF-8">
    <title>test</title>
  </head>
  <body>
    <!-- 定义页面片段header -->
    <div th:fragment="header">
      <h3>Spring MVC Web Thymeleaf示例视图</h3>
    </div>
    <!-- 定义页面片段main，通过th:text渲染后端传递的message数据 -->
    <div th:fragment="main">
      <span th:text="'Hello,' + ${message}"></span>
    </div>
  </body>
</html>
```

## 关键说明
1. Thymeleaf 命名空间：HTML 页面需引入`xmlns:th="http://www.thymeleaf.org"`才能使用 Thymeleaf 语法
2. 视图解析规则：SpringMVC 通过视图解析器自动拼接视图路径（前缀 + 视图名 + 后缀）
3. 数据传递：通过 Model 对象向视图传递数据，视图中使用`${key}`获取数据
4. 常用语法：`th:text`用于渲染文本内容，`th:fragment`用于定义可复用的页面片段
