##  修改pom.xml  
```xml
<!-- spring-boot热部署 -->
<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-devtools</artifactId>
</dependency>
```

## IDEA配置
1.打开设置>构建，执行，部署>编译器>自动执行项目
![[Pasted image 20260703180549.png]]

2.高级设置>搜索：自动>编译器打勾
![[Pasted image 20260703180543.png]]

 然后退出 IDEA 编辑器, 重启项⽬应⽤ 即可 

接下来修改任意内容以后 Ctrl+s 保存后,应⽤就会⾃动编译
