## 一、核心问题与解决方案
### 1. 核心问题
+ 直接使用 `new ClassPathXmlApplicationContext("Spring配置文件")` 获取上下文时，会导致配置文件多次加载、应用上下文对象重复创建，浪费资源。
+ Filter 运行在 ApplicationContext 之外，无法直接注入其他 Bean。

### 2. 解决方案
利用 `ServletContextListener` 监听器监听 Web 应用启动，在应用启动时加载 Spring 配置文件、创建 ApplicationContext 上下文对象，并存储到 ServletContext 域中，后续可在任意位置从该域获取上下文对象。

## 四、获取 ApplicationContext 的两种方法
### 方法一：通过 Spring 内置监听器实现
#### 0.思路逻辑
##### （1）核心思想
直接使用 Spring 框架提供的现成监听器和工具类，无需自定义组件，快速实现 ApplicationContext 的全局获取。

##### （2）涉及文件及作用
| **文件名** | **路径** | **核心作用** |
| --- | --- | --- |
| `web.xml` | `./src/main/webapp/WEB-INF/web.xml` | 1. 配置**全局初始化参数**：指定 Spring 配置文件路径（告诉监听器要加载哪个配置）；2. 配置**Spring 内置监听器**：触发 Spring 自动加载配置、创建 ApplicationContext 并存储到 ServletContext。 |
| `springmvc.xml` | `./src/main/resources/springmvc.xml` | Spring 核心配置文件：声明 Bean（如测试用的`helloWorldTest`），供 ApplicationContext 加载管理。 |
| `HelloWorld.java` | `./src/main/com/test/bean/HelloWorld.java` | 测试用 Bean：被 Spring 管理，用于验证 ApplicationContext 能否成功获取并调用 Bean。 |
| `BeanGetTest1Controller.java` | `./src/main/com/test/controller/BeanGetTest1Controller.java` | 测试控制器：通过 Spring 内置工具类从 ServletContext 中获取 ApplicationContext，调用 Bean 并返回结果，验证功能可用性。 |
| `WebApplicationContextUtils`（Spring 内置类） | Spring 框架自带（`org.springframework.web.context.support`包下） | 工具类：提供`getWebApplicationContext(ServletContext)`方法，从 ServletContext 域中获取 Spring 创建的 ApplicationContext。 |


##### （3）文件间的依赖关系
```plain
web.xml（配置参数+内置监听器）→ 启动时触发Spring内置监听器 → 加载springmvc.xml → 创建ApplicationContext → 存储到ServletContext
↓
BeanGetTest1Controller → 调用Spring内置WebApplicationContextUtils → 从ServletContext获取ApplicationContext → 调用springmvc.xml中声明的HelloWorld Bean
```

##### （4）关键流程
1. Web 应用启动，Tomcat 读取`web.xml`中的全局参数（`contextConfigLocation`）和内置监听器（`ContextLoaderListener`）；
2. 内置监听器`ContextLoaderListener`触发，根据参数指定的路径加载`springmvc.xml`，创建 ApplicationContext；
3. 监听器自动将 ApplicationContext 存储到 ServletContext 域中（键为 Spring 约定的固定值）；
4. 控制器`BeanGetTest1Controller`接收请求时，通过`HttpServletRequest`获取 ServletContext，再调用 Spring 内置工具类`WebApplicationContextUtils`提取域中的 ApplicationContext；
5. 用 ApplicationContext 获取`springmvc.xml`中配置的`helloWorldTest` Bean，调用其方法并返回结果。

#### 1. 配置 web.xml 文件
路径：`./src/main/webapp/WEB-INF/web.xml`，添加全局初始化参数和 Spring 内置监听器（放在`<display-name>`标签后）：

```xml
<!-- 全局初始化参数：指定Spring配置文件路径 -->
<context-param>
  <param-name>contextConfigLocation</param-name>
  <param-value>classpath:springmvc.xml</param-value>
</context-param>

<!-- 配置Spring内置监听器 -->
<listener>
  <listener-class>org.springframework.web.context.ContextLoaderListener</listener-class>
</listener>
```

#### 2. 创建测试控制器
路径：`./src/main/com/test/controller/BeanGetTest1Controller.java`

```java
package test.controller;

import org.springframework.context.ApplicationContext;
import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.ResponseBody;
import org.springframework.web.context.support.WebApplicationContextUtils;
import javax.servlet.ServletContext;
import javax.servlet.http.HttpServletRequest;
import test.bean.HelloWorld;

@Controller
@RequestMapping(value = "/beanGetTest1")
public class BeanGetTest1Controller {
    @RequestMapping(value = "/test")
    @ResponseBody
    public String test(HttpServletRequest request) {
        // 从ServletContext中获取ApplicationContext
        ServletContext servletContext = request.getSession().getServletContext();
        ApplicationContext app = WebApplicationContextUtils.getWebApplicationContext(servletContext);

        // 调用Bean
        HelloWorld helloWorld = (HelloWorld) app.getBean("helloWorldTest");
        return helloWorld.getMessage();
    }
}
```

#### 3. 测试结果
访问地址：`http://xxx.com/beanGetTest1/test`，返回结果：`Your Message:Hello World!`

### 方法二：自定义监听器实现
#### 0.思路逻辑
##### (1). 核心思想
不依赖 Spring 内置监听器，自己实现 “监听 Web 启动、加载配置、创建上下文、提供获取工具” 的全流程，灵活控制每个环节。

##### (2). 涉及文件及作用
| **文件名**                           | **路径**                                                       | **核心作用**                                                                                                                                                                                                                    |
| --------------------------------- | ------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `web.xml`                         | `./src/main/webapp/WEB-INF/web.xml`                          | 1. 配置**全局初始化参数**：与内置方式一致，指定 Spring 配置文件路径；2. 配置**自定义监听器**：替换 Spring 内置监听器，触发自定义的上下文创建逻辑。                                                                                                                                    |
| `springmvc.xml`                   | `./src/main/resources/springmvc.xml`                         | 与内置方式一致：声明测试 Bean`helloWorldTest`，供自定义监听器加载。                                                                                                                                                                                |
| `HelloWorld.java`                 | `./src/main/com/test/bean/HelloWorld.java`                   | 与内置方式一致：测试用 Bean，验证功能可用性。                                                                                                                                                                                                   |
| `ContextLoaderListener.java`      | `./src/main/com/test/listener/ContextLoaderListener.java`    | 自定义监听器：实现`ServletContextListener`接口，在 Web 应用启动时（`contextInitialized`方法）执行：① 读取`web.xml`中的全局参数；② 加载`springmvc.xml`创建 ApplicationContext；③ 将 ApplicationContext 存储到 ServletContext 域（键为自定义的`"app"`）。 |
| `WebApplicationContextUtils.java` | `./src/main/com/test/utils/WebApplicationContextUtils.java`  | 自定义工具类：提供静态方法`getWebApplicationContext(ServletContext)`，从 ServletContext 域中提取自定义存储的 ApplicationContext（根据键`"app"`获取）。                                                                                             |
| `BeanGetTest2Controller.java`     | `./src/main/com/test/controller/BeanGetTest2Controller.java` | 测试控制器：与内置方式逻辑一致，但调用**自定义工具类**获取 ApplicationContext，验证自定义流程的可用性。                                                                                                                                                             |


##### (3). 文件间的依赖关系
```plain
web.xml（配置参数+自定义监听器）→ 启动时触发自定义ContextLoaderListener → 读取参数→加载springmvc.xml→创建ApplicationContext→存储到ServletContext（键"app"）
↓
BeanGetTest2Controller → 调用自定义WebApplicationContextUtils → 从ServletContext（键"app"）获取ApplicationContext → 调用springmvc.xml中声明的HelloWorld Bean
```

##### (4). 关键流程
1. Web 应用启动，Tomcat 读取`web.xml`中的全局参数和自定义监听器`ContextLoaderListener`；
2. 自定义监听器的`contextInitialized`方法执行：读取`contextConfigLocation`参数，通过`new ClassPathXmlApplicationContext`加载`springmvc.xml`，创建 ApplicationContext；
3. 监听器手动将 ApplicationContext 以自定义键`"app"`存储到 ServletContext 域中；
4. 控制器`BeanGetTest2Controller`接收请求时，通过`HttpServletRequest`获取 ServletContext，调用**自定义工具类**`WebApplicationContextUtils`，根据键`"app"`提取域中的 ApplicationContext；
5. 后续获取 Bean、返回结果的逻辑与内置方式一致。

#### 1. 创建自定义监听器
路径：`./src/main/com/test/listener/ContextLoaderListener.java`

```java
package test.listener;

import org.springframework.context.ApplicationContext;
import org.springframework.context.support.ClassPathXmlApplicationContext;
import javax.servlet.ServletContext;
import javax.servlet.ServletContextEvent;
import javax.servlet.ServletContextListener;

public class ContextLoaderListener implements ServletContextListener {
    @Override
    public void contextInitialized(ServletContextEvent servletContextEvent) {
        // 获取ServletContext对象
        ServletContext servletContext = servletContextEvent.getServletContext();

        // 读取web.xml中的全局参数（Spring配置文件路径）
        String contextConfigLocation = servletContext.getInitParameter("contextConfigLocation");

        // 创建ApplicationContext对象
        ApplicationContext app = new ClassPathXmlApplicationContext(contextConfigLocation);

        // 将上下文对象存储到ServletContext域中
        servletContext.setAttribute("app", app);

        System.out.println("初始化Init Listener");
    }

    @Override
    public void contextDestroyed(ServletContextEvent servletContextEvent) {
        // 销毁时无需额外操作
    }
}
```

#### 2. 创建自定义工具类
路径：`./src/main/com/test/utils/WebApplicationContextUtils.java`

```java
package test.utils;

import org.springframework.context.ApplicationContext;
import javax.servlet.ServletContext;

public class WebApplicationContextUtils {
    // 从ServletContext中获取自定义存储的ApplicationContext
    public static ApplicationContext getWebApplicationContext(ServletContext servletContext) {
        return (ApplicationContext) servletContext.getAttribute("app");
    }
}
```

#### 3. 配置 web.xml 文件
路径：`./src/main/webapp/WEB-INF/web.xml`，替换为自定义监听器：

```xml
<!-- 全局初始化参数（与方法一一致） -->
<context-param>
  <param-name>contextConfigLocation</param-name>
  <param-value>classpath:springmvc.xml</param-value>
</context-param>

<!-- 配置自定义监听器 -->
<listener>
  <listener-class>test.listener.ContextLoaderListener</listener-class>
</listener>
```

#### 4. 创建测试控制器
路径：`./src/main/com/test/controller/BeanGetTest2Controller.java`

```java
package test.controller;

import org.springframework.context.ApplicationContext;
import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.ResponseBody;
import javax.servlet.ServletContext;
import javax.servlet.http.HttpServletRequest;
import test.bean.HelloWorld;
import test.utils.WebApplicationContextUtils;

@Controller
@RequestMapping(value = "/beanGetTest2")
public class BeanGetTest2Controller {
    @RequestMapping(value = "/test")
    @ResponseBody
    public String test(HttpServletRequest request) {
        // 从ServletContext中获取自定义存储的ApplicationContext
        ServletContext servletContext = request.getSession().getServletContext();
        ApplicationContext app = WebApplicationContextUtils.getWebApplicationContext(servletContext);

        // 调用Bean
        HelloWorld helloWorld = (HelloWorld) app.getBean("helloWorldTest");
        return helloWorld.getMessage();
    }
}
```

#### 5. 测试结果
访问地址：`http://xxx.com/beanGetTest2/test`，返回结果：`Your Message:Hello World!`
