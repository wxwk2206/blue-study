# 1.@Controller与@RestController的共同点
```java
都是用来表示SpringMVC某个类的是否可以接收HTTP请求
```

# 2.@Controller与@RestController的不同点
```plain
@Controller: 是SpringMVC的Controller处理器
@Controller类中的方法可以直接通过返回String跳转到jsp、html等模版页面,在方法上加@ResponseBody注解,也可以返回实体对象

@RestController: 是SpringMVC的@Controller加@ResponseBody的结合体,也就是两个标注合并起来的作用
@RestController类中的所有方法只能返回String、Object、Json等实体对象,不能跳转到模版页面
```
