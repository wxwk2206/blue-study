# 1.前言
```plain
在日常使用中,页面传递的参数都是String,而在控制器中接收参数类型是不确定的
对于基础数据类型,SpringMVC已经提供了类型转换器
对于不支持的目标类型,例如日期类型,自定义的对象类型
则可以通过继承org.springframework.core.convert.converter.Converter接口的方法来实现
```

```plain
该接口定义如下:
@FunctionalInterface
public interface Converter<S, T> {
    @Nullable
    T convert(S source);
  	...
}

其中S表示源类型,T表示目标类型

更简单的说法可以如下理解:
Converter<S, T>,中的S代表的是页面参数的类型,一般是String,T代表控制器中接收参数的类型
Converter<S, T>接口的,convert(S source)方法,就是我们要实现的从S转换为T的方法
```

# 3.例子
这里编写一个通过字符串向自定义对象进行类型转换的例子

```java
// 第一步
// 创建People实体类,用于类型转换使用
// 路径: ./项目/src/main/com/test/model/People.java
package test.model;

public class People {
    private String name;
    private Integer age;
    private Double height;

    public void setName(String name) {
        this.name = name;
    }

    public String getName() {
        return name;
    }

    public void setAge(Integer age) {
        this.age = age;
    }

    public Integer getAge() {
        return age;
    }

    public void setHeight(Double height) {
        this.height = height;
    }

    public Double getHeight() {
        return height;
    }
}
```

```java
// 第二步
// 创建自定义类型转换器,PeopleConverter,用于将前台传输的数据进行转换与赋值返回
// 路径: ./项目/src/main/com/test/converter/PeopleConverter.java
package test.converter;

import org.springframework.core.convert.converter.Converter;
import org.springframework.stereotype.Component;
import test.model.People;

@Component
public class PeopleConverter implements Converter<String, People> {
    @Override
    public People convert(String source) {
        People people = new People();
        // 以","分隔数据
        String stringValues[] = source.split(",");
        if (stringValues.length == 3) {
            // 为user实例赋值
            people.setName(stringValues[0]);
            people.setAge(Integer.parseInt(stringValues[1]));
            people.setHeight(Double.parseDouble(stringValues[2]));
            return people;
        } else {
            throw new IllegalArgumentException(String.format("类型转换失败,标准格式:'小王,18,1.7',但格式是[%s] ", source));
        }
    }
}
```

```xml
<!-- 第三步 -->
<!-- 在建好的项目下找到springmvc.xml文件并打开 -->
<!-- 路径: ./项目/src/main/resources/springmvc.xml -->
<!-- 在<beans></beans>标签中添加如下内容: -->

<!-- 自定义类型转换器配置 -->
<!-- 第一步,将peopleConverter自定义类型转换器,纳入SpringIOC容器 -->
<bean id="peopleConverter" class="test.converter.PeopleConverter"></bean>
<!-- 第二步,peopleConverter再纳入SpringMVC提供的转换器Bean -->
<bean id="conversionService" class="org.springframework.context.support.ConversionServiceFactoryBean">
  <property name="converters">
    <set>
      <ref bean="peopleConverter"/>
    </set>
  </property>
</bean>
<!-- 第三步,将conversionService注册到annotation-driven中 -->
<!-- 此配置是SpringMVC的基础配置,许多功能都需要通过该注解来协调 -->
<mvc:annotation-driven conversion-service="conversionService"></mvc:annotation-driven>
```

```java
// 第四步
// 创建控制器进行测试
package test.controller;

import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RequestParam;
import org.springframework.web.bind.annotation.RestController;

import test.model.People;

@RestController
public class ConverterTestController {
    @RequestMapping(value = "/converterTest")
    public People converterTest(@RequestParam("people") People people) {
        return people;
    }
}





访问方式: http://xxx.com/converterTest?people=小刚,18,1.7
返回: {"name": "小刚", "age": 18, "height": 1.7}
```
