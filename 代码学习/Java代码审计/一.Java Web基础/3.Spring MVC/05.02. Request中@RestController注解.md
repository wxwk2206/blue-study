# 1.前言
```plain
@RestController注解,它将类标记为控制器,其中每个方法返回一个对象而不是视图

@RestController注解相当于@Controller+@ResponseBody合在一起的作用
相当于一个便捷式的注解效果等同于在类上同时添加上述两个注解

@RestController类中的所有方法只能返回String、Object、Json等实体对象,不能跳转到模版页面
```

# 2.例子
```java
// 路径: ./项目/src/main/com/test/controller/Test13Controller.java
package test.controller;

import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class Test13Controller {
    @RequestMapping(value = "/helloTest")
    public String helloTest() {
        return "helloTest";
    }
}





访问方式: http://xxx.com/helloTest
运行结果: helloTest
```
