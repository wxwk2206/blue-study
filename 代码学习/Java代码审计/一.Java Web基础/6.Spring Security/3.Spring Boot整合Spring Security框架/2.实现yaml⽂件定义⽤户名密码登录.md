## <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">一、前言</font>
1. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">前置基础：已完成 Spring Security 环境搭建并成功启动</font>
2. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">核心需求：解决默认账号密码不可控的问题，通过 yaml 文件自定义登录的用户名和密码</font>

## <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">二、实现步骤（yaml 文件定义用户名密码）</font>
### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">1. 找到配置文件路径</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">项目中配置文件位置：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">./项目/src/main/resources/application.yml</font>`

### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">2. 添加配置内容</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">在 application.yml 文件中添加以下配置：</font>

```yaml
spring:
  security:
    user:
      name: admin  # 自定义用户名
      password: 123456  # 自定义密码
```

### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">3. 生效说明</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">配置后重启项目，即可使用自定义的用户名（admin）和密码（123456）登录</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">默认账号（user）会失效，控制台不再输出 user 对应的随机密码</font>
