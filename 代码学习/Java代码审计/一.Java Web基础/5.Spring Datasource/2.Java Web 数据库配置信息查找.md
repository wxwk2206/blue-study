## 一、配置文件位置查找
### （一）传统 Web 应用
+ 核心目录：WEB-INF 目录 或 WEB-INF/classes 目录
+ 支持文件类型：_.properties、_.yml、*.xml

### （二）SpringBoot 单 jar 应用
+ 前提操作：需先解压缩 jar 包
+ 核心目录：classes 目录 或 BOOT-INF/classes 目录
+ 支持文件类型：_.properties、_.yml

### （三）常见配置文件路径及名称
1. xml 格式：db.xml、applicationContext.xml、hibernate.cfg.xml、c3p0-config.xml、server.xml
2. properties 格式：jdbc.properties、db.properties、application.properties、datasource.properties、dbconfig.properties
3. yml 格式：application.yml、db.yml
4. 具体路径示例：

```plain
./WEB-INF/db.xml
./WEB-INF/applicationContext.xml
./WEB-INF/hibernate.cfg.xml
./WEB-INF/jdbc.properties
./WEB-INF/jdbc/jdbc.properties
./WEB-INF/classes/db.xml
./WEB-INF/classes/dbconfig.properties
./resources/db.yml
./resources/db.properties
./resources/application.yml
./resources/application.properties
```

## 二、常见驱动程序类名称
+ 驱动是依赖的一种：驱动是具体JDBC实现（如mysql-connector-java）,依赖是泛指所有需要的jar包

| **驱动程序类名称** | **备注说明** |
| --- | --- |
| com.mysql.jdbc.Driver | 适用于 mysql-connector-java 5 版本 |
| com.mysql.cj.jdbc.Driver | 适用于 mysql-connector-java 6 版本 |
| oracle.jdbc.driver.OracleDriver | Oracle 数据库专用驱动类 |
| com.microsoft.sqlserver.jdbc.SQLServerDriver | 适用于 SQL Server 2005 和 2008 的 JDBC 驱动 |
| com.microsoft.jdbc.sqlserver.SQLServerDriver | 适用于 SQL Server 2000 的 JDBC 驱动 |
| dm.jdbc.driver.DmDriver | 达梦数据库（国产数据库）专用驱动类 |


## 三、Linux 系统快速查找方法
### （一）注意事项
该方法**无法查找 SpringBoot 单 jar 应用项目**的数据库配置（重要提示重复多次，需重点关注）

### （二）查找语法
```plain
find "[路径]" -type f | xargs grep "[驱动程序类名称]"
```

### （三）示例
+ 项目路径：springBootDemo（注：此处示例路径名称与注意事项不冲突，仅为演示语法）
+ 目标驱动类：com.mysql.jdbc.Driver
+ 执行命令：plaintext

```plain
find "SpringBootDemo" -type f | xargs grep "com.mysql.jdbc.Driver"
```

## 四、Windows 系统快速查找方法
### （一）注意事项
该方法**无法查找 SpringBoot 单 jar 应用项目**的数据库配置（重要提示重复多次，需重点关注）

### （二）查找语法
```plain
findstr /s /n "[驱动程序类名称]" "[路径]"
```

### （三）示例
+ 项目路径：SpringBootDemo
+ 目标驱动类：com.mysql.jdbc.Driver
+ 执行命令：

```plain
findstr /s /n "com.mysql.jdbc.Driver" "SpringBootDemo\*"
```

+ 命令行执行示例：

```plain
C:\Usrs\miao>findstr /s /n "com.mysql.jdbc.Driver" "C:\Users\miao\Desktop\SpringBootDemo\*"
C:\Users\miao\Desktop\SpringBootDemo\.idea\datasources.xml:7 <jdbc-driver>com.mysql.jdbc.Driver</jdbc-driver>
C:\Users\miao\Desktop\SpringBootDemo\target\classes\application.yml:39 driver-class-name com.mysql.jdbc.Driver
```
