## 一、EL 表达式基础
1. **定义**：JSP Expression Language，JSP2.0 新增功能，用于简化 JSP 中的表达式编写，让 JSP 代码更简洁，包含内置对象、运算符、保留关键字。
2. **核心语法**：`${表达式}`
3. **停用方法**：通过`page`指令设置`isELIgnored`属性，语法`<%@ page isELIgnored ="true|false" %>`
    - 停用：`<%@ page isELIgnored ="true" %>`
    - 启用：`<%@ page isELIgnored ="false" %>`（默认值，不写该指令则默认启用）
    - 注意：该指令可放在 JSP 页面任意位置。

## 二、EL 基础操作符
EL 支持大部分 Java 算术、逻辑操作符，包含访问符、运算符、比较符、逻辑符等，部分操作符有两种写法：

| **类型** | **操作符**           | **描述**            |
| ------ | ----------------- | ----------------- |
| 访问符    | `.`               | 访问 Bean 属性 / 映射条目 |
| 访问符    | `[]`              | 访问数组 / 链表元素       |
| 优先级符   | `()`              | 组织子表达式，修改优先级      |
| 算术符    | `+、-、*、div、%/mod` | 加、减 / 负、乘、除、取模    |
| 比较符    | `==/eq、!=/ne`     | 等于、不等于            |
| 比较符    | `< /lt、> /gt`     | 小于、大于             |
| 比较符    | `<=/le、>=/ge`     | 小于等于、大于等于         |
| 逻辑符    | `&&/and、/or`      | 逻辑与、逻辑或           |
| 逻辑符    | `!/not`           | 取反                |
| 空值判断   | `empty`           | 测试是否为空值           |


## 三、EL 运算符优先级（由高到低）
1. `[]` 、 `.`
2. `()`
3. `-`(负) 、 `empty`
4. `*` 、 `div` 、 `mod`
5. `+` 、 `-`(减)
6. `<` 、 `>` 、 `lt` 、 `gt` 、 `le` 、 `ge`
7. `==` 、 `!=`
8. `&&` 、 `and`
9. `||` 、 `or`
10. 三目运算`${A?B:C}`

## 四、EL 内置对象（11 个）
EL 内置对象用于快速获取不同范围、不同类型的数据，无需手动创建，核心分类及作用如下：

| **内置对象** | **说明** | |
| --- | --- | --- |
| 域对象 | `pageScope` | 从 page 范围获取值 |
| 域对象 | `requestScope` | 从 request 范围获取值 |
| 域对象 | `sessionScope` | 从 session 范围获取值 |
| 域对象 | `applicationScope` | 从 application 范围获取值 |
| 请求参数对象 | `param` | 获取请求单个参数值 |
| 请求参数对象 | `paramValues` | 获取请求多个参数值（返回集合） |
| 请求头对象 | `header` | 获取请求头单个值 |
| 请求头对象 | `headerValues` | 获取请求头多个值（返回集合） |
| Cookie 对象 | `cookie` | 获取请求中的 Cookie 数据 |
| 初始化参数对象 | `initParam` | 获取 Web 初始化参数 |
| 页面上下文对象 | `pageContext` | 通用于 page/request/session/application 域，可获取请求、会话等信息 |


## 五、EL 保留关键字（不可用作命名）
lt、le、gt、ge、eq、ne、true、false、and、or、not、instanceof、div、mod、empty、null

## 六、EL 核心使用示例
### 1. 读取请求参数
```plain
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<html>
<body>欢迎你, ${param.name}</body>
</html>
```

访问地址：`http://127.0.0.1:8081/项目名/页面.jsp?name=测试`，页面显示：`欢迎你,测试`

### 2. 停用 EL 表达式
```plain
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<%@ page isELIgnored ="true" %>
<html>
<body>不解析EL, ${param.name}</body>
</html>
```

页面直接显示：`不解析EL, ${param.name}`，不做表达式解析

### 3. 运算符使用
```plain
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<html>
<body>(1+1)*2=${(1+1)*2}</body>
</html>
```

页面显示：`(1+1)*2=4`

### 4. 读取 Session 域数据
```plain
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<%
HttpSession s = request.getSession();
s.setAttribute("sk", "test");
%>
<html>
<body>你好, ${sessionScope.get("sk")}</body>
</html>
```

页面显示：`你好,test`

### 5. pageContext 对象使用（获取请求 / 会话信息）
```plain
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<html>
<body>
<div>协议: ${pageContext.request.scheme}</div>
<div>服务器IP: ${pageContext.request.serverName}</div>
<div>服务器端口: ${pageContext.request.serverPort}</div>
<div>工程路径: ${pageContext.request.contextPath}</div>
<div>会话ID: ${pageContext.session.id}</div>
</body>
</html>
```

可获取 http 协议、127.0.0.1、8081、项目路径、会话编号等信息

### 6. 获取 JavaBean 内容
**步骤 1：创建 JavaBean**

```plain
package 包名;
public class Person {
    private String name;
    private int age;
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    public String getName() {return name;}
    public int getAge() {return age;}
}
```

**步骤 2：JSP 中调用**

```plain
<%@ page import="包名.Person" %>
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<%
Person p = new Person("test", 18);
pageContext.setAttribute("person", p);
%>
<html>
<body>
<div>名称: ${person.name}</div>
<div>年龄: ${person["age"]}</div>
</body>
</html>
```

页面显示：`名称:test`、`年龄:18`（`.`和`[]`均可访问 Bean 属性）

### 7. 读取不同域对象数据
```
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
  <%
    application.setAttribute("testName", "aa");
    session.setAttribute("testName", "bb");
    request.setAttribute("testName", "cc");
    pageContext.setAttribute("testName", "dd");
    %>
    <html>
      <body>
        <div>application: ${applicationScope.testName}</div>
        <div>session: ${sessionScope.testName}</div>
        <div>request: ${requestScope.testName}</div>
        <div>page: ${pageScope.testName}</div>
      </body>
    </html>
```

页面依次显示：`aa、bb、cc、dd`，可精准获取指定域的属性值
