##  前⾔  
 JSP全称Java Server Pages,是⼀种动态⽹⻚开发技术

 它使⽤JSP标签在HTML⽹⻚中插⼊Java代码,常⽤标签是 <%java 代码 %> 



JSP是为了简化Servlet的处理流程⽽出现的替代品 

早期的Java EE因为只能使⽤Servlet来处理客户端请求⽽显得⾮常的繁琐和不便 

于是就诞⽣了JSP⽅便快速的完成后端逻辑请求 



从本质上说 JSP 就是⼀个Servlet ,因为JSP⽂件最终会被编译成class⽂件 

⽽这个class⽂件实际上就是⼀个特殊的Servlet 

需要注意的是,是先出现了Servlet然后才出现了JSP的  

##  JSP脚本标签  
脚本元素是JSP中使⽤最频繁的元素,脚本程序可以包含任意量的Java语句、变量、⽅法或表达式

###  JSP声明语句  
```html
作⽤: ⽤于声明⽅法、属性、全局变量, 其中写的内容会直接编译在Servlet类中
语法格式: 
<%!
  常量
  静态变量
  实例变量
  静态代码块
  实例代码块
  构造函数
  实例⽅法
  静态⽅法
  %>
  或者,您也可以编写与其等价的XML语句:
  <jsp:declaration>
    代码⽚段
  </jsp:declaration>
```

```html
<%!
  int sum = 1;
  String name = "name";
  public String printName() {
  return name + "ooooo";
  }
  %>
  或是
  <jsp:declaration>
    int sum = 1;
    String name = "name";
    Plain Text
    public String printName() {
    return name + "ooooo";
    }
  </jsp:declaration>
```

###  JSP表达式  
```plain
作⽤: ⽤于将已经声明的变量或者表达式输出到⽹⻚上⾯
语法格式: 
<%= 表达式 %>

或者,您也可以编写与其等价的XML语句:
<jsp:expression>
表达式
</jsp:expression>
```

```plain
// 输出今⽇时间
<%= (new java.util.Date()).toLocaleString() %>
或是
<jsp:expression>
   (new java.util.Date()).toLocaleString()
</jsp:expression>
```

###  JSP脚本⽚段  
```plain
作⽤: JSP脚本⽚段(JSP Scriptlet),常常⽤于在Service⽅法中定义局部变量或者调⽤其他⽅法
语法格式:
<% Java代码 %>

或者,您也可以编写与其等价的XML语句:
<jsp:scriptlet>
代码⽚段
</jsp:scriptlet>
```

```plain
<%out.println("Your IP address is " + request.getRemoteAddr());%>
或是
<jsp:scriptlet>
  out.println("Your IP address is " + request.getRemoteAddr());
</jsp:scriptlet>
```

##  JSP注释  
```plain
1. JAVA注释_1
作⽤: 单⾏注释
语法格式: //注释内容
2. JAVA注释_2
作⽤: 多⾏注释
语法格: /*注释内容*/
3. JSP注释
作⽤: JSP注释
语法格式: <%-注释内容--%>
```

```plain
// ⽂件名test3.jsp
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<%
  // JAVA注释
  // int name = "laowang";
  /* System.out.print(name); */
%>
<%-
  int name = "xiaowang";
  System.out.print(name);
--%>
// 执⾏test3.jsp
// 执⾏结果,啥也没有,因为被代码被注释了
```
