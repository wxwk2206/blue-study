##  什么是监听器  
 监听器是 Servlet 规范中定义的⼀种特殊类 ，⽤于监听 ServletContext 、 HttpSession 和 ServletRequest 等域对象的创建和销毁事件 ，它还可以监听域对象的属性发⽣修改的事件 , 可以在事件发⽣前或者发⽣后做⼀些必要的处理  

 在 Servlet 中要创建监听器类⾸先需要新建⼀个类并继承相应的监听器接⼝ ，

实现接⼝中定义的⽅法 , 然后在 web.xml ⽂件中注册相应的监听器即可 ：

如果⼀个 web.xml ⽂件中注册了**多个监听器** , 则监听器的启动顺序按照在 web.xml 中的注册顺序启动 ；如果⼀个 web.xml ⽂件中**同时定义了监听器、过滤器、 Servlet **，那么 WEB 容器 , 会先加载监听器 , 再加载过滤器 , 最后加载 Servlet  

##  ServletContextListener  
```arkts
监听ServletContext域对象的创建与销毁:
继承ServletContextListener
并实现该接⼝中的contextInitialized和contextDestroyed⽅法

ServletContext域对象的⽣命周期:
创建:启动服务器时创建
销毁:关闭服务器或者从服务器移除项⽬

作⽤:利⽤ServletContextListener监听器在创建ServletContext域对象时
完成⼀些想要初始化的⼯作或者执⾏⾃定义任务调度
```

```java
// 第⼀步
// 创建ServletContextListener类,测试接⼝
// 路径: ./项⽬/src/main/webapp/com/Listener/MyServletContextListener.java
package com.Listener;

import javax.servlet.ServletContextEvent;
import javax.servlet.ServletContextListener;
public class MyServletContextListener implements ServletContextListener {
//启动服务器创建应⽤程序上下⽂时执⾏
    @Override
    public void contextInitialized(ServletContextEvent sce) {
//获取服务器启动的初始参数
// web.xml⽂件中配置的
        String name = sce.getServletContext().getInitParameter("name");
        System.out.println("MyServletContextListener 类的 contextInitialized⽅法, 执⾏了,name 参数的值为:" + name);
    }
//关闭服务器销毁程序上下⽂时执⾏
    @Override
    public void contextDestroyed(ServletContextEvent sce) {
//获取服务器启动的初始参数
// web.xml⽂件中配置的
        String name = sce.getServletContext().getInitParameter("name");
        System.out.println("MyServletContextListener 类的 contextDestroyed⽅法 , 执⾏了 ,name 参数的值为:" + name);
    }
}
```

##  ServletRequestListener  
```arkts
监听ServletRequest域对象的创建与销毁：实现ServletRequestListener接⼝
继承ServletRequestListener,并实现该接⼝中的requestInitialized和requestDestroyed方法

ServletRequest域对象的⽣命周期:
创建:访问服务器任何资源都会发送请求(ServletRequest)出现,访问.html和.jsp和.servlet都会创建请求
销毁:服务器已经对该次请求做出了响应
```

```java
// 第⼀步
// 创建ServletRequestListener类,测试接⼝
// 路径: ./项⽬/src/main/webapp/com/Listener/MyServletRequestListener.java
package com.Listener;

import javax.servlet.ServletRequestEvent;
import javax.servlet.ServletRequestListener;

public class MyServletRequestListener implements ServletRequestListener {
    @Override
    public void requestInitialized(ServletRequestEvent sre) {
        System.out.println("MyServletRequestListener 类的 requestInitialized⽅法 , 执⾏了 ,nameTest 参数的值为:" + sre.getServletRequest().getParameter("nameTest"));
    }
    @Override
    public void requestDestroyed(ServletRequestEvent sre) {
        System.out.println("MyServletRequestListener 类的 requestDestroyed⽅法，执⾏了,nameTest 参数的值为:" + sre.getServletRequest().getParameter("nameTest"));
}
}
```
