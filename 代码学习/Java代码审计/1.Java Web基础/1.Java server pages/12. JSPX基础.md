## 一、JSP 与 JSPX 的核心区别
JSPX 是采用 XML 语法规范书写的 JSP 文件，本质上是 JSP 的 XML 格式变体，语法更严谨，遵循 XML 的结构要求。

## 二、核心语法对照

| **JSP 语法** | **JSPX 语法** | **说明** |
| --- | --- | --- |
| `<%@ page ... %>` | `<jsp:directive.page ... />` | 页面指令，用于设置页面属性（如编码、语言等） |
| `<%@ page import="..." %>` | `<jsp:directive.page import="..."/>` | 导入 Java 类的指令 |
| `<%@ include ... %>` | `<jsp:directive.include file="..." />` | 静态包含指令，引入其他文件内容 |
| `<%@ taglib ... %>` | `<元素名 xmlns:prefix="URL">` | 标签库引入指令，通过 XML 命名空间绑定标签库 URL |
| `<%! 代码片段 %>` | `<jsp:declaration> 代码片段 </jsp:declaration>` | 声明全局变量或方法 |
| `<% java代码 %>` | `<jsp:scriptlet> java代码 </jsp:scriptlet>` | 嵌入 Java 执行代码 |
| `<%= 表达式 %>` | `<jsp:expression> 表达式 </jsp:expression>` | 输出表达式结果到页面 |
| `<%--注释内容--%>` | `<!-- 注释内容 -->` | 注释格式，JSPX 直接使用 XML 注释 |


## 三、关键注意事项
JSPX 遵循 XML 语法，部分特殊字符需进行 HTML 实体编码，否则会导致编译报错：

+ `<` 需替换为 `&lt;`
+ `&` 需替换为 `&amp;`
+ `>` 需替换为 `&gt;`

**示例**：

```xml
<jsp:scriptlet>
  if (t != null &amp;&amp; t.isEmpty()) { 
  String t = "a123";
  out.println("1234");
  }
</jsp:scriptlet>
```

## 四、实战示例
### 1. 基础示例（1.jspx）
+ 路径：`./src/main/webapp/1.jspx`
+ 代码：

```xml
<jsp:root xmlns:jsp="http://java.sun.com/JSP/Page"
  xmlns="http://www.w3.org/1999/xhtml"
  version="2.0">
  gogogogogo
</jsp:root>
```

### 2. 完整功能示例（2.jspx）
+ 路径：`./src/main/webapp/2.jspx`
+ 代码：

```xml
<jsp:root xmlns:jsp="http://java.sun.com/JSP/Page"
  xmlns="http://www.w3.org/1999/xhtml"
  xmlns:prefix="http://java.sun.com/jsp/jstl/core">
  <!-- 页面指令：设置编码和语言 -->
  <jsp:directive.page contentType="text/html;charset=UTF-8" language="java" version="2.0"/>
  <!-- 静态包含1.jspx -->
  <jsp:directive.include file="/1.jspx"/>
  <!-- 声明全局变量 -->
  <jsp:declaration>
    private static final String S = "hello";
  </jsp:declaration>
  <!-- 嵌入Java代码：获取请求参数并输出 -->
  <jsp:scriptlet>
    out.println(S + "," + request.getParameter("name"));
  </jsp:scriptlet>
  <!-- 输出表达式结果 -->
  <jsp:expression>"这是一段文字"</jsp:expression>
  <!-- XML注释 -->
  <!-- 这是注释 -->
</jsp:root>
```

### 3. 访问与结果
+ 访问地址：`http://127.0.0.1:8081/mavenJspTest_war/2.jspx?name=test`
+ 页面输出：`gogogogogo hello,test 这是一段文字`
+ 核心功能：静态包含文件、声明变量、获取请求参数、输出表达式结果
