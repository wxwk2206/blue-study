#### 一、REST 简介
1. **定义**：REST（Representational State Transfer）是一种用于网络架构的软件设计模式，也是目前最流行的互联网软件架构。
2. **核心特点**：结构清晰、符合标准、易于理解、扩展方便，被广泛采用。
3. **URL 对比**：
    - 传统请求 URL：`http://xxx.com/user/detail?id=1`（通过参数传递资源标识）
    - REST 风格 URL：`http://xxx.com/users/detail/1`（资源标识直接包含在路径中）

#### 二、URL 风格与请求方法
REST 风格核心通过**4 种请求方法**区分操作类型，对应 URL 及使用说明如下：

| **URL 地址** | **请求方式** | **body 数据** | **操作解释** |
| --- | --- | --- | --- |
| `http://xxx.com/users` | HTTP GET | 无 | 查询所有用户信息 |
| `http://xxx.com/users/1` | HTTP GET | 无 | 查询 ID 为 1 的指定用户信息 |
| `http://xxx.com/users` | HTTP POST | `user=admin&pass=123456` | 添加用户信息 |
| `http://xxx.com/users` | HTTP PUT | `id=1&pass=123456` | 修改 ID 为 1 的用户信息 |
| `http://xxx.com/users/1` | HTTP DELETE | 无 | 删除 ID 为 1 的指定用户信息 |

#### 四、实践例子（Controller 实现）
1. **代码位置**：`./项目/src/main/com/test/controller/UserTestController.java`
2. **核心注解**：
    - `@RestController`：标识该类为 REST 风格控制器
    - `@RequestMapping`：定义路由路径及请求方法，属性包括`value`（路径）和`method`（请求方式）
    - `@PathVariable`：获取 URL 路径中的参数（如资源 ID）
3. **完整代码**

```java
package test.controller;
import org.springframework.web.bind.annotation.RestController;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RequestMethod;
import org.springframework.web.bind.annotation.PathVariable;
@RestController
public class UserTestController {
    @RequestMapping(value = "/userTest/{id}", method = RequestMethod.GET)
    public String testRestGet(@PathVariable Integer id) {
        return "test rest GET:" + id;
    }

    @RequestMapping(value = "/userTest/{id}", method = RequestMethod.POST)
    public String testRestPost(@PathVariable Integer id) {
        return "test rest POST:" + id;
    }

    @RequestMapping(value = "/userTest/{id}", method = RequestMethod.DELETE)
    public String testRestDelete(@PathVariable Integer id) {
        return "test rest DELETE:" + id;
    }

    @RequestMapping(value = "/userTest/{id}", method = RequestMethod.PUT)
    public String testRestPUt(@PathVariable Integer id) {
        return "test rest PUT:" + id;
    }
}
```

| PUT 请求 | `http://xxx.com/userTest/1` | `test rest PUT:1` |
