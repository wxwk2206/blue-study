## <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">一、核心简介</font>
1. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Spring MVC 中定义控制器无需继承特定类或实现接口，核心依赖两个注解：</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@Controller</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">：标记类为控制器，使其能被 Spring MVC 识别</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@RequestMapping</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">：建立请求与控制器方法的映射关系</font>
2. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">工作流程：Spring MVC 接收请求后，通过映射关系找到对应控制器方法处理请求，使控制器可被外界访问</font>

## <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">二、@Controller 注解</font>
### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（一）功能</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">标记在类上，将该类声明为 Spring MVC 的 Controller 类</font>

### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（二）使用要求</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">需在 spring-mvc 配置文件（如 springmvc.xml）中添加组件扫描配置：</font>

1. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">引入 context 命名空间：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">xmlns:context="http://www.springframework.org/schema/context"</font>`
2. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">配置扫描路径：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"><context:component-scan base-package="控制器目录路径（例：test.controller）"/></font>`

## <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">三、@RequestMapping 注解</font>
### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（一）基本说明</font>
1. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">必须与</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@Controller</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">配合使用，用于处理请求地址映射</font>
2. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">可标注位置：</font>
    - <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">类上：设置请求路径的初始信息（相当于 WEB 应用根目录）</font>
    - <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">方法上：设置请求路径的具体信息（相当于类定义处 URL 的延伸）</font>
3. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">示例：</font>
    - <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">类上标注</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@RequestMapping("/user")</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"> + 方法上标注</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@RequestMapping("/hello")</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"> → 访问 URL：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">http://xxx.com/user/hello</font>`
    - <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">仅方法上标注</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@RequestMapping("/hello2")</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"> → 访问 URL：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">http://xxx.com/hello2</font>`

### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（二）核心属性总览</font>
| **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">属性名</font>** | **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">功能描述</font>** |
| --- | --- |
| <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">value</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">通过请求地址匹配映射（必填）</font> |
| <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">method</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">通过请求方式（GET/POST 等）匹配映射</font> |
| <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">params</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">通过请求参数匹配映射</font> |
| <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">headers</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">通过请求头信息匹配映射</font> |
| <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">consumes</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">通过请求头的 Content-Type 信息匹配映射</font> |
| <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">produces</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">设置返回数据的类型及编码（需与 @ResponseBody 配合使用）</font> |


<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">注意：属性越多，匹配条件越精确，需所有条件同时满足才会触发映射</font>

### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（三）属性详细解析</font>
#### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">1. value 属性</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">功能：通过请求地址匹配映射，支持多个地址</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">类型：字符串数组，至少配置一个地址</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">示例：	</font>

```plain
@RequestMapping(value = {"/hello", "/hello2"}) // 匹配/hello和/hello2两个地址
@ResponseBody
public String sayHello() { return "test1234"; }
```

+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">访问方式：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">http://xxx.com/hello</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"> 或 </font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">http://xxx.com/hello2</font>`

#### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">2. value 属性（Ant 风格路径）</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">支持通配符实现灵活路径匹配：</font>

| **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">通配符</font>** | **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">含义</font>** | **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">注意事项</font>** |
| --- | --- | --- |
| <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">?</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">匹配任意单个字符</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">仅允许单个字符</font> |
| <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">*</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">匹配任意 0 个或多个字符</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">可多个字符，不跨目录</font> |
| <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">**</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">匹配任意一层或多层目录</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">仅能使用</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">/**/xxx</font>`<br/><font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">格式</font> |


+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">示例及访问方式：</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@RequestMapping("/hello?")</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"> → 匹配</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">/helloa</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">/hellob</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">等</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@RequestMapping("/a*a/hello3")</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"> → 匹配</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">/aa/hello3</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">/aba/hello3</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">等</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@RequestMapping("/**/hello4")</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"> → 匹配</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">/hello4</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">/x/hello4</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">/x/y/hello4</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">等</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@RequestMapping("/test/**")</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"> → 匹配</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">/test</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">/test/a</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">/test/a/b</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">等</font>

#### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">3. method 属性</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">功能：限制请求方式，支持多种方式</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">类型：RequestMethod 数组（可选值：GET、HEAD、POST、PUT、PATCH、DELETE、OPTIONS、TRACE）</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">派生注解（简化写法）：</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@GetMapping</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">：等价于</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">method = RequestMethod.GET</font>`
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@PostMapping</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">：等价于</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">method = RequestMethod.POST</font>`
    - <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">同理：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@PutMapping</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@PatchMapping</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@DeleteMapping</font>`
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">示例：</font>

```plain
@RequestMapping(value = "/hello", method = {RequestMethod.GET, RequestMethod.POST})
@ResponseBody
public String sayHello() { return "test1234"; }
```

+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">访问结果：GET/POST 请求正常，DELETE 请求返回 405（方法不允许）</font>

#### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">4. params 属性</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">功能：通过请求参数匹配，支持四种表达式：</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">{"param"}</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">：必须携带 param 参数</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">{"!param"}</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">：必须不携带 param 参数</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">{"param=value"}</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">：必须携带 param 参数且值为 value</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">{"param!=value"}</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">：必须携带 param 参数且值不为 value</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">示例：</font>

```plain
@RequestMapping(value = "/hello", params = {"t1", "!t2", "t3=123456", "t4!=qwerty"})
@ResponseBody
public String sayHello() { return "test1234"; }
```

+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">访问结果：不满足参数条件时返回 400（错误的请求）</font>

#### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">5. headers 属性</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">功能：通过请求头信息匹配，表达式格式同 params：</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">{"header"}</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">：必须携带 header 头</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">{"!header"}</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">：必须不携带 header 头</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">{"header=value"}</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">：必须携带 header 头且值为 value</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">{"header!=value"}</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">：必须携带 header 头且值不为 value</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">示例：</font>

```plain
@RequestMapping(value = "/hello", headers = {"h1", "!h2", "h3=123456", "h4!=qwerty"})
@ResponseBody
public String sayHello() { return "test1234"; }
```

+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">访问结果：不满足头信息条件时返回 404（未找到）</font>

#### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">6. consumes 属性</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">功能：匹配请求头的 Content-Type 信息</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">支持格式：</font>
    - <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">精确匹配：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">consumes = "text/plain"</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（仅匹配 text/plain 类型）</font>
    - <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">排除匹配：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">consumes = {"!text/plain"}</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（排除 text/plain 类型）</font>
    - <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">通配匹配：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">consumes = "application/*"</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（匹配 application / 开头的所有类型）</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">示例：</font>

```plain
@RequestMapping(value = "/hello", consumes = {"text/plain", "application/*"})
@ResponseBody
public String sayHello() { return "test1234"; }
```

+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">访问结果：不满足 Content-Type 时返回 415（不支持的媒体类型）</font>

#### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">7. produces 属性</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">功能：设置返回数据的类型及编码，需与</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@ResponseBody</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">配合使用</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">常用写法：</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">produces = "text/html;charset=UTF-8"</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（返回 HTML 格式，UTF-8 编码）</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">produces = "application/json;charset=UTF-8"</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（返回 JSON 格式，UTF-8 编码）</font>
    - `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">produces = "application/xml;charset=UTF-8"</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（返回 XML 格式，UTF-8 编码）</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">示例：</font>

```plain
@RequestMapping(value = "/hello", produces = "text/html;charset=UTF-8")
@ResponseBody
public String sayHello() { return "这是一段中文"; }
```

## <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">四、@RequestMapping 派生注解</font>
