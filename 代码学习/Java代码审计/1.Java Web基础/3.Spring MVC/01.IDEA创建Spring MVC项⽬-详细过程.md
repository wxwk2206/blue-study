## 一、项目创建
### （一）新建项目
1. 打开 IntelliJ IDEA 2020.1.4，选择 “创建新项目”。
2. 创建方式与组件选择：选择 “Maven”，SDK 指定为 jdk1.8，组件选择 org.apache.maven.archetypes 下的 maven-archetype-webapp。
3. 完善项目信息：
    - Name：项目根文件名（示例：SpringMVCTest）
    - Group：一般填写公司名（示例：com.github.wxwk2206）
    - Artifact：项目名（示例：SpringMVCTest）
    - Version：默认 1.0-SNAPSHOT

### （二）配置依赖（修改 pom.xml）
在项目根目录下找到 pom.xml 文件，添加以下配置（首次添加报红需点击 Maven 按钮刷新导入）：

####  关键配置项：
##### 静态资源放行
+ `<mvc:default-servlet-handler/>`（解决静态资源访问 404 问题）

```xml
如果在web.xml中配置DispatcherServlet时配置的url-pattern为“/”时
代表所有请求都交由Spring容器管理
动态资源能通过Controller中的请求映射来实现请求管理
⽽静态资源则需要进⾏静态放⾏,⽐如图⽚资源, js资源,html资源等
因此就需要添加该配置让静态资源可以被正常访问
打开: ./SpringMVCTest/src/main/resources/springmvc.xml
在<beans></beans>标签中添加如下内容:
<mvc:default-servlet-handler/>
```

##### 注解驱动
+ `<mvc:annotation-driven/>`（解决 @Controller 失效问题）

```xml
在设置完<mvc:default-servlet-handler/>以后,你会发现,静态资源能正常访问了
但是设置的@Controller访问不到了,这个时候就需要添加该配置进⾏解决了
打开: ./SpringMVCTest/src/main/resources/springmvc.xml
在<beans></beans>标签中添加如下内容:
<mvc:annotation-driven/>
```

##### 中文乱码处理
+ 配置 StringHttpMessageConverter，指定 charset=UTF-8

```xml
<!-- 在创建控制器的时候如果使⽤了@ResponseBody,然后返回的是中⽂,你会发现前端显示的是中⽂乱码 -->
<!-- 这是因为SpringMVC默认处理的字符集是ISO-8859-1,所以需要全局处理这个问题 -->
<!-- 这个时候就需要添加该配置进⾏解决了 -->
<!-- 打开: ./SpringMVCTest/src/main/resources/springmvc.xml -->
<!-- 在<beans></beans>标签中添加如下内容: -->
<mvc:annotation-driven>
<mvc:message-converters register-defaults="true">
<bean class="org.springframework.http.converter.StringHttpMessageConverter">
<property name="supportedMediaTypes" value="text/html;charset=UTF-8"/>
</bean>
</mvc:message-converters>
</mvc:annotation-driven>
```

##### 文件上传解析器
+ 配置 CommonsMultipartResolver，最大上传尺寸 5MB（5242880 字节）

```xml
<!-- 为了避免上传的时候爆由于没有提供multi-part配置,⽆法处理parts的错误 -->

#### 3. 设置字符集防⽌乱码  
+  请求参数如果含有中⽂ , 会出现中⽂乱码问题

</init-param>
<init-param>
<param-name>forceEncoding</param-name>

// 创建个测试使⽤的控制器
// ⽬录: ./SpringMVCTest/src/main/com/test/controller/TestController.java
// ⽂件名: TestController.java
package test.controller;
import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.servlet.ModelAndView;
@Controller
public class TestController {
    @RequestMapping("/hello")
    public ModelAndView sayHello(String name) {
        ModelAndView mv = new ModelAndView();
        mv.addObject("msg", "你好啊," + name + ",这是⼀个欢迎界⾯");
        // 注:
        // 因为springmvc.xml设置了<property name="suffix" value=".jsp"></property>
        // 指定了请求⽂件的后缀是.jsp
        // 所以直接写成/hello就可以了
        // 如果没有在springmvc.xml设置后缀,那就需要写成/hello.jsp
        mv.setViewName("/hello");
        return mv;
    }
}
```
