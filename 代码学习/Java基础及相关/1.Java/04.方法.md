## 方法
方法：程序中的独立功能，也是**最小的执行单元**，可提高程序的复用性和可维护性

方法的**定义格式**（参数为形参）
![[Pasted image 20260703180015.png]]

方法的**调用格式**（参数为实参）

       方法名（参数1，参数2...）;

注意点：

+ 方法与方法是平级关系，不能进行嵌套
+ 方法不会主动运行，需要去主动调用
+ 方法格式里，（）里面的参数需要一一对应（个数，类型），如果没有才会隐式转换
