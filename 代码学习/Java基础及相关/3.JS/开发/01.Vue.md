Vue是一款用于**构建用户界面**的**渐进式**的JavaScript**框架**
（官方：[https://cn.vuejs.org/](https://cn.vuejs.org/)）
![[Pasted image 20260703174643.png]]

**局部模块改造**：Vue核心包开发
  - 适合小场景、逐步迁移老项目
**整站开发**：Vue核心包 + Vue插件工程化开发
  - 功能更强大，适合全新的SPA项目

**框架优势**：
- 基于**数据驱动**和**组件化**思想，易于开发和维护
- 轻量高效，适合各种规模的项目
- 性能好，底层基于虚拟DOM

**框架功能**：核心包 + 插件 = 功能完备的现代框架

### 准备工作

1. 参考Vue官方文档 [https://cn.vuejs.org/](https://cn.vuejs.org/)
2. 通过CDN或npm安装引入Vue核心JS文件
3. 创建Vue根组件，完成基本配置

Vue基于数据驱动（data-driven）思想：
- 视图（View）是数据（Model）的映射
- 当数据发生改变时，Vue会自动帮我们更新视图
- 我们只需要关注数据，无需手动操作DOM

Vue 3 入门示例（CDN + createApp）：
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Vue 快速入门</title>
  <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
</head>
<body>
  <div id="app">
    <h1>{{ message }}</h1>
  </div>
  <script>
    const { createApp } = Vue;
    createApp({
      data() {
        return {
          message: 'Hello Vue!'
        };
      }
    }).mount('#app');
  </script>
</body>
</html>
```

### 常用指令
指令：HTML标签上带有 v-前缀的特殊属性，不同的指令具有不同的含义，可以实现不同的功能

例：`v-if`、`v-for`、`v-bind` 等，均以 `v-` 开头

| 指令 | 作用 |
| --- | --- |
| `v-bind` | 为 HTML 标签绑定属性值，如设置 href、CSS 样式等 |
| `v-model` | 在表单元素上创建双向数据绑定 |
| `v-on` | 为 HTML 标签绑定事件 |
| `v-if` / `v-else-if` / `v-else` | 条件性的渲染某元素，判定为 true 时渲染，否则不渲染 |
| `v-show` | 根据条件展示某元素，区别在于切换的是 display 属性的值 |
| `v-for` | 列表渲染，遍历容器的元素或者对象的属性 |

#### v-for

作用：列表渲染，遍历容器的元素或者对象的属性。

语法：`v-for="(item, index) in items"` —— 需要循环的标签上加上该属性。

**遍历数组**：
```html
<tr v-for="(e, index) in emplist" :key="e.id">
  <td>{{ index + 1 }}</td>
  <td>{{ e.name }}</td>
</tr>
```


- `:key` 是 v-bind:key 的简写，给每个循环项一个唯一标识，Vue 用它来跟踪节点身份，高效重用和排序现有元素
- **在Vue中，插值表达式是不能直接写在标签内部的**

#### v-bind
作用：动态为HTML标签绑定属性值，如设置href，src，style样式等


语法：`v-bind:属性名="Vue变量"`，简写为 `:属性名="Vue变量"`

```html
<!-- v-bind动态绑定属性 -->
<a v-bind:href="url">链接</a>
<a :href="url">链接（简写）</a>

<!-- 动态绑定图片src -->
<img :src="imgUrl" />

<!-- 动态绑定class样式 -->
<div :class="{ active: isActive }">根据数据切换样式</div>
```


- 所有html标签的属性想要动态绑定，都用 `v-bind`
- `v-bind` 可以简写为 `:`


```html
<!-- v-bind 绑定style行内样式 -->
<div :style="{ color: textColor, fontSize: fontSize + 'px' }">样式绑定示例</div>
```

#### v-if或v-show
作用：这两类指令，都是用来控制元素的显示与隐藏的


- `v-if`：条件性的渲染某元素（或组件），判定为true时渲染，否则不渲染
- `v-else-if`：充当 `v-if` 的 "else-if 块"，可以连续使用
- `v-else`：充当 `v-if` 的 "else 块"


- `v-show`：根据条件展示某元素，本质是切换元素的 `display` CSS属性（true显示，false隐藏）

**v-if vs v-show 区别**：
- `v-if`：不展示的DOM元素直接被移除（DOM树中不存在）
- `v-show`：不展示的DOM元素通过CSS `display:none` 隐藏（DOM树中存在）

**v-else-if 注意事项**：
- `v-if` 和 `v-else-if`、`v-else` 三个指令必须配合使用
- 必须配合使用的这三个标签中间，不能有其他元素打断

```vue
<!-- v-if: 控制元素的显示与隐藏 -->
<td>
  <span v-if="e.job == 1">班主任</span>
  <span v-else-if="e.job == 2">讲师</span>
  <span v-else-if="e.job == 3">学工主管</span>
  <span v-else-if="e.job == 4">教研主管</span>
  <span v-else-if="e.job == 5">咨询师</span>
  <span v-else>其他</span>
</td>
```

#### v-model

作用：在表单元素上创建**双向数据绑定**。

- 当视图（表单输入）变化时，数据自动更新
- 当数据变化时，视图自动更新

```html
<div id="app">
  <input type="text" v-model="username" />
  <p>你的名字是：{{ username }}</p>
</div>
```

#### v-on
作用：为html标签绑定事件（添加事件监听）

语法：
- v-on:事件名="方法名"
- 简写为 @事件名="..."

```vue
<div id="app">
  <button type="button" v-on:click="handle">点我</button>
  <button type="button" @click="handle">再点我</button>
</div>
```

```js
const app = createApp({
  data() {
    // ...
  },
  methods: {
    handle() {
      console.log('试试就试试');
    }
  },
}).mount("#app");
```
