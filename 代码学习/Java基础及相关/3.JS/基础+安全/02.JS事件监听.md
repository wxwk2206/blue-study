## 简述

**事件**：HTML事件是发生在HTML元素上的"事情"。比如：
- 按钮被点击
- 鼠标移动到元素上
- 按下键盘按键

**事件监听**：JavaScript可以在事件触发时，就立即调用一个函数做出响应，也称为**事件绑定或注册事件**。
## 事件监听语法

**语法一：addEventListener**
```js
元素对象.addEventListener('事件类型', 要执行的函数);
```

**三步走**：
1. 获取事件源（DOM元素）
2. 绑定事件（addEventListener方式或on事件方式）
3. 编写事件处理程序（响应），完成效果

**事件监听三要素**：
- **事件源**：哪个DOM元素被事件触发了，要获取dom元素
- **事件类型**：用什么方式触发，比如鼠标单击 click、鼠标移入 mouseenter 等
- **事件处理程序**：要做什么事

```html
<!-- 点击按钮，做出对应的响应 -->
<body>
  <button id="btn">点我</button>
  <script>
    // 1. 获取事件源（按钮）
    const btn = document.querySelector('#btn');
    // 2. 绑定事件、3. 编写事件处理程序
    btn.addEventListener('click', function() {
      alert('你点击了按钮');
    });
  </script>
</body>
```

```html
<!-- 鼠标悬停下拉菜单效果 -->
<body>
  <div id="menu">下拉菜单</div>
  <div id="submenu" style="display:none;">
    <p>选项1</p>
    <p>选项2</p>
  </div>
  <script>
    const menu = document.querySelector('#menu');
    const submenu = document.querySelector('#submenu');
    // 鼠标移入显示子菜单
    menu.addEventListener('mouseenter', function() {
      submenu.style.display = 'block';
    });
    // 鼠标移出隐藏子菜单
    menu.addEventListener('mouseleave', function() {
      submenu.style.display = 'none';
    });
  </script>
</body>
```

+ addEventListener可以多次绑定同一事件

**语法二：on事件**
```js
元素对象.on事件类型 = function() { ... };
```

示例：点击按钮显示"你好呀~"
```html
<button id="btn2">点我试试</button>
<script>
  const btn2 = document.querySelector('#btn2');
  btn2.onclick = function() {
    alert('你好呀~');
  };
</script>
```

+ on事件多次绑定同一事件会覆盖

## 常见事件
### 鼠标事件
+ click鼠标点击
+ mouseenter鼠标移入
+ mouseleave鼠标移出

### 键盘事件
+ keydown键盘按下触发
+ keyup键盘抬起触发

### 焦点事件
+ focus获取焦点触发
+ blur失去焦点触发

### 表单事件
+ input用户输入时触发
+ submit表单提交时触发
