### 作用
**数据交换，异步交互**

- 可以在页面不刷新的情况下，向服务端发送HTTP请求，得到HTTP响应
- 服务器端获得了数据，可以把数据通过HTTP响应返回给Ajax
- Ajax利用返回的数据，就可以动态更新页面中的内容

### Axios
介绍：Axios对原生的Ajax进行了封装，简化书写，快速开发

官网：[https://www.axios-http.cn/](https://www.axios-http.cn/)

使用Axios的基本步骤：
1. 引入 Axios 的 js 文件
2. 使用 Axios 发送请求，并获取响应结果

```html
<head>
  <script src="https://unpkg.com/axios/dist/axios.min.js"></script>
</head>
```

**axios发送GET请求示例**：
```js
// 通过axios发送GET请求获取员工列表
axios.get('http://localhost:8080/emps').then(result => {
  // 请求成功，result是服务器返回的响应数据
  console.log(result.data);
}).catch(err => {
  // 请求失败
  console.log(err);
});
```

**axios发送POST请求示例**：
```js
// 通过axios发送POST请求新增员工
axios.post('http://localhost:8080/emps', {
  name: 'Tom',
  age: 20,
  gender: '男'
}).then(result => {
  console.log(result.data);
}).catch(err => {
  console.log(err);
});
```

**Axios 请求方法别名**：

| 方法别名 | 对应 HTTP 方法 | 用途 |
| --- | --- | --- |
| `axios.get(url[, config])` | GET | 查询数据 |
| `axios.post(url[, data[, config]])` | POST | 新增数据 |
| `axios.put(url[, data[, config]])` | PUT | 修改数据 |
| `axios.delete(url[, config])` | DELETE | 删除数据 |

### async&await
可以通过async，await让异步变为同步操作。async就是来声明一个异步方法，await是用来等待异步任务执行，用来增加代码的可读性和可维护性

```js
// async 声明异步方法，await 等待异步任务完成
async function fetchData() {
  // 使用 await 等待 axios 请求完成，获取响应数据
  const result = await axios.get('http://localhost:8080/emps');
  // 这里的代码会等上面请求完成后才执行
  console.log(result.data);
}
```


**await 注意事项**：
- `await` 关键字**必须**在 `async` 声明的异步方法中使用
- `await` 后面跟的是一个Promise对象（axios方法的返回值就是Promise）

### Vue生命周期
生命周期：指一个对象从创建到销毁的整个过程。
生命周期的八个阶段：每触发一个生命周期事件，会自动执行一个生命周期方法（钩子）。

 **Vue 生命周期示意图（8个阶段）** ![](https://cdn.nlark.com/yuque/0/2026/png/62702379/1772632216324-85fb58d9-c1ce-4706-bced-ed34b0cffd27.png)


| 钩子函数 | 触发时机 | 常用场景 |
|---------|---------|---------|
| `beforeCreate` | 实例初始化之后，数据观测和事件配置之前 | - |
| `created` | 实例创建完成后，数据观测、属性和方法已配置完成 | 发送初始化请求获取数据 |
| `beforeMount` | 挂载开始之前，render函数首次被调用 | - |
| `mounted` | 实例被挂载后，DOM已渲染 | 操作DOM、发送Ajax请求获取数据 |
| `beforeUpdate` | 数据更新时，DOM重新渲染之前 | - |
| `updated` | 数据更新且DOM重新渲染之后 | - |
| `beforeUnmount` | 实例销毁之前 | 清除定时器、取消事件监听 |
| `unmounted` | 实例销毁后 | - |

借用Vue生命周期的知识，利用钩子函数在页面加载完毕就执行某一操作：

```js
createApp({
  data() {
    return {
      emplist: []
    };
  },
  // ⭐ mounted钩子：通过钩子函数在页面加载完成后就立即执行某一项操作，常用于初始化数据
  async mounted() {
    const result = await axios.get('http://localhost:8080/emps');
    this.emplist = result.data;
  }
}).mount('#app');
```
