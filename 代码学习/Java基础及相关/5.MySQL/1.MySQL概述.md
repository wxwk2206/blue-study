## 安装
### 1.下载解压
下载地址：[https://dev.mysql.com/downloads/mysql/](https://dev.mysql.com/downloads/mysql/)

  下载压缩包解压到没有中文的路径下

### 2.配置环境变量
![[Pasted image 20260702202435.png]]

![[Pasted image 20260702202440.png]]

### 3.初始化
以管理员身份，运行命令行窗口：

```sql
mysqld --initialize-insecure
```

![[Pasted image 20260702202447.png]]

稍微等待一会，如果出现没有出现报错信息，则证明data目录初始化没有问题，此时再查看MySQL目录下已经有data目录生成。

![[Pasted image 20260702202451.png]]

### 4.注册MySQL服务
命令行（注意必须以管理员身份启动）中，输入如下的指令，回车执行： 

```sql
mysqld -install
```

![[Pasted image 20260702202454.png]]

显示成功即安装好了MySQL服务了

### 5.启动MySQL服务
在黑框里敲入`net start mysql`，回车。

```sql
net start mysql  // 启动mysql服务
    
net stop mysql  // 停止mysql服务
```

### 6.修改默认账户密码
在黑框里敲入`mysqladmin -u root password 1234`，这里的`1234`就是指默认管理员(即root账户)的密码。

```sql
mysqladmin -u root password 1234
```


![[Pasted image 20260702202459.png]]

## 登录
右键开始菜单，选择`命令提示符`，打开黑框。

在黑框中输入，`mysql -uroot -p1234`，回车，出现下图且左下角为`mysql>`，则登录成功。

```sql
mysql -uroot -p1234
```


![[Pasted image 20260702202501.png]]

退出mysql：

```sql
exit
```

登录参数：

```sql
mysql -u用户名 -p密码 -h要连接的mysql服务器的ip地址(默认127.0.0.1) -P端口号(默认3306)
```

![[Pasted image 20260702202506.png]]

登陆时，可以直接-p，然后回车，密码以密文形式输入

## 卸载
点击开始菜单，输入cmd，选择 "命令提示符"，选择右侧的 "以管理员身份运行"。
![[Pasted image 20260702202514.png]]

1. 敲入`net stop mysql`，回车。
![[Pasted image 20260702202519.png]]

2. 再敲入`mysqld -remove mysql`，回车。
![[Pasted image 20260702202523.png]]

3. 最后删除MySQL目录及相关的环境变量。
