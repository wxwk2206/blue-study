#### `<span>` （元素选择器）
是 HTML 中的**行内通用容器标签**，本身没有任何默认的样式和语义，就像一个 “无标记的透明盒子”

### 盒子模型
盒子：页面中所有的元素（标签），都可以看做是一个盒子，由盒子将页面中的元素包含在一个矩形区域内，通过盒子的视角更方便的进行页面布局

盒子模型组成：内容区域（content）、内边距区域（padding）、边框区域（border）、外边距区域（margin）

## 总结
### 一、HTML 常用核心标签
HTML 是网页的骨架，这些标签是构建网页最基础、最常用的，我按功能分类说明：

#### 1. 文档结构标签（页面骨架）
| **标签** | **作用** |
| --- | --- |
| `<!DOCTYPE html>` | 声明文档类型，告诉浏览器这是 HTML5 文档（必须放在第一行） |
| `<html>` | 整个 HTML 文档的根标签，所有内容都包裹在里面 |
| `<head>` | 页面的头部，存放元信息（如标题、编码、CSS 链接），不显示在页面主体 |
| `<title>` | 定义网页标题，显示在浏览器标签栏，是 SEO 核心 |
| `<body>` | 页面的主体内容区域，所有可见内容（文字、图片、按钮等）都放在这里 |


#### 2. 文本内容标签（最常用）

| **标签**        | **作用**                                          |
| ------------- | ----------------------------------------------- |
| `<h1>`-`<h6>` | 标题标签，h1 最大（页面主标题，建议一个页面只一个），h6 最小，自带加粗和换行       |
| `<a>`         | 超链接标签，核心属性`href`（跳转地址），`target="_blank"`（新窗口打开） |
| `<p>`         | 段落标签，包裹一段文字，自带上下外边距，区分不同段落                      |
| `<span>`      | 行内标签，包裹小段文本（如单个字 / 词），用于局部样式修改                  |
| `<div>`       | 块级标签，无默认样式，用于布局分区（网页布局的 “容器”）                   |
| `<b>`         | 加粗文本                                            |
| `<strong>`    | 加粗文本，具有强调语义                                     |
| `<i>`         | 斜体文本                                            |
| `<em>`        | 斜体文本，具有强调语义                                     |
| `<u>`         | 下划线                                             |
| `<ins>`       | 下划线，具有强调语义                                      |
| `<s>`         | 删除线                                             |
| `<del>`       | 删除线，具有强调语义                                      |


#### 3. 多媒体 / 表单常用标签

| **标签**  | **作用**                                                           |
| -------------- | ----------------------------------------------------------------------- |
| `<img>` | 图片标签，核心属性：`src`（图片地址）、`alt`（图片加载失败提示）、`width/height`（尺寸）  |
| `<video>`      | 插入视频                                                                    |
| `<audio>`      | 插入音频                                                                    |
| `<input>`      | 表单输入框，核心属性：`type`（类型，如text/button/checkbox/radio/email/password/submit） |
| `<select>`     | 定义表单中的下拉列表                                                              |
| `<textarea>`   | 定义文本域                                                                   |
| `<button>`     | 按钮标签，可包裹文字 / 图标，比 input 按钮更灵活                                           |
| `<ul>/<li>`    | 无序列表（ul 是列表容器，li 是列表项），默认带圆点，常用于导航、列表                            |
| `<ol>/<li>`    | 有序列表，默认带数字序号                                                            |


### 二、CSS 常用核心样式
CSS 是网页的 “皮肤”，用于美化 HTML 标签，以下是最常用的样式属性，按功能分类：

#### 1. 布局与盒模型（最核心）
盒模型是 CSS 布局的基础，每个标签都是一个 “盒子”，核心样式：

| **样式属性** | **作用**                                                                                           |
| --------------- | ------------------------------------------------------------------------------------------------------- |
| `width/height`  | 设置盒子的宽度 / 高度（如`width: 200px;`）                                                                          |
| `margin`        | 外边距（盒子与其他元素的间距），`margin: 0 auto;`可实现块级元素水平居中                                                            |
| `padding`       | 内边距（盒子内容与边框的间距）                                                                                         |
| `border`        | 边框（如`border: 1px solid #ccc;`：1px 宽、实线、灰色边框）                                                            |
| `display`       | 控制元素显示方式：- `block`：块级（独占一行，可设宽高）- `inline`：行内（不独占一行，不可设宽高）- `inline-block`：行内块（不独占一行，可设宽高）- `none`：隐藏元素 |
| `float`         | 浮动（常用`left/right`），用于简单布局（如图文环绕、多列布局）                                                                   |
| `position`      | 定位：`relative`：相对定位（相对于自身原位置）`absolute`：绝对定位（相对于最近的已定位父元素）`fixed`：固定定位（相对于浏览器窗口，如固定导航栏）                  |


#### 2. 文本样式（美化文字）

| **样式属性**   | **作用**                                            |
| ----------------- | -------------------------------------------------------- |
| `color`           | 文字颜色（如`color: #333;`、`color: red;`）                      |
| `font-size`       | 文字大小（如`font-size: 16px;`，网页默认 16px）                      |
| `font-weight`     | 文字粗细（`bold`加粗，`normal`正常，数值 400=normal，700=bold）         |
| `font-family`     | 字体（如`font-family: "微软雅黑", Arial, sans-serif;`）           |
| `text-indent`     | text-indent:2em首行缩进两个空格                           |
| `text-align`      | 文本对齐（`left/center/right`，用于块级元素内的文本 / 行内元素居中）            |
| `text-decoration` | 文本装饰：`none`（去除下划线，常用于 a 标签）、`underline`（下划线）             |
| `line-height`     | 行高（设置文字行间距，`line-height: 24px;`或`line-height: 1.5;`（倍数）） |


#### 3. 背景样式


| **样式属性**            | **作用**                                            |
| ------------------- | ------------------------------------------------- |
| `background-color`  | 背景颜色（如`background-color: #f5f5f5;`）               |
| `background-image`  | 背景图片（`background-image: url("bg.png");`）          |
| `background-repeat` | 背景重复（`no-repeat`不重复，`repeat-x`水平重复,repeat-y垂直重复）  |
| `background-size`   | 背景尺寸（`cover`覆盖整个容器，`contain`完整显示图片，`100% 100%`拉伸） |


#### 4. 其他常用样式

| **样式属性**        | **作用**                              |
| --------------- | ----------------------------------- |
| `cursor`        | 鼠标指针样式（`pointer`手型，`default`默认箭头）   |
| `opacity`       | 透明度（0-1，如`opacity: 0.8;`）           |
| `border-radius` | 圆角（如`border-radius: 5px;`，50% 可做圆形） |
