### 文件操作

操作文件，基本分为三步操作：打开、读 / 写、关闭。
**** 读文件
```python
# 1. 打开文件
f = open("resources/望庐山瀑布.txt", "r", encoding="utf-8")

# 2. 读取文件
content = f.read()
print(content)

# 3. 关闭文件
f.close()
```
**** 写文件
```python
# 1. 打开文件
f = open("resources/静夜思.txt", "w", encoding="utf-8")

# 2. 写入文件
f.write("窗前明月光，\n")
f.write("疑是地上霜。\n")
f.write("举头望明月，\n")
f.write("低头思故乡。\n")

# 3. 关闭文件
f.close()
```
 补充说明：
- **`open()`** 函数用于打开文件，参数分别为：文件路径、打开模式（`r` 读 / `w` 写）、编码格式。
- **`read()`** 用于读取文件全部内容，`write()` 用于写入字符串。
- 必须手动调用 `close()` 关闭文件，否则会造成资源泄漏；更推荐使用 `with` 语句自动管理文件关闭。
### 文件操作 - 资源释放
**方案一：try-finally 手动释放**（繁琐）
```python
# 1. 打开文件
f = open("resources/静夜思.txt", "w", encoding="utf-8")
try:
    # 2. 写入文件
    f.write("静夜思(李白)\n")
    f.write("窗前明月光，\n")
    f.write("疑是地上霜。\n")
    f.write("举头望明月，\n")
    f.write("低头思故乡。\n")
finally:
    # 3. 关闭文件
    f.close()
```
特点：必须手动写 `finally` 块保证文件关闭，代码繁琐。
**方案二：with 语句（推荐）**
```python
with open("resources/静夜思.txt", "w", encoding="utf-8") as f:
    f.write("静夜思(李白)\n")
    f.write("窗前明月光，\n")
    f.write("疑是地上霜。\n")
    f.write("举头望明月，\n")
    f.write("低头思故乡。\n")
```
特点：`with` 语句（上下文管理器）会**自动管理资源**，代码块结束后自动调用 `close()` 释放文件， 即使代码块中发生异常，也能保证文件被正确关闭，是项目开发中的推荐方式。
### 读写 JSON 

JSON 是软件开发中最常用的数据交换格式，Python 标准库提供了 `json` 模块来简化 JSON 数据处理。

---
- **序列化**：将 Python 字典 / 列表等对象 → 写入 JSON 文件（`json.dump()`）
- **反序列化**：从 JSON 文件 → 读取为 Python 字典 / 列表（`json.load()`）

```python
import json

# 1. 序列化：Python 字典 → JSON 文件
user_data = {
    "name": "荒",
    "age": 18,
    "gender": "男",
    "hobbies": ["reading", "swimming"]
}

with open("user.json", "w", encoding="utf-8") as f:
    json.dump(user_data, f, ensure_ascii=False, indent=4)
#   关键参数说明：
# ensure_ascii=False：保证中文正常显示，避免被转义为 Unicode 编码
# indent=4：让 JSON 文件格式化缩进，更易阅读

# 2. 反序列化：JSON 文件 → Python 字典
with open("user.json", "r", encoding="utf-8") as f:
    loaded_data = json.load(f)
    print(loaded_data)
```
