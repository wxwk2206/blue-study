# 1.网络基础
## 1.1 IP 地址
可以理解为就是设备在互联网中的地址（唯一身份证），每一个连入网络的设备都有一个 IP 地址，那就是设备在互联网中的位置。分为ipv4和ipv6

## 1.2域名
（Domain Name），是由一串用点分隔的英文字母组成的。IP 地址不便于记忆，因此设计出了域名，并通过 DNS（域名解析服务器）来将域名和 IP 地址相互映射，便于记忆和访问。

## 1.3端口号
（Port）是整数，取值范围在 0-65535，它是用来标识计算机设备中的运行的程序。

## 1.4网络模型

互联网连接了数以亿计的设备，网络四通八达如同城市的复杂道路，网络中传输的数据就像城市道路中的车流，若不加以管理必然会出现混乱。国际 ISO 组织统一了程序在网络中通信的模型和标准，主要包括：
- **OSI 网络模型**：全球网络互连的标准模型，将网络通信划分为 7 个层级，是理论上的参考标准。
- **TCP/IP 网络模型**：实际应用最广泛的模型，可认为是 OSI 模型的简化版，通常划分为 4 层，是互联网的核心通信基础。
## 1.5 HTTP 协议
- **概念**：Hyper Text Transfer Protocol（超文本传输协议），它规定了客户端与服务器之间数据传输的统一规则。（核心作用：只有在请求和响应过程中都遵循这套规则，服务端才能正确解析客户端的请求，客户端也才能正确理解服务端返回的结果。）
 **特点 
1. 基于文本的协议：请求和响应的协议内容为文本格式，底层通过 TCP 协议传输，稳定性强。
2. 基于请求 - 响应模型：一次请求对应一次响应，必须由客户端先发起请求，服务端才会返回响应。
3. 无状态：服务端不会记忆与客户端的历史交互信息，每次请求 - 响应都是独立的。

### JSON 格式 
JSON（JavaScript Object Notation）是一种轻量级的数据交换格式，形式上类似 Python 字典，采用`key:value`键值对结构，**所有 key 必须用双引号包裹**。
**数据类型
- **对象**：用`{}`包裹，内部是键值对，键为字符串，值可为任意类型
- **数字**：包含整数和小数，如`12`、`3.14`
- **字符串**：用双引号`""`包裹，如`"jack"`
- **布尔**：仅`true`或`false`两个值
- **列表**：用`[]`包裹，元素间用逗号`,`分隔

**** 示例代码
```json
{
  "model": "deepseek-chat",
  "messages": [
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "Hello!"}
  ],
  "stream": false
}
```
### 会话记忆 - 滚雪球方案 

这是一种实现 AI 会话上下文记忆的基础方法，核心是**将历史对话（用户提问 + AI 回复）不断追加到请求消息列表中**，让 AI 能基于之前的交互继续回答。

- **第 1 轮**：
    ```
    "messages": [
      {"role": "system", "content": "你是一名可爱的AI助手..."},
      {"role": "user", "content": "12个苹果,3个人怎么均分"}
    ]
    ```
    AI 回复后，将回复加入列表：
    ```
    {"role": "assistant", "content": "嘻嘻,12个苹果分给3个人,每个人可以分到**4个苹果**哦~"}
    ```
 
- **第 2 轮**：
    新提问时，把完整历史 + 新问题一起发送：
    ```
    "messages": [
      {"role": "system", "content": "你是一名可爱的AI助手..."},
      {"role": "user", "content": "12个苹果,3个人怎么均分"},
      {"role": "assistant", "content": "嘻嘻,12个苹果分给3个人,每个人可以分到**4个苹果**哦~"},
      {"role": "user", "content": "那2个人呢?"}
    ]
    ```
  AI 就能基于之前的对话，回答 “2 个人分的话每人 6 个”。
# 3.python代码调用
- **PyPI**：全称 Python Package Index，是 Python 官方和社区共同维护的**第三方软件包仓库**，存放着海量可复用的 Python 代码库。
- **pip**：Python 官方提供的**包管理工具**，负责从 PyPI 等仓库中查找、下载、安装、卸载 Python 包，是开发者管理项目依赖的核心工具。

```python
# Please install OpenAI SDK first: `pip3 install openai`
#在终端中输入pip3 install openai
import os
from openai import OpenAI

# 初始化客户端
client = OpenAI(
    api_key=os.environ.get('DEEPSEEK_API_KEY'),
    base_url="https://api.deepseek.com"
)

# 发送请求
response = client.chat.completions.create(
    model="deepseek-chat",
    messages=[
        {"role": "system", "content": "You are a helpful assistant"},
        {"role": "user", "content": "Hello"},
    ],
    stream=False
)

# 输出结果
print(response.choices[0].message.content)
```
## 第三方包管理命令（以 `openai` 为例）

- **安装最新版本**：`pip install openai`
- **安装指定版本**：`pip install openai==2.13.0`
- **卸载软件包**：`pip uninstall openai`
- **列出已安装的所有包**：`pip list`
- **查看指定包的详细信息**：`pip show openai`
