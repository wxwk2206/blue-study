#### 泛型
限定集合当中的数据类型 <数据类型>

## 基本数据类型对应的包装类
集合当中无法添加基本数据类型，只能添加引用数据类型

如果一定要在集合当中添加基础数据类型，那么可以转换成对应的包装类
