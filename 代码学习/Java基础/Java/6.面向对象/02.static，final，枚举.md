## static关键字
static：表示静态，是Java的修饰符，用来修饰（成员变量/成员方法）

#### 修饰成员变量
**特点：**

+ 叫做静态变量，被该类所有对象**共享**
+ 不属于对象，属于类
+ 随着类的加载而加载，优先于对象而存在

**调用方法：**

+ 类名调用（推荐）
+ 对象名调用

#### 修饰成员方法
**特点：**

+ static修饰的方法叫做静态方法
+ 该方法多用在**测试类**和**工具类**当中
+ JavaBean类中很少会用

**调用方法：**

+ 类名调用（推荐）
+ 对象名调用

## 枚举
当对象的个数是有限个时，就不能用JavaBean的形式

枚举是一个特殊类型的JavaBean类，这个类的对象是有限个

如订单只有以下六个对象

### 枚举定义格式
```java
public enum 枚举类名{
    枚举项1（参数），枚举项2（参数），枚举项3（参数），...;
    属性
    行为
}
```

**注意事项：**

+ 所有枚举项默认使用public static final修饰
+ 每一个枚举项，都是该枚举类的对象，每一个对象都是通过构造方法创建出来的
+ 枚举类的第一行必须是枚举项，枚举项之间用逗号隔开，以分号作为结尾
+ 枚举类的构造方法必须是private修饰，不让外界创建本类的对象
+ 编译器会给枚举类新增两个默认的方法：values（），valueOf（）

values（）表示获取本类的所有枚举项

valueOf（）表示获取一个指定的枚举项
