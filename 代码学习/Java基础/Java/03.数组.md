数组：是一种容器，可以用来存储同种数据类型的多个值

+ 特点1：连续的空间
+ 特点2：数组一旦定义，长度不可改变

## 数组的静态初始化
初始化：指在定义**变量，数组，对象**的时候进行赋值

静态：在定义**变量，数组，对象**的时候，数据是静止的，确定的

静态初始化：创建数组的时候，直接给数组赋值
