# 1.简介
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">在 Mybatis 中，既可以通过注解配置 SQL 语句，也可以通过 XML 配置文件配置 SQL 语句。</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">默认规则：</font>
1. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">XML 映射文件的名称与 Mapper 接口名称一致，并且将 XML 映射文件和 Mapper 接口放置在相同包下</font>**<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（同包同名）</font>**<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">。</font>
2. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">XML 映射文件的 namespace 属性为 Mapper 接口全限定名一致。</font>
3. <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">XML 映射文件中 sql 语句的 id 与 Mapper 接口中的方法名一致，并保持返回类型一致。</font>

在 Mybatis 的开发中，到底**使用注解开发还是使用 XML **开发？

使用 Mybatis 的注解，主要是来完成一些简单的增删改查功能。如果需要实现复杂的 SQL 功能，建议使用 XML 来配置映射语句。
