## 4.排序查询order by
<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/62702379/1772961556781-ac6d48d4-017a-4aac-a463-5f9418af14d9.png)

**排序方式**：升序（asc），降序（desc）；默认为升序asc，可以不写

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/62702379/1772961665531-be10e97e-d6e6-4cac-ad75-8996bddc641c.png)

### SQL注入相关：
**order by 常数** 是按第常数列排序，如 order by 3 就是按第三列排序

以此方法可以知道数据表中有多少列

在新版本中 **order by 常数 ，**当常数超过列数时不会报错，而是按常数排序，即**不排序**

**新方法：**用union select null，...试字段数
