# C 语言语法基础

## 1. C 程序基本结构

```c
#include <stdio.h>

int main()
{
    printf("hello world \n");

    return 0;
}
```

- `#include <stdio.h>`：预处理指令，引入标准输入输出头文件。
- `.c`：C 语言源文件。
- `.h`：头文件，通常用于放函数声明、变量声明等。
- `int main()`：主函数，程序从 `main` 函数开始执行。
- `{}`：表示函数或代码块的作用范围。
- `return 0;`：主函数正常结束。

## 3. 常见数据类型

| 类型 | 常见大小 | 示例 | 说明 |
| --- | --- | --- | --- |
| `int` | 4 字节 | `100` | 整数 |
| `float` | 4 字节 | `3.14` | 单精度浮点数 |
| `double` | 8 字节 | `3.14` | 双精度浮点数 |
| `char` | 1 字节 | `'A'` | 单个字符 |
| `long` | 4 字节 | `100L` | 长整型 |
| `long long` | 8 字节 | `100LL` | 更长的整型 |

注意：

- 字符串使用双引号，例如 `"hello"`。
- 字符使用单引号，例如 `'h'`。
- 变量使用前通常要先定义类型。

```c
int a = 100;
int b = 200;
int c = a + b;

printf("%d \n", c);
```

## 4. `printf` 格式化输出

`printf` 可以通过格式字符控制输出的数据类型。

| 格式字符        | 作用                               |
| ----------- | -------------------------------- |
| `%d` / `%i` | 以带符号十进制形式输出整数                    |
| `%u`        | 以无符号十进制形式输出整数                    |
| `%o`        | 以无符号八进制形式输出整数                    |
| `%x` / `%X` | 以无符号十六进制形式输出整数，`x` 输出小写，`X` 输出大写 |
| `%c`        | 输出单个字符                           |
| `%s`        | 输出字符串                            |
| `%f`        | 以小数形式输出单精度或双精度数，默认保留 6 位小数       |
| `%e` / `%E` | 以指数形式输出实数                        |
| `%g` / `%G` | 自动选择 `%f` 或 `%e` 中较短的格式          |

示例：

```c
int a = 100;
printf("%d \n", a);

char ch = 'h';
printf("%c \n", ch);

printf("%s \n", "hello world");
```

## 5. 输入 `scanf`

`scanf` 用于从键盘读取输入。格式字符串要和输入格式严格匹配。
         （必须 &，因为要把输入的值存进 i 里）
```c
int a;
int b;
int c;

scanf("%d,%d,%d", &a, &b, &c);
printf("a: %d  b: %d  c: %d \n", a, b, c);
```

如果写的是：
```c
scanf("%d,%d,%d", &a, &b, &c);
```

输入时就应使用逗号分隔：
```text
1,2,3
```

如果写的是：
```c
scanf("%d %d %d", &a, &b, &c);
```

输入时就应使用空格分隔：
```text
1 2 3
```

### 多分支判断
成绩等级示例：
```c
int a;
scanf("%d", &a);

if (a >= 80 && a <= 100) {
    printf("成绩等级：优秀\n");
}
else if (a >= 60 && a < 80) {
    printf("成绩等级：良好\n");

    printf("成绩等级：不及格\n");
}
else {
    printf("成绩无效\n");
}
```

### 嵌套判断：求三个数最大值
```c
int a;
int b;
int c;
int max;

scanf("%d,%d,%d", &a, &b, &c);

if (a > b)
{
    if (a > c)
        max = a;
    else
        max = c;
}
else
{
    if (b > c)
        max = b;
    else
        max = c;
}

```

## 8.`switch` 选择结构

`switch` 适用于将一个变量与多个常量值进行比较的场景，比多层 `if...else if` 更清晰。

**基本格式：**
```c
switch (表达式)
{
    case 常量1:
        语句;
        break;
    case 常量2:
        语句;
        break;
    default:
        语句;
        break;
}
```

**执行过程：**

1. 计算 `switch` 后面括号中表达式的值。
2. 从上到下依次与每个 `case` 后面的常量比较。
3. 匹配到某个 `case` 后，从该位置开始执行代码。
4. 遇到 `break` 时跳出 `switch` 结构。
5. 如果所有 `case` 都不匹配，执行 `default` 分支（`default` 可以省略）。

**示例：用 `switch` 实现成绩等级判断**
```c
int score;
scanf("%d", &score);

// 将分数映射到等级编号
int grade;
if (score >= 80 && score <= 100)
    grade = 1;
else if (score >= 60 && score < 80)
    grade = 2;
else if (score >= 0 && score < 60)
    grade = 3;
else
    grade = 0;

switch (grade)
{
    case 1:
        printf("成绩等级：优秀\n");
        break;
    case 2:
        printf("成绩等级：良好\n");
        break;
    case 3:
        printf("成绩等级：不及格\n");
        break;
    default:
        printf("成绩无效\n");
        break;
}
```

**示例：星期判断**
```c
int day;
scanf("%d", &day);

switch (day)
{
    case 1:
        printf("星期一\n");
        break;
    case 2:
        printf("星期二\n");
        break;
    case 3:
        printf("星期三\n");
        break;
    case 4:
        printf("星期四\n");
        break;
    case 5:
        printf("星期五\n");
        break;
    case 6:
        printf("星期六\n");
        break;
    case 7:
        printf("星期日\n");
        break;
    default:
        printf("无效的星期编号\n");
        break;
}
```

**示例：多个 `case` 共享同一段代码**
```c
int month;
scanf("%d", &month);

switch (month)
{
    case 1:
    case 2:
    case 3:
        printf("第一季度\n");
        break;
    case 4:
    case 5:
    case 6:
        printf("第二季度\n");
        break;
    case 7:
    case 8:
    case 9:
        printf("第三季度\n");
        break;
    case 10:
    case 11:
    case 12:
        printf("第四季度\n");
        break;
    default:
        printf("无效月份\n");
        break;
}
```

**注意事项：**

- `switch` 后面括号中的表达式必须是整型（`int`、`char` 等），不能是浮点型（`float`、`double`）。
- 每个 `case` 后面必须是常量，不能是变量。
- **`break` 很重要**——如果某个 `case` 没有 `break`，程序会继续执行下一个 `case` 的代码（称为"穿透"或 fall-through），直到遇到 `break` 或 `switch` 结束。
- `default` 可以放在任意位置（不一定在最后），但习惯上放在最后。
- `switch` 与 `if...else if` 的区别：`switch` 适合**等值比较**（变量 == 常量），`if` 适合**范围比较**（变量 > 某值、变量 < 某值）。

**`switch` vs `if...else if` 对照：**

| 场景 | 推荐 |
|------|------|
| 判断 1~7 对应的星期 | `switch`（等值比较，结构清晰） |
| 判断分数区间（0~59、60~79...） | `if...else if`（范围比较） |
| 分支数量很多（5 个以上等值比较） | `switch`（可读性更好） |

## 9. 变量交换
题目：如果 `a > b`，交换 `a` 和 `b` 的值，否则不做任何操作。
```c
int a = 350;
int b = 100;
int tmp;

if (a > b)
{
    tmp = a;
    a = b;
    b = tmp;
}

printf("a: %d  b: %d \n", a, b);
```

交换变量时需要第三个临时变量 `tmp`：

1. 先把 `a` 保存到 `tmp`。
2. 再把 `b` 赋值给 `a`。
3. 最后把 `tmp` 赋值给 `b`。

## 10. `for` 循环
### 基本格式
```c
for (初始化; 条件表达式; 更新表达式)
{
    循环体;
}
```

执行过程：

1. 执行初始化语句。
2. 判断条件表达式，如果为真，执行循环体。
3. 执行更新表达式。
4. 回到第 2 步继续判断。

### 1 加到 n
```c
int a = 0;
int b;
int i;

scanf("%d", &b);

for (i = 1; i <= b; i++) {
    a = a + i;
}

printf("1加到%d，总计：%d \n", b, a);
```

### 输出两个数之间能被 3 整除的数字
```c
int a;
int b;
int i;

scanf("%d,%d", &a, &b);

for (i = a; i <= b; i++) {
    if (i % 3 == 0)
        printf("被3整除的数字%d \n", i);
}
```

### 打印三角形
```c
int i;
int j;

for (i = 1; i <= 5; i++) {
    for (j = 1; j <= i; j++) {
        printf("*");
    }
    printf("\n");
}
```

输出效果：
```text
*
**
***
****
*****
```

### 九九乘法表
```c
int i;
int j;
int sum;

for (i = 1; i <= 9; i++) {
    for (j = 1; j <= i; j++) {
        sum = i * j;
        printf("%d*%d=%d ", j, i, sum);
    }
    printf("\n");
}
```

## 11. `while` 循环

### 基本格式
```c
while (条件表达式)
{
    语句;
}
```

执行过程：

1. 初始化循环变量。
2. 判断条件表达式。
3. 如果条件为真，执行循环体。
4. 更新循环变量。
5. 回到第 2 步继续判断。

示例：
```c
int i = 0;
int j = 0;

while (i < 3)
{
    printf("%d   \n", i);
    j = 0;

    while (j < 5)
    {
        if (j == 3)
            break;

        printf("%d    ", j);
        j++;
    }

    i++;
    printf("\n");
}
```

注意：循环变量要记得更新，否则容易出现死循环。

## 12. `do...while` 循环

### 基本格式
```c
do {
    语句;
} while (条件表达式);
```

执行过程：

1. 先执行一次循环体。
2. 判断 `while` 条件表达式。
3. 如果条件为真，继续执行循环体；如果为假，退出循环。

示例：
```c
int i = 1;

do {
    printf("%d   ", i);
    i++;
} while (i < 10);
```

注意：`do...while` 结尾的 `while (条件表达式);` 后面有分号。

## 13. `break` 和 `continue`

### `break`

`break` 用于直接结束当前循环，跳出循环体。
```c
for (int j = 0; j < 5; j++)
{
    if (j == 3)
        break;

    printf("%d    ", j);
}
```

输出：
```text
0    1    2
```

注意：`break` 默认只能跳出一层循环。

### `continue`

`continue` 用于结束本次循环，直接进入下一次循环判断。
```c
for (int j = 0; j < 5; j++)
{
    if (j == 3)
        continue;

    printf("%d    ", j);
}
```

输出：
```text
0    1    2    4
```
