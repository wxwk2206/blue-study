## 1. C 程序基本结构

```c
#include <stdio.h>

int main()
{
    printf("hello world \n");

    return 0;
}
```

- `#include <stdio.h>`：预处理指令，引入标准输入输出头文件。
- `.c`：C 语言源文件。
- `.h`：头文件，通常用于放函数声明、变量声明等。
- `int main()`：主函数，程序从 `main` 函数开始执行。
- `{}`：表示函数或代码块的作用范围。
- `return 0;`：主函数正常结束。

## 3. 常见数据类型

| 类型 | 常见大小 | 示例 | 说明 |
| --- | --- | --- | --- |
| `int` | 4 字节 | `100` | 整数 |
| `float` | 4 字节 | `3.14` | 单精度浮点数 |
| `double` | 8 字节 | `3.14` | 双精度浮点数 |
| `char` | 1 字节 | `'A'` | 单个字符 |
| `long` | 4 字节 | `100L` | 长整型 |
| `long long` | 8 字节 | `100LL` | 更长的整型 |

注意：

- 字符串使用双引号，例如 `"hello"`。
- 字符使用单引号，例如 `'h'`。
- 变量使用前通常要先定义类型。

```c
int a = 100;
int b = 200;
int c = a + b;

printf("%d \n", c);
```

## 4. `printf` 格式化输出

`printf` 可以通过格式字符控制输出的数据类型。

| 格式字符 | 作用 |
| --- | --- |
| `%d` / `%i` | 以带符号十进制形式输出整数 |
| `%u` | 以无符号十进制形式输出整数 |
| `%o` | 以无符号八进制形式输出整数 |
| `%x` / `%X` | 以无符号十六进制形式输出整数，`x` 输出小写，`X` 输出大写 |
| `%c` | 输出单个字符 |
| `%s` | 输出字符串 |
| `%f` | 以小数形式输出单精度或双精度数，默认保留 6 位小数 |
| `%e` / `%E` | 以指数形式输出实数 |
| `%g` / `%G` | 自动选择 `%f` 或 `%e` 中较短的格式 |

示例：

```c
int a = 100;
printf("%d \n", a);

char ch = 'h';
printf("%c \n", ch);

printf("%s \n", "hello world");
```

## 5. 输入 `scanf`

`scanf` 用于从键盘读取输入。格式字符串要和输入格式严格匹配。

```c
int a;
int b;
int c;

scanf("%d,%d,%d", &a, &b, &c);
printf("a: %d  b: %d  c: %d \n", a, b, c);
```

如果写的是：

```c
scanf("%d,%d,%d", &a, &b, &c);
```

输入时就应使用逗号分隔：

```text
1,2,3
```

如果写的是：

```c
scanf("%d %d %d", &a, &b, &c);
```

输入时就应使用空格分隔：

```text
1 2 3
```

## 7. 条件判断

### `if...else`

```c
{
    printf("a > b \n");
}
else
{
    printf("a < b \n");

```

### 多分支判断

成绩等级示例：

```c
int a;
scanf("%d", &a);

if (a >= 80 && a <= 100) {
    printf("成绩等级：优秀\n");
}
else if (a >= 60 && a < 80) {
    printf("成绩等级：良好\n");
}
else if (a >= 0 && a < 60) {
    printf("成绩等级：不及格\n");
}
else {
    printf("成绩无效\n");
}
```

### 嵌套判断：求三个数最大值

```c
int a;
int b;
int c;
int max;

scanf("%d,%d,%d", &a, &b, &c);

if (a > b)
{
    if (a > c)
        max = a;
    else
        max = c;
}
else
{
    if (b > c)
        max = b;
    else
        max = c;
}

printf("max: %d \n", max);
```

## 8. 变量交换

题目：如果 `a > b`，交换 `a` 和 `b` 的值，否则不做任何操作。

```c
int a = 350;
int b = 100;
int tmp;

if (a > b)
{
    tmp = a;
    a = b;
    b = tmp;
}

printf("a: %d  b: %d \n", a, b);
```

交换变量时需要第三个临时变量 `tmp`：

1. 先把 `a` 保存到 `tmp`。
2. 再把 `b` 赋值给 `a`。
3. 最后把 `tmp` 赋值给 `b`。

## 9. `for` 循环

### 基本格式

```c
for (初始化; 条件表达式; 更新表达式)
{
    循环体;
}
```

执行过程：

1. 执行初始化语句。
2. 判断条件表达式，如果为真，执行循环体。
3. 执行更新表达式。
4. 回到第 2 步继续判断。

### 1 加到 n

```c
int a = 0;
int b;
int i;

scanf("%d", &b);

for (i = 1; i <= b; i++) {
    a = a + i;
}

printf("1加到%d，总计：%d \n", b, a);
```

### 输出两个数之间能被 3 整除的数字

```c
int a;
int b;
int i;

scanf("%d,%d", &a, &b);

for (i = a; i <= b; i++) {
    if (i % 3 == 0)
        printf("被3整除的数字%d \n", i);
}
```

### 打印三角形

```c
int i;
int j;

for (i = 1; i <= 5; i++) {
    for (j = 1; j <= i; j++) {
        printf("*");
    }
    printf("\n");
}
```

输出效果：

```text
*
**
***
****
*****
```

### 九九乘法表

```c
int i;
int j;
int sum;

for (i = 1; i <= 9; i++) {
    for (j = 1; j <= i; j++) {
        sum = i * j;
        printf("%d*%d=%d ", j, i, sum);
    }
    printf("\n");
}
```

## 10. `while` 循环

### 基本格式

```c
while (条件表达式)
{
    语句;
}
```

执行过程：

1. 初始化循环变量。
2. 判断条件表达式。
3. 如果条件为真，执行循环体。
4. 更新循环变量。
5. 回到第 2 步继续判断。

示例：

```c
int i = 0;
int j = 0;

while (i < 3)
{
    printf("%d   \n", i);
    j = 0;

    while (j < 5)
    {
        if (j == 3)
            break;

        printf("%d    ", j);
        j++;
    }

    i++;
    printf("\n");
}
```

注意：循环变量要记得更新，否则容易出现死循环。

## 11. `do...while` 循环

### 基本格式

```c
do {
    语句;
} while (条件表达式);
```

执行过程：

1. 先执行一次循环体。
2. 判断 `while` 条件表达式。
3. 如果条件为真，继续执行循环体；如果为假，退出循环。

示例：

```c
int i = 1;

do {
    printf("%d   ", i);
    i++;
} while (i < 10);
```

注意：`do...while` 结尾的 `while (条件表达式);` 后面有分号。

## 12. `break` 和 `continue`

### `break`

`break` 用于直接结束当前循环，跳出循环体。

```c
for (int j = 0; j < 5; j++)
{
    if (j == 3)
        break;

    printf("%d    ", j);
}
```

输出：
```text
0    1    2
```

注意：`break` 默认只能跳出一层循环。

### `continue`

`continue` 用于结束本次循环，直接进入下一次循环判断。
```c
for (int j = 0; j < 5; j++)
{
    if (j == 3)
        continue;

    printf("%d    ", j);
}
```

输出：
```text
0    1    2    4
```
