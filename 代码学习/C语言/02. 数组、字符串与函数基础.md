
## 一、一维数组

### 1.1 定义与初始化

```c
// 完整初始化：指定长度，给出所有元素
int a[8] = {100, 200, 300, 400, 500, 600, 700, 800};

// 自动推断长度：不写长度，编译器根据初始化列表确定
int a[] = {100, 200, 300, 400, 500, 600, 700, 800};

// 部分初始化：未指定的元素自动赋 0
int a[8] = {100, 200, 300, 400, 500};  // 后三个元素为 0

// 全零初始化
int a[8] = {0};

// 仅声明，不初始化（值是垃圾值）
int a[8];
```

> **核心概念**：数组是**相同数据类型**元素的集合，在内存中**连续存储**。

### 1.2 数组下标

| 规则 | 说明 |
|------|------|
| 起始下标 | **0**（第一个元素是 `a[0]`） |
| 末尾下标 | **长度 - 1**（如 `a[8]` 的有效下标为 0~7） |
| 越界访问 | `a[8]` 访问的是**第 9 个元素**，属于**数组越界**，C 语言不报错但行为未定义 |

### 1.3 遍历数组

```c
int i;
for (i = 0; i < 8; i++)
    printf("%d  ", a[i]);
```

## 二、字符与字符串

### 2.1 char 类型

- `char` 占用 1 字节，值范围 **0~127**（ASCII 标准字符），128~255 不可靠。
- 在 0~127 范围内，**char 和 int 可以通用**，只是显示格式不同：
  - `%c`：以字符形式输出
  - `%d`：以十进制整数值输出
  - `%x`：以十六进制输出

```c
char ch = 'h';
printf("%c  %d  %x\n", ch, ch, ch);  // h  104  68
```

### 2.2 字符数组 vs 字符串

| 对比项 | 字符数组 | 字符串 |
|--------|---------|--------|
| 初始化方式 | `{'h','e','l','l','o'}` | `"hello"` |
| 结尾标记 | **无** | 自动追加 `\0` |
| 用 `%s` 打印 | ❌ 不可靠（无结束标记） | ✅ 正确 |

```c
// 字符数组（逐个赋值，没有 \0 结尾）
char c[11] = {'h', 'e', 'l', 'l', 'o', 'w', 'o', 'r', 'l', 'd'};

// 字符串（系统自动在末尾加 \0）
char c[11] = "helloworld";  // 实际存储：helloworld\0
```

### 2.3 字符串结束标记 `\0`

`\0`（空字符，ASCII 码为 0）是 C 语言字符串的**唯一结束标志**，所有字符串操作函数都依赖它。

```c
// \0 的截断效果
char c[100] = "hellhh\0howorld";
// printf("%s", c);  输出：hellhh（\0 后面的内容被忽略）

// strlen 也只计算到 \0 之前
int n = strlen(c);  // n = 6
```

### 2.4 计算字符串长度

```c
// 方式一：for 循环配合 \0 检测
int count = 0;
for (int i = 0; i < 100; i++) {
    if (c[i] != '\0')
        count++;
    else
        break;
}

// 方式二：while 循环配合 \0 检测
int count = 0, i = 0;
while (c[i] != '\0') {
    count++;
    i++;
}

// 方式三：库函数 strlen()（需 #include <string.h>）
int n = strlen(c);
```

### 2.5 scanf 输入字符串

```c
char str[100];
scanf("%s", str);  // 注意：数组名本身就是地址，不需要 &
// scanf 遇到空格、回车会停止读取
```

## 三、字符串操作函数

> 需 `#include <string.h>`

### 3.1 strlen —— 求字符串长度

```c
int n = strlen(str);  // 返回 \0 之前的字符个数，不含 \0
```

### 3.2 strcpy / strncpy —— 字符串拷贝

```c
char src[32] = "helloworld";
char dest[16] = {0};

strcpy(dest, src);   // 将 src 拷贝到 dest（不安全，不检查边界）
strncpy(dest, src, n);  // 安全版本，最多拷贝 n 个字符
```

**手动实现**：

```c
int i = 0;
while (src[i] != '\0') {
    dest[i] = src[i];
    i++;
}
dest[i] = '\0';  // 别忘了手动补 \0！
```

> ⚠️ `strcpy` 不检查目标缓冲区大小，可能造成**缓冲区溢出**。推荐使用 `strncpy`。

### 3.3 strcmp —— 字符串比较

```c
char a[] = "abc";
char b[] = "abc";

if (strcmp(a, b) == 0) {
    // 两字符串相等
}
```

| 返回值 | 含义 |
|--------|------|
| `0` | 两字符串相等 |
| `> 0` | 第一个不匹配字符的 ASCII 差值（a > b） |
| `< 0` | 第一个不匹配字符的 ASCII 差值（a < b） |

> ⚠️ 比较字符串**不能**用 `==`，`==` 比较的是地址而不是内容。

## 四、二维数组

### 4.1 定义与初始化

```c
int aa[3][5] = {
    {111, 222, 333, 444, 555},
    {666, 777, 888, 999, 1000},
    {1001, 1002, 1003, 1004, 1005}
};
// 3 行 5 列，内存按行优先存储
```

### 4.2 元素访问

```c
int n = aa[2][2];       // 读取第 3 行第 3 列（下标从 0 开始）→ 1003
aa[2][2] = 1234;        // 修改
```

### 4.3 嵌套循环遍历

```c
int i, j;
for (i = 0; i < 3; i++) {        // 外层：行
    for (j = 0; j < 5; j++) {    // 内层：列
        printf("%d ", aa[i][j]);
    }
    printf("\n");
}
```

## 五、函数

### 5.1 函数定义语法

```c
返回类型  函数名(参数类型 参数名, 参数类型 参数名, ...)
{
    // 函数体
    return 返回值;   // void 类型不需要 return
}
```

```c
// 示例：检查密码
int checkpasswd(int passwd)   // 返回 int，接收一个 int 参数
{
    // ...
    if (ppasswd == target)
        return 1;    // 匹配成功
    return 0;        // 匹配失败
}
```

### 5.2 函数调用

```c
int ret = checkpasswd(777);   // 用变量接收返回值
if (ret == 1)
    printf("passwd ok\n");
```

### 5.3 参数传递（值传递）

```c
int test(int pa)   // pa 是形参，接收的是实参的副本
{
    pa = 200;      // 修改形参，不影响外部实参
}

int main()
{
    int passwd = 100;
    test(passwd);  // passwd 仍然是 100
}
```

> C 语言默认是**值传递**，函数内修改形参不影响实参。

### 5.4 return 语句

- `return` 会**立即退出函数**，返回到**调用处**继续执行。
- 可在函数中多处 `return`，用于提前返回。

```c
// 在循环中匹配到就提前返回
for (i = 0; i < 3; i++) {
    for (j = 0; j < 5; j++) {
        if (passwd == aa[i][j])
            return 1;   // 找到，直接返回
    }
}
return 0;  // 遍历完都没找到
```

## 七、综合练习

### 7.1 字符串逆序输出

```c
char str[100];
scanf("%s", str);

// 计算长度
int len = 0;
while (str[len] != '\0') len++;

// 首尾交换，只需交换 len/2 次
for (int i = 0; i < len / 2; i++) {
    char temp = str[i];
    str[i] = str[len - 1 - i];
    str[len - 1 - i] = temp;
}
printf("%s\n", str);
```

**要点**：交换次数只需 `len/2`，否则会换回去。

### 7.2 统计字符类型（大写/小写/数字）

```c
char str[100];
scanf("%s", str);

int upper = 0, lower = 0, digit = 0;
int i = 0;
while (str[i] != '\0') {
    if (str[i] >= 'A' && str[i] <= 'Z')   upper++;  // 大写字母
    if (str[i] >= 'a' && str[i] <= 'z')   lower++;  // 小写字母
    if (str[i] >= '0' && str[i] <= '9')   digit++;  // 数字
    i++;
}
printf("大写: %d  小写: %d  数字: %d\n", upper, lower, digit);
```

**要点**：利用 ASCII 码的连续性判断范围，不需要记住具体数值。
