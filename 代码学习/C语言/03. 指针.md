## 一、指针与数组

### 核心概念

| 概念 | 说明 |
|------|------|
| 数组名 | 代表数组的**首地址**，等价于 `&a[0]` |
| `*(pa + i)` | 等价于 `a[i]`，访问第 i 个元素 |
| 指针 + 1 | 偏移量 = **数据类型大小**（int +4字节，char +1字节） |
| 数组越界 | C 语言**不检查**，由程序员自己控制 |

### 代码示例

```c
#include <stdio.h>

// 数组名代表数组的首地址
// 指针变量 + 1 是指：指针变量的数据类型 + 1
// （int 指针 +4 个字节，char 指针 +1 个字节）
int test(int* pa, int len)
{
    printf(" a[0]: %d \n", *pa);
    printf(" a[1]: %d \n", *(pa + 1));
    printf(" a[2]: %d \n", *(pa + 2));

    int i;
    for(i = 0; i < len; i++)
        printf("%d   ", *(pa + i));
}

int main()
{
    int a[5] = {11, 22, 33, 44, 55};
    test(a, 5);   // 传入首地址 + 长度
    return 0;
}
```

> [!tip] 要点
> 函数参数写 `int* pa` 和写 `int pa[]` **等价**，都是接收一个指向 int 的指针。
> 传数组时必须同时传长度，因为函数内无法用 `sizeof` 获得原始数组大小。

## 二、指针与字符串

### 核心概念

- 字符串本质是 `char` 数组，末尾有 `\0`
- 通过 `char*` 指针可以**直接修改**字符串中的字符
- `strlen(ps)` 计算字符串长度（不含 `\0`）

### 代码示例

```c
#include <stdio.h>
#include <string.h>

int test(char* ps)
{
    printf("str1[0]: %c \n", *(ps + 0));  // 'h'
    printf("str1[1]: %c \n", *(ps + 1));  // 'e'
    printf("str1[2]: %c \n", *(ps + 2));  // 'l'

    *(ps + 2) = 'w';   // 修改第 3 个字符 'l' → 'w'

    printf("修改后:\n");
    printf("str1[0]: %c \n", *(ps + 0));  // 'h'
    printf("str1[1]: %c \n", *(ps + 1));  // 'e'
    printf("str1[2]: %c \n", *(ps + 2));  // 'w'

    int len = strlen(ps);
    printf("len: %d \n", len);  // 11
}

int main()
{
    char str1[13] = "hello world";
    test(str1);
    printf("%s \n", str1);  // hewlo world
    return 0;
}
```

> [!warning] 注意区分
> `char str[] = "hello"` → 可修改（栈上字符数组）
> `char* str = "hello"` → **不可修改**（字符串字面量在只读段）

> [!danger] 段错误演示
> 修改只读区的字符串字面量会导致 **Segmentation Fault（段错误）**：

```c
#include <stdio.h>

int main()
{
    char str[13] = "hello world";   // 栈区字符数组，可修改
    char* ps = "hello yanan";       // 指针指向只读区字符串字面量

    printf("ps: %s \n", ps);

    // *(ps + 4) = 'w';   // 尝试写入只读区 → 段错误！

    return 0;
}
```

> [!abstract] 原理
> - `char str[] = "..."` 是把字符串字面量**复制**到栈区，所以可以改。
> - `char* ps = "..."` 是让指针直接指向**只读数据段**中的字面量，改它就炸。

## 三、指针示例

### 核心概念

- 函数内修改变量必须传**地址**（`&a`），接收端用 `*pa` 解引用
- `*` 是"解引用"操作符：取指针所指地址的值

### 1.swap 函数（交换函数）

```c
void swap(int *pa, int *pb)
{
    int tmp;
    tmp = *pb;
    *pb = *pa;
    *pa = tmp;
}
```

### 2.三数排序（冒泡思路，三次比较）

```c
int test(int* pa, int* pb, int* pc)
{
    if(*pa > *pb) swap(pa, pb);  // 最小值换到 a
    if(*pa > *pc) swap(pa, pc);  // 最小值确定在 a
    if(*pb > *pc) swap(pb, pc);  // b、c 排好序
}

int main()
{
    int a = 300, b = 2000, c = 1;
    test(&a, &b, &c);
    printf("%d  %d  %d \n", a, b, c);  // 1  300  2000
    return 0;
}
```

### 3.指针实现有序插入

题目：在已排序的一维数组中插入一个新数，保持数组依然有序。**全程用指针操作**。

#### 代码实现
```c
#include <stdio.h>

// 基于你的代码框架实现插入排序功能
int test(int* pa, int len)
{
    // 1. 取出用户放在数组最后一位的待插入元素
    int insert = *(pa + len - 1);
    
    // 2. 初始有效元素个数是5（你在main函数中定义的前5个有序数）
    int currentSize = 5;
    
    // 3. 用指针找到正确的插入位置
    int *pos = pa;
    // 遍历到第一个大于等于待插入数的位置
    while (pos < pa + currentSize && *pos < insert) {
        pos++;
    }
    
    // 4. 用指针将插入位置及之后的元素向后移动一位
    // 从最后一个有效元素开始移动，避免数据覆盖
    int *p = pa + currentSize;
    while (p > pos) {
        *p = *(p - 1);
        p--;
    }
    
    // 5. 插入新元素到正确位置
    *pos = insert;
    
    return 0;
}

int main()
{
	int i;
	int b;
	int a[10] = {11, 22, 33, 44, 55};
	
	scanf("%d", &b);
	a[9] = b;  // 你原来的写法：把输入的数放在数组最后一位
	test(a, 10);
	
	for(i=0; i<10; i++)
		printf("%d  ", a[i]);
	
	return 0;
}
```

## 四、易错点

> [!bug] 常见错误
> 1. **数组越界**：`*(pa + 99)` C 编译器不报错，运行时行为未定义
> 2. **修改字符串字面量**：`char* s = "abc"; *(s+1) = 'x';` → 段错误
> 3. **传值 vs 传址**：想在函数内改变外部变量，必须传 `&变量`，不能传值
> 4. **忘记传长度**：函数内 `sizeof(pa)` 只返回指针大小（4或8字节），不是数组长度

## 五、二维数组与数组指针

### 核心概念

| 概念 | 说明 |
|------|------|
| 数组指针 | `int (*pa)[3]` —— 指向含有 3 个 int 元素的一维数组的指针 |
| `[3]` | 表示二维数组的**列数** |
| `*(*(pa + i) + j)` | 等价于 `pa[i][j]`，访问第 i 行第 j 列 |

#### 数组指针和指针数组的区别
 `int *pa[3]` → 指针数组（存放 3 个 `int*`）
 `int (*pa)[3]` → 数组指针（指向 `int[3]` 的指针）

### 访问方式对照

| 方式 | 含义 |
|------|------|
| `a[i][j]` | 直观下标访问 |
| `*(*(a + i) + j)` | 指针偏移访问 |
| `(*(a + i))[j]` | 混合写法 |

> [!note] 图解
> 二维数组 `a[2][3]` 在内存中是**连续线性**存储的：
> ```
> a[0][0]  a[0][1]  a[0][2]  a[1][0]  a[1][1]  a[1][2]
> [  1  ]  [  2  ]  [  3  ]  [  4  ]  [  5  ]  [  6  ]
> ```
> `a` 指向第 0 行，`a+1` 指向第 1 行（跳过 3 个 int = 12 字节）。

### 矩阵转置示例（2×3 → 3×2）

```c
#include <stdio.h>

void test(int (*pa)[3])       // 数组指针接收二维数组
{
    int b[3][2] = {{0}};
    int (*pb)[2] = b;          // 指向目标数组的数组指针
    int i, j;

    // 转置核心：b[j][i] = a[i][j]
    for (i = 0; i < 2; i++)
    {
        for (j = 0; j < 3; j++)
        {
            *(*(pb + j) + i) = *(*(pa + i) + j);
        }
    }

    printf("转置结果 b[3][2]：\n");
    for (i = 0; i < 3; i++)
    {
        for (j = 0; j < 2; j++)
        {
            printf("%d  ", *(*(pb + i) + j));
        }
        printf("\n");
    }
}

int main()
{
    int a[2][3] = {1, 2, 3,
                    4, 5, 6};
    test(a);    // 传入数组名（首地址）
    return 0;
}
```

> [!tip] 关键理解
> `*(*(pb + j) + i) = *(*(pa + i) + j)`
> - 左边：`j` 是行（目标 3 行），`i` 是列（目标 2 列）
> - 右边：`i` 是行（来源 2 行），`j` 是列（来源 3 列）
> - 这就是把 `(i,j)` 位置的值赋给 `(j,i)` 位置 → **转置**
