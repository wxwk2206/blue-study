## 一、指针与数组

### 核心概念

| 概念          | 说明                                   |
| ----------- | ------------------------------------ |
| 数组名         | 代表数组的**首地址**，等价于 `&a[0]`             |
| `*(pa + i)` | 等价于 `a[i]`，访问第 i 个元素                 |
| 指针 + 1      | 偏移量 = **数据类型大小**（int +4字节，char +1字节） |
| 数组越界        | C 语言**不检查**，由程序员自己控制                 |
| \*p++       | 等价于 \*(p++) —— 先取 \*p 的值，然后 p 自增     |

### 代码示例

```c
#include <stdio.h>

// 数组名代表数组的首地址
// 指针变量 + 1 是指：指针变量的数据类型 + 1
// （int 指针 +4 个字节，char 指针 +1 个字节）
int test(int* pa, int len)
{
    printf(" a[0]: %d \n", *pa);
    printf(" a[1]: %d \n", *(pa + 1));
    printf(" a[2]: %d \n", *(pa + 2));

    int i;
    for(i = 0; i < len; i++)
        printf("%d   ", *(pa + i));
}

int main()
{
    int a[5] = {11, 22, 33, 44, 55};
    test(a, 5);   // 传入首地址 + 长度
    return 0;
}
```

> [!tip] 要点
> 函数参数写 `int* pa` 和写 `int pa[]` **等价**，都是接收一个指向 int 的指针。
> 传数组时必须同时传长度，因为函数内无法用 `sizeof` 获得原始数组大小。

## 二、指针与字符串

### 核心概念

- 字符串本质是 `char` 数组，末尾有 `\0`
- 通过 `char*` 指针可以**直接修改**字符串中的字符
- `strlen(ps)` 计算字符串长度（不含 `\0`）

### 代码示例

```c
#include <stdio.h>
#include <string.h>

int test(char* ps)
{
    printf("str1[0]: %c \n", *(ps + 0));  // 'h'
    printf("str1[1]: %c \n", *(ps + 1));  // 'e'
    printf("str1[2]: %c \n", *(ps + 2));  // 'l'

    *(ps + 2) = 'w';   // 修改第 3 个字符 'l' → 'w'

    printf("修改后:\n");
    printf("str1[0]: %c \n", *(ps + 0));  // 'h'
    printf("str1[1]: %c \n", *(ps + 1));  // 'e'
    printf("str1[2]: %c \n", *(ps + 2));  // 'w'

    int len = strlen(ps);
    printf("len: %d \n", len);  // 11
}

int main()
{
    char str1[13] = "hello world";
    test(str1);
    printf("%s \n", str1);  // hewlo world
    return 0;
}
```

> [!warning] 注意区分
> `char str[] = "hello"` → 可修改（栈上字符数组）
> `char* str = "hello"` → **不可修改**（字符串字面量在只读段）

> [!danger] 段错误演示
> 修改只读区的字符串字面量会导致 **Segmentation Fault（段错误）**：

```c
#include <stdio.h>

int main()
{
    char str[13] = "hello world";   // 栈区字符数组，可修改
    char* ps = "hello yanan";       // 指针指向只读区字符串字面量

    printf("ps: %s \n", ps);

    // *(ps + 4) = 'w';   // 尝试写入只读区 → 段错误！

    return 0;
}
```

> [!abstract] 原理
> - `char str[] = "..."` 是把字符串字面量**复制**到栈区，所以可以改。
> - `char* ps = "..."` 是让指针直接指向**只读数据段**中的字面量，改它就炸。

## 三、指针示例

### 核心概念

- 函数内修改变量必须传**地址**（`&a`），接收端用 `*pa` 解引用
- `*` 是"解引用"操作符：取指针所指地址的值

### 1.swap 函数（交换函数）

```c
void swap(int *pa, int *pb)
{
    int tmp;
    tmp = *pb;
    *pb = *pa;
    *pa = tmp;
}
```

### 2.三数排序（冒泡思路，三次比较）

```c
int test(int* pa, int* pb, int* pc)
{
    if(*pa > *pb) swap(pa, pb);  // 最小值换到 a
    if(*pa > *pc) swap(pa, pc);  // 最小值确定在 a
    if(*pb > *pc) swap(pb, pc);  // b、c 排好序
}

int main()
{
    int a = 300, b = 2000, c = 1;
    test(&a, &b, &c);
    printf("%d  %d  %d \n", a, b, c);  // 1  300  2000
    return 0;
}
```

### 3.指针实现有序插入

题目：在已排序的一维数组中插入一个新数，保持数组依然有序。**全程用指针操作**。

```c
#include <stdio.h>

#define MAX 100  // 数组最大容量

/* ⭐ 排序数组插入——纯指针操作，禁止使用 [] */
void insert(int* arr, int* size, int newVal) {
    int* p   = arr;             // 游标指针，用于查找插入位置
    int* end = arr + *size;     // 尾后指针（arr[size] 的位置）

    // ⭐ 第1步：找插入位置——第一个大于 newVal 的位置
    while (p < end && *p < newVal) {
        p++;  // p 每次后移一个 int 的距离
    }
    // 循环结束后，p 指向应插入的位置

    // ⭐ 第2步：从后往前移动元素，腾出空间
    int* q = end;               // q 从尾后开始
    while (q > p) {              // 只要还没退到插入位置
        *q = *(q - 1);        // 把前一个元素复制到当前位置
        q--;                   // q 往前移
    }

    // ⭐ 第3步：在空位写入新值
    *p = newVal;
    (*size)++;                  // ⚠️ 注意括号！先解引用再自增
}

int main() {
    int a[MAX] = {2, 5, 8, 12, 16};  // 已排序数组
    int n = 5;                       // 当前有效元素个数

    insert(a, &n, 10);  // 插入 10

    // 打印结果——用指针遍历
    int* p = a;
    while (p < a + n) {
        printf("%d ", *p++);
    }
    printf("\n");  // 输出：2 5 8 10 12 16

    return 0;
}
```

> [!bug]  常见翻车点：
> 1.从前往后移元素 → 覆盖还没读的数据，结果全是重复值
> 2.忘记 (*size)++ 的括号 → *size++ 是先移动指针再加
> 3.用 p++ 遍历完忘了 p 已经指向新位置 → 插入位置错误
> 4.没判断数组满 → 越界写入，经典缓冲区溢出

改造题目：写一个 delete(arr, &n, target) 函数，删除数组中第一个等于 target 的元素，同样全程指针操作。
```c
#include <stdio.h>

#define MAX 100  

//改造：写一个 delete(arr, &n, target) 函数，删除数组中第一个等于 target 的元素，同样全程指针操作。
void delet(int *arr, int *size, int target) {
    int *p   = arr;
    int *end = arr + *size;

    // ⭐第1步：找目标位置
    while (p < end && *p != target) {
        p++;
    }

    // ⭐关键：没找到就直接返回
    if (p == end) {
        printf("未找到 %d，未删除\n", target);
        return;
    }

    // ⭐ 第2步：从目标位置开始，前移后续元素
    int *q = p;
    while (q < end - 1) {       // 注意：end-1，不越界
        *q = *(q + 1);
        q++;
    }

    // ⭐ 第3步：数组长度减1
    (*size)--;
}

int main() {
    int a[MAX] = {2, 5, 8, 12, 16}; 
    int n = 5;                    

	delet(a, &n, 8);
	
    int *p = a;
    while (p < a + n) {
        printf("%d ", *p++);
    }
    printf("\n"); 

    return 0;
}

```

## 四、二维数组与数组指针

### 核心概念

| 概念                 | 说明                                        |
| ------------------ | ----------------------------------------- |
| 数组指针               | `int (*pa)[3]` —— 指向含有 3 个 int 元素的一维数组的指针 |
| `[3]`              | 表示二维数组的**列数**                             |
| `*(*(pa + i) + j)` | 等价于 `pa[i][j]`，访问第 i 行第 j 列               |

#### 数组指针和指针数组的区别
 `int (*pa)[3]` → 数组指针（指向 `int[3]` 的指针）→ **对应二维数组**
  `int *pa[3]` → 指针数组（存放 3 个 `int*`）→ **对应二级指针**
  `指针数组是数组，里面每个元素是一级指针；数组名在表达式中会退化为二级指针（T**）二者类型兼容，可以互相赋值，但本质完全不是同一个东西。`
`C 语言规则：数组名在绝大多数表达式里，会自动转换成指向数组首元素的指针`
`int *arr[5] 数组首元素类型是 int *，指向首元素的指针就是 int **（二级指针）。`
### 访问方式对照

| 方式 | 含义 |
|------|------|
| `a[i][j]` | 直观下标访问 |
| `*(*(a + i) + j)` | 指针偏移访问 |
| `(*(a + i))[j]` | 混合写法 |

> [!note] 图解
> 二维数组 `a[2][3]` 在内存中是**连续线性**存储的：
> ```
> a[0][0]  a[0][1]  a[0][2]  a[1][0]  a[1][1]  a[1][2]
> [  1  ]  [  2  ]  [  3  ]  [  4  ]  [  5  ]  [  6  ]
> ```
> `a` 指向第 0 行，`a+1` 指向第 1 行（跳过 3 个 int = 12 字节）。

### 矩阵转置示例（2×3 → 3×2）

```c
#include <stdio.h>

void test(int (*pa)[3])       // 数组指针接收二维数组
{
    int b[3][2] = {{0}};
    int (*pb)[2] = b;          // 指向目标数组的数组指针
    int i, j;

    // 转置核心：b[j][i] = a[i][j]
    for (i = 0; i < 2; i++)
    {
        for (j = 0; j < 3; j++)
        {
            *(*(pb + j) + i) = *(*(pa + i) + j);
        }
    }

    printf("转置结果 b[3][2]：\n");
    for (i = 0; i < 3; i++)
    {
        for (j = 0; j < 2; j++)
        {
            printf("%d  ", *(*(pb + i) + j));
        }
        printf("\n");
    }
}

int main()
{
    int a[2][3] = {1, 2, 3,
                    4, 5, 6};
    test(a);    // 传入数组名（首地址）
    return 0;
}
```

> [!tip] 关键理解
> `*(*(pb + j) + i) = *(*(pa + i) + j)`
> - 左边：`j` 是行（目标 3 行），`i` 是列（目标 2 列）
> - 右边：`i` 是行（来源 2 行），`j` 是列（来源 3 列）
> - 这就是把 `(i,j)` 位置的值赋给 `(j,i)` 位置 → **转置**
