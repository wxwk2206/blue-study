## 一、void* 空指针类型

### 核心概念

| 关键字     | 用途              | 可修饰     |
| ------- | --------------- | ------- |
| `void`  | 修饰函数，表示无返回值     | 函数返回类型  |
| `void*` | 空指针类型，可接收任意类型地址 | 函数参数、变量 |

> [!important] 核心规则
> `void*` 是 C 中的"万能指针"——可以接收任何类型的地址，但使用前**必须强制转换**为目标类型才能解引用。

### 常见用途：函数参数泛型化

```c
#include <stdio.h>

// ⭐ void* 作为参数，可接收任意类型的指针
int test(void* pp)
{
    // 使用前必须强制转换为实际类型
    char* pps = (char*)pp;         // 将 void* 转换为 char*
    printf("hello pps: %s \n", pps);
}

int main()
{
    char* ps = (char*)"hello world";  // (char*) 强转：防止只读警告
    test((void*)ps);                  // 传入时转换为 void*

    return 0;
}
```

> [!note] 强转 `(char*)` 的意义
> `"hello world"` 是字符串字面量，类型是 `const char*`。
> 赋值给普通 `char*` 时，用 `(char*)` 强转是为了类型匹配，避免编译警告。
> 但此时不要通过该指针修改字符串内容，字符串存在只读区！

### 字符串拷贝：`mystrcpy`

```c
#include <stdio.h>

// ⭐ 手动实现字符串拷贝，逐字符拷贝直到遇到 '\0'
void mystrcpy(char* pdst, char* psrc)
{
    int i = 0;

    while(*(psrc + i) != '\0')     // 遇到字符串结束符停止
    {
        *(pdst + i) = *(psrc + i); // 拷贝当前字符
        i++;
    }
}

int main()
{
    char dst[32] = {0};
    char* src = (char*)"hello world";  // 方案：(char*) 强制类型转换

    mystrcpy(dst, src);

    printf("dst: %s \n", dst);   // 输出：dst: hello world

    return 0;
}
```

> [!tip] 两种处理字面量的方案
> - 方案一：`const char* src = "hello world"` — 保留常量修饰，安全
> - 方案二：`char* src = (char*)"hello world"` — 强转消除警告，别去写它

## 二、函数指针与 typedef

### 核心概念

**概括**：函数指针是存放函数入口地址的指针变量。
**比喻**：就像快递单上的地址，告诉系统去哪找代码执行。
**细节**：

| 概念 | 语法 | 说明 |
|------|------|------|
| 函数名 | `test` | 代表函数的**入口地址** |
| 函数指针定义 | `int (*ptest)(int, char*)` | 返回类型 + `(*名)` + 参数列表 |
| typedef 别名 | `typedef int (*ptest)(int, char*)` | 把函数指针类型取个别名，直接用别名声明 |

> [!note] 定义语法拆解
> `int (*ptest)(int, char*)`
> - `int` → 函数返回类型
> - `(*ptest)` → 函数指针变量名（括号不能省！）
> - `(int, char*)` → 函数参数列表

### 代码示例

```c
#include <stdio.h>

// ⭐ 用 typedef 给函数指针类型起别名
typedef int (*ptest)(int, char*);
// 等价于：类型名 ptest 代表"接受 int 和 char* 参数、返回 int 的函数指针"

typedef int typeint_t;   // 给 int 起别名 typeint_t

int test(int a, char* ps)
{
    printf("hello test \n");
    printf("a: %d \n", a);
    printf("ps: %s \n", ps);
    return 0;
}

int main()
{
    typeint_t b = 200;         // 等价于 int b = 200
    printf("b: %d \n", b);

    ptest pp = test;           // ⭐ 用别名声明函数指针，赋值为函数地址
    pp(100, "hello world");    // 通过函数指针调用函数

    return 0;
}
```

> [!tip] 不用 typedef 的写法
> ```c
> int (*ptest)(int, char*) = test;   // 直接定义并赋值
> ptest(100, "hello world");
> ```

### 函数指针的使用场景

```c
// ⭐ 回调函数：把函数指针作为参数传给另一个函数
// 调用者决定"做什么"，被调用者只管"何时做"
void run(ptest func, int a, char* ps)
{
    func(a, ps);   // 间接调用
}
```

> [!abstract] 回调函数 (Callback Function) 原理
> 将函数 A 的地址传给函数 B，由 B 在合适的时机调用 A。
> 常见于：事件处理、排序比较器、信号处理。

## 三、信号与回调函数

### 核心概念

**概括**：`signal()` 注册信号处理函数，当指定信号到来时自动调用。
**比喻**：就像设置了一个"警报响了就执行这段代码"的规则。
**细节**：

| 函数/宏 | 说明 |
|---------|------|
| `signal(signum, handler)` | 注册信号处理函数，`handler` 是函数指针 |
| `alarm(seconds)` | 设置定时器，`seconds` 秒后发送 `SIGALRM` 信号 |
| `SIGALRM` | 定时器信号，由 `alarm()` 触发 |
| `sleep(n)` | 让进程暂停 n 秒 |

### 代码示例

```c
#include <stdio.h>
#include <signal.h>   // signal、SIGALRM
#include <unistd.h>   // alarm、sleep

// ⭐ 信号处理函数，必须接受一个 int 参数（信号编号）
void test(int a)
{
    printf("hello test \n");   // 10 秒后 SIGALRM 触发，打印此行
}

int main()
{
    int count = 0;

    alarm(10);                  // ⭐ 10 秒后发送 SIGALRM 信号
    signal(SIGALRM, test);      // ⭐ 注册处理函数：收到 SIGALRM 就调用 test

    while(1)
    {
        printf("count: %d \n", count);
        count++;
        sleep(1);               // 每秒打印一次计数
    }
}
```

> [!warning] 注意
> `signal()` 的第二个参数就是一个**函数指针**，本质是回调机制的经典应用。
> 此代码依赖 POSIX 接口（`<unistd.h>`、`<signal.h>`），Windows 下行为可能有差异。

## 四、二级指针与指针数组

### 核心概念

**概括**：二级指针存放一级指针的地址；指针数组是存放指针的数组。
**比喻**：一级指针指向数据，二级指针指向"指向数据的那根线"。
**细节**：

| 概念 | 声明 | 说明 |
|------|------|------|
| 二级指针 | `int** ppa` | 存放一级指针的地址，`**ppa` 取最终数据 |
| 指针数组 | `char* pa[4]` | 数组，每个元素都是 `char*` 指针 |
| 数组指针 | `int (*pa)[3]` | 指向含 3 个元素的一维数组的指针（见 `03. 指针`） |

> [!important] 指针数组 vs 数组指针
> - `char* pa[4]` → **指针数组**：本质是数组，元素是指针
> - `int (*pa)[3]` → **数组指针**：本质是指针，指向一维数组

### 二级指针基础

```c
#include <stdio.h>

int main()
{
    int a = 100;
    int* pa   = &a;     // pa 存放 a 的地址
    int** ppa = &pa;    // ⭐ ppa 存放 pa 的地址（地址的地址）

    printf("%d \n", **ppa);   // 解两次引用，输出：100

    return 0;
}
```

### 指针数组 + 二级指针传参

```c
#include <stdio.h>

// ⭐ 用 char** 接收指针数组首地址
int test(char** pps)
{
    printf("ps[0]: %s \n", *pps);    // 解引用得到第一个字符串
    printf("ps[0]: %c \n", **pps);   // 二次解引用得到第一个字符 'h'
}

int main()
{
    char* str  = "hello world";
    char* str1 = "hello world1";
    char* str2 = "hello world2";

    char* ps[3] = {str, str1, str2};  // ⭐ 指针数组：3 个字符串指针

    printf("ps[0]: %s \n", *ps);      // 等价于 ps[0]
    printf("ps[1]: %s \n", ps[1]);
    printf("ps[2]: %s \n", ps[2]);

    test(ps);    // ⭐ 数组名 ps 是 char**，可传给 char** 参数

    return 0;
}
```

> [!note] 内存示意
> ```
> ps[0] → 地址A → "hello world\0"
> ps[1] → 地址B → "hello world1\0"
> ps[2] → 地址C → "hello world2\0"
>
> ps（char**）→ 指向 ps[0]
> *ps   → ps[0]（char*）→ "hello world"
> **ps  → 'h'（第一个字符）
> ```

> [!bug] 常见混淆
> - `ps` 是数组名，类型是 `char**`（指向 `char*` 的指针）
> - `*ps` 取到 `ps[0]`，类型是 `char*`（字符串指针）
> - `**ps` 再解引用，取到第一个字符 `'h'`

## 五、手动实现字符串比较

### 逻辑分析

实现类似 `strcmp()` 的功能：逐字符比较两个字符串，返回大小关系。

```c
#include <stdio.h>
#include <string.h>

int main()
{
    char* ps1 = "hello";
    char* ps2 = "world";

    int j = strlen(ps1);    // 获取 ps1 的长度
    int k = strlen(ps2);    // 获取 ps2 的长度
    int i, len, ret = 0;

    // ⭐ 取较长者的长度作为比较范围
    if(j < k)
        len = k;
    else
        len = j;

    // ⭐ 逐字符比较（通过指针偏移访问）
    for(i = 0; i < len; i++)
    {
        if(*(ps1+i) > *(ps2+i))
        {
            ret = 1;     // ps1 > ps2
            break;
        }
        else if(*(ps1+i) < *(ps2+i))
        {
            ret = -1;    // ps1 < ps2
            break;
        }
        // 相等则继续比较下一个字符
    }

    // 输出比较结果
    if(ret < 0)
        printf("ps1 < ps2 \n");
    else if(ret > 0)
        printf("ps1 > ps2 \n");
    else
        printf("ps1 == ps2 \n");
    // 输出：ps1 < ps2（'h' < 'w'）
}
```

比较原理：字符串比较本质是**逐字符按 ASCII 码比较**：
 - `'h'` = 104，`'w'` = 119 → `"hello" < "world"`
 - 遇到第一个不相等的字符就决出胜负，后面的字符不再比较

> [!note] 和标准库 `strcmp` 的对比
> | | 手动实现 | `strcmp()` |
> |--|---------|-----------|
> | 遇到 `\0` | 不处理（用长度控制） | 自动停止 |
> | 返回值含义 | -1 / 0 / 1 | 负数 / 0 / 正数（值不固定） |
> | 安全性 | 取较长长度，存在越界风险 | 安全 |

> [!warning] 此实现的 Bug
> 取较长字符串的长度进行比较时，若短字符串已结束（遇到 `\0`），继续访问可能读到垃圾值。
> 正确做法：以**较短长度**或**遇到 `\0` 为止**作为结束条件。
