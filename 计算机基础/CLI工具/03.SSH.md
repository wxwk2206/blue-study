## 常用 SSH 工具命令
1. 远程登录
```shell
ssh user@IP
ssh -p 2222 user@IP  # 指定端口
```

2. 生成密钥
```shell
ssh-keygen -t ed25519
```

3. 免密推送公钥
```plain
ssh-copy-id user@IP
```

4. SCP 文件传输
```plain
# 本地上传服务器
scp 文件 user@IP:/路径
# 服务器下载到本地
scp user@IP:/文件 本地路径
```

5. SFTP 安全文件传输
```plain
sftp user@IP
```

## 端口转发（三大核心）
### 1. 本地转发（L 本地→远程）
把本地端口映射到服务器，访问本地等于访问服务器内网服务
```plain
ssh -L 本地端口:目标IP:目标端口 user@SSH服务器
```

### 2. 远程转发（R 远程→本地）

 服务器端口映射到本地，外网可访问本地内网服务（[[内网穿透]]常用）

```plain
ssh -R 远程端口:本地IP:本地端口 user@SSH服务器
```

### 3. 动态转发（D socks5 代理）
搭建本地 socks5 代理，科学上网 / 内网漫游
```plain
ssh -D 1080 user@IP
```

## SSH 服务端配置（/etc/ssh/sshd_config）
常用安全配置项：
+ `Port 2222` 修改默认端口
+ `PermitRootLogin no` 禁止 root 直接登录
+ `PasswordAuthentication no` 关闭密码登录，只允许密钥
+ `PubkeyAuthentication yes` 开启密钥认证改完重启：`systemctl restart sshd`
