# 换源
5.打开终端，输入sudo vim etc/apt/sources.list

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/62702379/1763725282011-e7c72b7c-d24f-4a49-a4a1-a5e0de6a129b.png)

进入之后，单击i进入编辑模式，将源换为国内源

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/62702379/1763725097767-0b652b48-5717-43f2-9515-30bd12b33000.png)

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/62702379/1763725306194-64482187-9f9d-4464-bd3b-6855e78dc08d.png)

点esc，shift加；，输入wq退出

6.使用以下指令，刷新加升级

sudo apt-get update刷新软件列表

sudo apt upgrade升级已装软件
