## 总结
### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">一、HTML 常用核心标签</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">HTML 是网页的骨架，这些标签是构建网页最基础、最常用的，我按功能分类说明：</font>

#### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">1. 文档结构标签（页面骨架）</font>
| **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">标签</font>** | **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">作用</font>** |
| --- | --- |
| `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"><!DOCTYPE html></font>` | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">声明文档类型，告诉浏览器这是 HTML5 文档（必须放在第一行）</font> |
| `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"><html></font>` | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">整个 HTML 文档的根标签，所有内容都包裹在里面</font> |
| `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"><head></font>` | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">页面的头部，存放元信息（如标题、编码、CSS 链接），不显示在页面主体</font> |
| `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"><title></font>` | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">定义网页标题，显示在浏览器标签栏，是 SEO 核心</font> |
| `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"><body></font>` | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">页面的主体内容区域，所有可见内容（文字、图片、按钮等）都放在这里</font> |

### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">二、CSS 常用核心样式</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">CSS 是网页的 “皮肤”，用于美化 HTML 标签，以下是最常用的样式属性，按功能分类：</font>

#### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">1. 布局与盒模型（最核心）</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">盒模型是 CSS 布局的基础，每个标签都是一个 “盒子”，核心样式：</font>

| **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">样式属性</font>** | **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">作用</font>** |
| --- | --- |
| `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">width/height</font>` | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">设置盒子的宽度 / 高度（如</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">width: 200px;</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">）</font> |
