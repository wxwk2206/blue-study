## 引入方式
**内部脚本**：将JS代码定义在HTML页面中

+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">JavaScript 代码必须位于 </font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"><script></script></font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"> 标签之间</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">在 HTML 文档中，可以在任意地方，放置任意数量的 </font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"><script></font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"> 标签</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">一般会把脚本置于 </font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"><body></font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"> 元素的底部，可改善显示速度</font>

```html
<body>
  <!-- 页面内容 -->

  <script>
    alert('Hello World');
  </script>
</body>
```

## 基础语法
### 变量/常量
JS中用**let**关键字来声明变量（弱类型语言，变量可以存放不通类型的值）

变量名需遵循如下规则

+ 只能用字母、数字、下划线（_）、美元符号（$）组成，且数字不能开头。
+ 变量名严格区分大小写，如 name 和 Name 是不同的变量。
+ 不能使用关键字，如：let、var、if、for 等。

```html
<script>
  // 变量
  let a = 20;
  a = "Hello";
  alert(a);
</script>
```

JS中用**const**关键字来声明常量，一但声明，常量的值就不能改变

```html
<script>
  // 常量
  const PI = 3.14;
  console.log(PI);
</script>
```

**模板字符串**语法：

**反引号** `：在英文输入模式下，按下键盘上 Tab 键上方、波浪线 ~ 所在的按键

**变量拼接**：在模板字符串中，将变量放入 ${} 语法中

```html
<script>
    let name = 'Tom';
    let age = 18;
    // 传统字符串拼接方式
    console.log('大家好，我是新入职的' + name + '，今年' + age + '岁了，请多多关照');
    // 模板字符串（反引号）拼接方式
    console.log(`大家好，我是新入职的${name}，今年${age}岁了，请多多关照`);
</script>
```
