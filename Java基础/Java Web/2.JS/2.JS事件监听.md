## 常见事件
### 鼠标事件
+ click鼠标点击
+ mouseenter鼠标移入
+ mouseleave鼠标移出

### 键盘事件
+ keydown键盘按下触发
+ keyup键盘抬起触发

### 焦点事件
+ focus获取焦点触发
+ blur失去焦点触发

### 表单事件
+ input用户输入时触发
+ submit表单提交时触发
