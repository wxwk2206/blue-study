# 1.删除
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">需求：根据 ID 删除用户信息</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">SQL：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">delete from user where id = 5;</font>`
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Mapper 接口：</font>

```java
@Delete("delete from user where id = 5")
public void deleteById();
```

使用#占位符是安全的，执行时会将其替换为？，是预编译

```java
@Delete("delete from user where id = #{id}")
public void deleteById(Integer id);
```

<font style="color:rgb(0, 0, 0);">注意：DML语句执行完毕的返回值，表示该DML语句执行完毕影响的行数。</font>

```java
@Delete("delete from user where id = #{id}")
public Integer deleteById(Integer id);
```

### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">1.1Mybatis 中的 # 号与 $ 号：</font>
| **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">符号</font>** | **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">说明</font>** | **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">场景</font>** | **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">优缺点</font>** |
| :--- | :--- | :--- | :--- |
| `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">#{...}</font>` | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">占位符。执行时，会将</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">#{...}</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">替换为</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">?</font>`<br/><font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">，生成预编译 SQL。</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">参数值传递</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">安全、性能高（推荐）</font> |
| `<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">${...}</font>` | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">拼接符。直接将参数拼接在 SQL 语句中，存在SQL 注入问题。</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">表名、字段名动态设置时使用</font> | <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">不安全、性能低</font> |

# 2.新增
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">需求：添加一个用户</font>

+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">SQL：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">insert into user(username,password,name,age) values('zhouyu','123456','周瑜',20);</font>`
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Mapper 接口：</font>

<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">错误示例：</font>

```java
@Insert("insert into user(username,password,name,age) values('zhouyu', '123456', '周瑜', 20)")
public void insert();
```

<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">正确示例：</font>

```java
@Insert("insert into user(username,password,name,age) values(#{username},#{password},#{name},#{age})")
public void insert(User user);
```

# 3.修改
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">需求：根据 ID 更新用户信息</font>

+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">SQL：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">update user set username = 'zhouyu', password = '123456', name = '周瑜', age = 20 where id = 1;</font>`
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Mapper 接口：</font>

```java
@Update("update user set username=#{username}, password=#{password}, name=#{name}, age=#{age} where id=#{id}")
public void update(User user);
```

# 4.查询
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">需求：根据用户名和密码查询用户信息</font>

+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">SQL：</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">select * from user where username = 'zhouyu' and password = '666888';</font>`

<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Mapper 接口：</font>

```java
@Select("select * from user where username=#{username} and password=#{password}")
public User findByUsernameAndPassword(@Param("username") String username, @Param("password") String password);
```

**<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@Param 注解的作用是为接口的方法形参起名字的。</font>**

**<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);"></font>**

<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">说明：基于</font>**<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">官方骨架创建的 springboot 项目</font>**<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">中，接口编译时会保留方法形参名，</font>**<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@Param 注解可以省略</font>**<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">（#{形参名}）。</font>

<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Mapper 接口：</font>

```java
@Select("select * from user where username=#{uname} and password=#{pwd}")
public User findByUsernameAndPassword(String uname, String pwd);
```
