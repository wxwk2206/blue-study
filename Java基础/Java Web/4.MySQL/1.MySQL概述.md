## 安装
### 1.下载解压
下载地址：[https://dev.mysql.com/downloads/mysql/](https://dev.mysql.com/downloads/mysql/)

  下载压缩包解压到没有中文的路径下

### 2.配置环境变量

### 3.初始化
<font style="color:#000000;">以管理员身份，运行命令行窗口：</font>

```sql
mysqld --initialize-insecure
```

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/62702379/1772862728306-b082b0f1-7f17-42fc-9d58-85890e896303.png)

稍微等待一会，如果出现没有出现报错信息，则证明data目录初始化没有问题，此时再查看MySQL目录下已经有data目录生成。

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/62702379/1772862902375-93caa3bf-bc3d-4bba-ab02-0200880ae27f.png)

### 4.注册MySQL服务
命令行（注意必须以管理员身份启动）中，输入如下的指令，回车执行： 

```sql
mysqld -install
```

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/62702379/1772863031020-27f1191d-798a-4519-8e73-25671e393730.png)

显示成功即安装好了MySQL服务了

### 5.启动MySQL服务
在黑框里敲入`<font style="color:rgb(46,161,33);">net start mysql</font>`，回车。

```sql
net start mysql  // 启动mysql服务
    
net stop mysql  // 停止mysql服务
```

### 6.修改默认账户密码
在黑框里敲入`<font style="color:rgb(222,120,2);">mysqladmin -u root password 1234</font>`，这里的`1234`就是指默认管理员(即root账户)的密码。

```sql
mysqladmin -u root password 1234
```

## 登录
右键开始菜单，选择`<font style="color:rgb(222,120,2);">命令提示符</font>`，打开黑框。

在黑框中输入，`<font style="color:rgb(222,120,2);">mysql -uroot -p1234</font>`，回车，出现下图且左下角为`<font style="color:rgb(222,120,2);">mysql></font>`，则登录成功。

```sql
mysql -uroot -p1234
```

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/62702379/1772863566850-440bfd79-d923-42b0-b6c7-1102d75dcab7.png)

退出mysql：

```sql
exit
```

登录参数：

```sql
mysql -u用户名 -p密码 -h要连接的mysql服务器的ip地址(默认127.0.0.1) -P端口号(默认3306)
```

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/62702379/1772864238775-f56b76c3-ded8-4a76-b004-01886e6e9c9f.png)

登陆时，可以直接-p，然后回车，密码以密文形式输入
