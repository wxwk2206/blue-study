## 字面量
程序中的数据

| 字面量类型 | 说明 | 举例 |
| --- | --- | --- |
| 整数类型 | 直接写 | 12 |
| 小数类型 | 直接写，加小数点 | 1.2 |
| 字符串类型 | 用双引号引用 | “哈哈” |
| 字符类型 | 用单引号引起，内容只能有一个 | ‘哈’ |
| 布尔类型 | 布尔值，表示真假 | true，false |
| 空类型 | 一个特殊值，空值 | null |


## 变量
程序中，用来存储单个数据的容器，把经常变化的数据存入变量

**变量是存储数据的小空间**，不是里面的数据

变量的定义格式： **数据类型 变量名 =数据值；** int a =10；

## 计算机的存储单元
1字节（最小存储单元）=8bit比特位，一个0或1就是一个bit

### 引用数据类型
#### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">1. 类（Class）类型</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">这是最核心的引用类型，包含两种：</font>

+ **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">系统预定义类</font>**<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">：Java 内置的类（如</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">String</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Date</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">ArrayList</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">等）；</font>
+ **<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">自定义类</font>**<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">：自己编写的类（如</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Person</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Student</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">等）。</font>

```java
// 1. 系统预定义类（String是典型的引用类型，且不可变）
String str = "Java"; 
// 2. 自定义类
class Person {
    String name;
    int age;
}
Person p = new Person(); // p是引用，指向Person对象
```

#### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">2. 接口（Interface）类型</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">接口本身不能实例化，但接口的</font>**<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">实现类对象</font>**<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、接口变量都属于引用类型，包括：</font>

+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">系统内置接口（如</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">List</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Map</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Runnable</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">等）；</font>
+ <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">自定义接口。</font>

```java
// 自定义接口
interface Animal {
    void eat();
}
// 实现类
class Cat implements Animal {
    @Override
    public void eat() {
        System.out.println("猫吃鱼");
    }
}
// animal变量是接口类型的引用，指向Cat对象
Animal animal = new Cat();
```

#### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">3. 数组（Array）类型</font>
**<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">所有数组（无论存储基本类型还是引用类型）都是引用类型</font>**<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">，哪怕是</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">int[]</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">char[]</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">这类存储基本类型的数组。</font>

```java
// 存储基本类型的数组（引用类型）
int[] numArr = new int[]{1, 2, 3};
// 存储引用类型的数组（引用类型）
String[] strArr = new String[]{"a", "b"};
// 二维数组（依然是引用类型）
int[][] twoDArr = new int[2][3];
```

#### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">4. 枚举（Enum）类型</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">Java 中的枚举是</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">java.lang.Enum</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">类的子类，本质是特殊的类，属于引用类型。</font>

```java
// 定义枚举
enum Week {
    MONDAY, TUESDAY, WEDNESDAY
}
// week是枚举类型的引用
Week week = Week.MONDAY;
```

#### <font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">5. 注解（Annotation）类型</font>
<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">注解（</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@interface</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">）在 Java 中也是一种特殊的引用类型，常用于代码标记（如</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@Override</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">、</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">@Deprecated</font>`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">）。</font>
