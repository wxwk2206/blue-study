数据重复，所有要用到继承

继承是类与类之间的一种父子关系，Java中提供关键字extends，用于建立类与类之间的关系

```java
public class Student extends Person{
    
}
```

**好处**

+ 可以把多个子类中重复的代码抽取到父类中，提高代码的复用性
+ 子类可以在父类的基础上，增加其他功能，使子类更加强大

**特点**

Java只支持**单继承**，不支持**多继承**，但支持**多层继承**

+ 一个子类只能继承一个父类
+ 多层继承：有直接父类和间接父类
+ 顶级父类是object，每一个类都直接或间接继承	object

#### 方法重写
**@override**

方法重写：在子类中，把父类的方法再写一遍，方法申明保持一致

使用场景：如果父类的方法不能满足子类的要求了，子类中可以把该方法再重写一遍

重写时，父类的方法还想用，在父类的基础上加其他逻辑，可以使用super关键字调用父类方法得到一个结果，再对结果进行其他操作

##### 注意事项
+ 重写方法的名称，形参列表必须与父类中的一致，方法体按照实际需求书写
+ 了解：子类重写父类方法时，访问权限子类必须大于等于父类（private<空着不写-包私有 Package-private  <protected<public）

+ 了解：子类重写父类方法时，返回值类型子类必须小于等于父类
+ 重写方法申明与父类保持一致即可 
+ final修饰类为最终类，里面所有的方法不能被重写
+ **只有虚方法能被重写，private私有方法，static静态方法，final最终方法不能被重写**

### 构造方法
父类中的构造方法不会被子类继承，只能通过super关键字调用

## 多态
多态：事物的多种形态

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/62702379/1771914254257-1f2a81dc-b294-4f77-8548-fd542e783ee8.png)

多态的表现形式：父类类型 对象名称 = 子类对象；

多态的前提：

+ 有继承/实现关系                  （必须）
+ 有父类引用指向子类对象  Fu f = new Zi ( ) ;   （必须）
+ 有方法重写                          （可选）

### 多态调用成员的特点
变量调用：编译看左边（在把Java文件编译成class文件的时候，看父类当中有没有这个变量，有则成功，无则失败），运行也看左边（代码真正运行时，使用父类中的变量）

方法调用：编译看左边（看父类中有没有这个方法，如果没有代码报错），运行看右边（代码真正运行时，运行的是子类里面的方法），如果子类没有重写父类的方法，还是使用父类中的方法

弊端：多态下不能使用子类的独有功能

解决方法：**类型转换-->**调用子类独有功能

自动类型转换（从子到父）：子类对象赋值给父类类型的变量

强制类型转换（从父到子）：子类 对象变量 = （子类）父类类型的变量

```java
Person p = new Student（）;
Student s = （Student）p;
```

  运算符：**instanceof  **

核心功能是**<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">检查一个对象是否是某个类（或其子类 / 实现的接口）的实例</font>**，并返回布尔值（`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">true</font>`/`<font style="color:rgb(0, 0, 0);background-color:rgba(0, 0, 0, 0);">false</font>`）。
