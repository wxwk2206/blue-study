# 1.介绍
类型注解是 Python 的语法特性，用于**明确标注变量、函数参数和返回值的数据类型**，让代码更清晰、更易维护。只是一种**提示**，不会在运行时强检查类型，Python 依然是动态类型语言。
```
a: int = 695 
score: float = 98.5 
hobby: str = "Python" 
flag: bool = True 
pic: None = None
names: list[str] = ["A", "C", "E"] 
phones: set[str] = {"13309091111", "15209109121"} 
options: dict[str, int] = {"count": 0, "total": 0} 
goods: tuple[str, int, int] = ("手机", 5999, 1)
```

## 1.1类型推断
类型推断是指python解释器自动推断出变量、表达式或函数返回值的数据类型的能力，无需开发者显式声明

# 2.函数-类型注解

为函数添加类型注解，只需标注**参数**和**返回值**，语法如下：

```
def 函数名(参数名: 类型) -> 返回值类型:
    # 函数体
    ...
```

#### 示例1 ：（单返回值）

```
from typing import List

def calc(scores: list[int]) -> float:
    """
    计算分数列表的平均值
    :param scores: 整数类型的分数列表
    :return: 浮点数类型的平均值
    """
    return sum(scores) / len(scores)
```

- **参数注解**`scores: list[int]`：指定 `scores` 是**整数列表**。
- **返回注解**`-> float`：指定函数返回值是**浮点数**。
#### 示例 2：（多返回值）

```
from typing import List, Tuple

def calc_data(scores: list[int]) -> tuple[int, int, float]:
    """
    同时返回分数的最大值、最小值、平均值
    :param scores: 整数类型的分数列表
    :return: 元组 (最大值, 最小值, 平均值)
    """
    max_v = max(scores)
    min_v = min(scores)
    avg_v = sum(scores) / len(scores)
    return max_v, min_v, avg_v
```

- **参数注解**`scores: list[int]`：接收整数列表。
- **返回注解**`-> tuple[int, int, float]`：指定返回值是**三元组**，依次为（整数、整数、浮点数）。
