一个 `.py` 文件就是一个 **Python 模块**，它是 Python 程序的**基本组织单位**。
# 1.导入模块

| 导入语法                        | 代码样例                                 | 调用方式                                   |
| --------------------------- | ------------------------------------ | -------------------------------------- |
| `import 模块名`                | `import random, os`                  | `模块名.功能名`（如 `random.randint(10, 100)`） |
| `import 模块名 as 别名`          | `import random as rd`                | `别名.功能名`（如 `rd.randint(10, 100)`）      |
| `from 模块名 import 功能名`       | `from random import randint, choice` | `功能名`（如 `randint(10, 100)`）            |
| `from 模块名 import 功能名 as 别名` | `from random import randint as rint` | `别名`（如 `rint(10, 100)`）                |
| `from 模块名 import *`         | `from random import *`               | `功能名`（如 `randint(10, 100)`）            |

# 2.自定义模块
#### 2.1 `__name__` 内置变量

`__name__` 是 Python 核心内置变量，代表**当前模块的名称**，其值随运行方式变化：

|运行方式|`__name__` 的值|典型用法|
|---|---|---|
|模块**直接运行**（如 `python demo.py`）|`"__main__"`|`if __name__ == "__main__":` 包裹测试代码|
|模块**被导入**（如 `import demo`）|模块文件名（不含 `.py`）|被导入时不执行测试代码|

```
# demo.py
def add(a, b):
    return a + b

# 仅当直接运行 demo.py 时才执行
if __name__ == "__main__":
    print(add(1, 2))  # 输出 3
```

#### 2.2 `__all__` 变量

`__all__` 用于**控制 `from 模块名 import *` 时导入的功能列表**，只允许列表中指定的名称被导入。

```python
# demo.py
__all__ = ["add", "sub"]  # 仅允许导入 add、sub 两个功能

def add(a, b):
    return a + b

def sub(a, b):
    return a - b

def mul(a, b):
    return a * b
```

当执行 `from demo import *` 时，只会导入 `add` 和 `sub`，`mul` 不会被导入。

# 3.软件包package

## 3.1 核心定义

- **包的本质**：一个**文件夹**，里面可以包含多个 `.py` 模块文件，且必须包含一个 `__init__.py` 文件（标识这是一个 Python 包）。
- **包的本质也是模块**：可以像普通模块一样被导入，用于**管理多个模块**，解决模块文件过多时的组织问题。

## 3.2 包导入方式 

|导入形式|代码样例|调用方式|调用示例|
|---|---|---|---|
|`import 包名.模块名`|`import utils.my_fun`|`包名.模块名.功能名`|`utils.my_fun.log_separator1()`|
|`from 包名 import 模块名`|`from utils import my_fun`|`模块名.功能名`|`my_fun.log_separator1()`|
|`from 包名 import *`|`from utils import *`|`模块名.功能名`|`my_fun.log_separator1()`|
|`from 包名.模块名 import 功能名`|`from utils.my_fun import log_separator1`|`功能名`|`log_separator1()`|
|`from 包名.模块名 import *`|`from utils.my_fun import *`|`功能名`|`log_separator1()`|
