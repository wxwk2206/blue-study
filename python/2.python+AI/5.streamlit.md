# 1.介绍 

- **定义**：Streamlit 是一个开源 Python 库，专为数据工程师和机器学习工程师设计，能让开发者无需掌握前端技术，仅用 Python 代码快速构建交互式 Web 应用。
- **核心优势**：
    - 纯 Python 开发，学习成本低，无需写 HTML/CSS/JS
    - 一键启动服务，迭代速度快，适合快速原型验证
    - 内置丰富组件（滑块、下拉框、图表展示等），可快速实现数据可视化和交互功能
- **官方网站**：[https://streamlit.io](https://streamlit.io)
# 2.步骤
- 安装 streamlit: `pip install streamlit`
- 在 python 文件中引入 streamlit 模块
- 基于 streamlit 中提供的 API 来构建 Web 应用
- 运行程序: `streamlit run xxxx.py`
